function getBlogContentsList(page,added_params) {

        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');

나중에 가생이 부분은 버터로 구운뒤 설탕뿌려서 먹으면 맛나요!</li>
 <li>​그리고 밀대를 이용해서 쭉쭉 밀어서 납작하게 해줍니다.</li>
 <li>7장을 먼저 모두 밀어준뒤 시작하는게 좋아요.

 

사진에서 보시면 진짜 얇아졌죠?

 

만약에 밀대가 없으면 랩이나, 소주병? 요런걸로도 가능하고,

 

밀어도밀어도 자꾸 좀비처럼 안밀리고 살아난다면, 분무기에 물을 살짝 넣고 한번만 촥! 뿌려주면 잘 밀려요 ㅎㅎ</li>
 <li>​

​계란2개를 잘풀어줍니다.

​

여기서 소금+설탕 넣어서 간을 해도 좋지만

​

저는 패스했는데도 간은 괜찮았어요.​</li>
 <li>이제 베이컨 롤토스트를 만들어볼게요

 

밀어두었던 식빵을 준비합니다.</li>
 <li>딸기쨈을 골고루 발라줍니다.

 

여기서 딸기잼이 좋다고 많이 바르면 먹다가 몸서리칠만한 단맛을 경험하게 됩니다.

 

많이 넣지 마세요 ㅎㅎㅎㅎ</li>
 <li>딸기잼을 바른후 조심조심 돌돌 말아줍니다.</li>
 <li>베이컨을 올린뒤, 약간 대각선방향으로 말아둔 식빵을 올려서 돌돌 말아줍니다.</li>
 <li>식빵과 베이컨을 모두 말았다면, 말았던 끝부분이 밑으로 닿도록 해줍니다.

 

안그러면 풀리는거 아시죠?^^</li>
 <li>계란물에 식빵을 담구어 계란물을 입혀줍니다.</li>
 <li>사진에서 보이는 부분이 끝부분이라고 칭하고 저부분이 후라이팬 바닥쪽으로 닿게 해서 구워주셔야 분리되지 않아요.</li>
 <li>후라이팬에 버터를 넣고 버터가 녹으면 계란물을 입힌 식빵을 올려줍니다.</li>
 <li>앞뒤옆으로 굴려가면서 노릇하게 구워주면 완성입니다</li>
 </ol>

 <h1><b>3.영양만점 브런치~ [베이컨시금치 오믈렛]</b>
 <span>by.레이디스</span>

 <dt>[주재료]</dt>
 <dd>달걀 2개, 베이컨 1줄, 우유 1큰술, 시금치 25g, 모짜렐라치즈 1작은술</dd>

 <h1><b>4.브런치 메뉴로 좋은 고구마 치즈 베이컨말이 샌드위치</b>
 <span>by.예쁜포비</span>
 <div class="recipe_info">
 <b>분량 :</b>3인분 <b>난이도 :</b>중급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/04/09/68100dd39a686df7d809ec46a86bca5d1.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>곡물 식빵 3개(토스트 식빵도 가능), 고구마 200g</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>크림치즈 2큰술(우유대체가능), 연유 1큰술, 슬라이스 치즈 3개, 베이컨 6줄, 달걀 2개, 버터 1큰술</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>냄비에 물을 넣고 고구마를 삶아 주세요.</li>
 <li>식빵, 베이컨, 슬라이스치즈, 댤걀, 버터, 삶은고구마, 크림치즈, 연유를 준비해 주세요.</li>
 <li>삶은 고구마는 뜨거울 때 껍질을 벗겨 크림치즈와 연유를 넣고 으깨 주세요.</li>

 <li>식빵 위에 슬라이스치즈와 고무마 무스를 길게 올린 후 김밥 말듯이 돌돌 말아 주세요.</li>
 <li>베이컨 2장을 빵에 돌돌 감아 주세요. 베이컨을 감아서 구우면 빵이 풀리지 않습니다.</li>
 <li>잘 풀어 놓은 달걀에 빵을 담가 달걀 옷을 입혀 주세요.</li>
 <li>중불에서 달군 팬에 버터를 넣고 베이컨을 만 빵을 돌돌 굴려가면서 익혀 주세요.</li>
 <li>커피와 먹으면 완전 근사한 브런치 메뉴가 된답니다.</li>
 </ol>

 <h1><b>5.간단 브런치 </b>
 <span>by.쿡라이프</span>
 <div class="recipe_info">
 <b>분량 :</b>1인분 <b>조리시간 :</b>15분 이내 <b>난이도 :</b>초급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/07/02/25cb928630d24dfb83f13575ac9ee51e.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>식빵 2장, 달걀 2개, 양파 1/2개, 당근 30g, 피망 1/2개</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>케찹</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>피망, 양파, 당근은 굵기가 있게 썰어 줍니다. 0.2~0.3cm로 썹니다.</li>
 <li>달구어진팬에 기름을 두르고 당근, 양파를 넣고 볶다가 당근이 다 익으면 소금간 하시고, 피망을 넣고 슬쩍 볶아 냅니다. 완전히 식혀 두세요.</li>
 <li>식빵을 최대한 네모나게 오려냅니다.</li>
 <li>그릇에 달걀을 깨어 놓고, 소금, 우유 2큰술을 넣고 저어 줍니다. 여기에 볶아 두었던 채소를 섞어 줍니다.</li>
 <li>팬에 버터 조금 두르고 빵을 놓고 구멍뚫린 사각틀 속에 달걀물을 천천히 부은 후 스크램블에그 하듯 젓가락으로 저어 줍니다. 젓가락으로 저으시면 달걀물이 식빵 밖으로 나가지 않는답니다. 약불에서 해 주세요.</li>
 <li>저은 후 팬 뚜껑 닫고 약불에서 익힌 한면이 다 익었으면 뒤집어 팬 뚜껑 닫고 마저 익혀 줍니다. 남은 식빵도 토스트하여 같이 곁들여 드시면 된답니다.</li>
 </ol>
 </dd>
 </dl>

function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}

$(document).ready(function () {
    getViewRecipeList(1);
})
