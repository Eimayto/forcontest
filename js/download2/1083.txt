function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);

}

 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>탕수육의 고기는 등심이 적합해요.

​기름기가 없고 살코기라 담백하고 맛있어요.</li>
 
 <li>​

고기를 걸 반대로 썰어서 준비하고 파프리카, 오이, 양파, 파인애플을 준비하세요.</li>
 
 <li>파이애플은 먹기 좋게 썰어서 준비해요. 파인애플은 소화를 잘 되게 하고 고기를 부드럽게 해줘요 

파인애플 국물은 소스에 넣어주면 단맛이 더 상승시켜줘요</li>
 
 <li>탕수육 만들기에 각각 야채를 먹기 좋게 썰어 준비해줘요~

색감을 화사하게 파프리카, 오이는 맨 나중에 넣어 주셔야 해요

끓일 때 넣어주면 색감이 죽어버려서 볼품이 없어져요</li>
 
 <li>탕수육에 들어가는 등심은 결 반대로 손가락 길이로 썰어서 준비해줘요.

탕수육의 옷을 입힐 때 전분으로 하는데 저는 농심 건강한 쌀 튀김가루를 넣어 만들어 봤어요.</li>
 
 <li>등심 300g을 찬물로 한번 헹궈준 뒤 충분히 물기를 빼줘요 

 물기 뺀 고기에 간장 1/2큰 술을 넣고 충분히 스며들도록 주물러줘요
​
달걀물 2 큰 술을 넣고 농심 쌀 튀김가루 4 큰 술과 전분 덩어리 2 큰 술을 넣고 반죽을 해요

냄비에 기름을 부어주고 기름의 온도가 전분이 가라앉고 기포가 생길 때(약 130℃)

1차로 3분간 저어주며 튀겨줘요</li>
 
 <li>튀긴 고기를 건져낸 뒤 (약180℃) 2차로 1분간 튀겨줘요</li>
 
 <li>이때 체로 고기를 두드려주면 기름 안에서 골고루 익게 하고 바삭함이 있답니다</li>
 
 <li>두번 튀겨야 바삭함이 있어요</li>
 
 <li>소스를 만들고 있어요</li>
 
 <li>물 1컵 반(300cc), 설탕 8 큰 술, 2배 식초 5 큰 술, 중국 간장 1/3 큰 술, 파인애플 국물 2 큰 
양파 1/4개, 당근 1/5개, 오이 반 컵, 파프리카 1/2컵 , 파인애플 1컵, 전분 물 4 큰 술</li>
