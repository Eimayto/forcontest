function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();

            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');

}

 <li>쌈장도 만들어 준비합니다. 시판용 쌈장을 사용해도 상관없어요. 전 직접 만들었구요.... (쌈장만드는법 - 된장3스푼, 고추장1스푼, 들깨가루1스푼, 다진마늘1스푼, 매실액기스1스푼, 참기름2스푼, 해바라기씨, 아몬드슬라이스)</li>
 <li>밥은 참기름만 살짝 넣어서 섞어줍니다.
삼겹살 김밥에 쌈장이 충분히 들어가니 밥은 굳이 간을 안해도 됩니다.</li>
 <li>삼겹살김밥의 메인재료인 삼겹살도 2줄 구워주고요~</li>
 <li>센불에서 바싹하게 익히는데 기름이 튀니까 키친타올로 덮어주는 센스!!</li>
 <li>앞뒤로 노릇노릇하게 구워서 준비하구요~</li>
 <li>김밥용 김에 밥을 얇게 펼쳐줍니다.</li>
 <li>그리고 깻잎을 올려주는데요~ 깻잎을 골고루 펼쳐가며 총8장을 깔아줍니다. 취향에 따라 상추로 바꾸셔도 됩니다.

깻잎위에 노릇노릇하게 구운 삼겹살도 나란히 올려주고요~</li>
 <li>그 위에 오이채와 청양고추를 올려주는데요~ 청양고추 속을 파낸곳에 쌈장으로 채워서 올려줍니다.</li>
 <li>그리고 돌돌 말아주기!!!</li>
 <li>삼겹살이 두툼하게 감싸고 있는 김밥, 보고만 있어도 군침이 흐르지요? ㅎㅎㅎ</li>
 <li>삼겹살 김밥을 적당한 크기로 썰어주고요~</li>

 <h1><b>11.소시지 초밥과 김밥</b>
 <span>by.행복하자하루하루</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>30분 이내 <b>난이도 :</b>초급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2016/06/27/736b8e7fcd9641a1d0baf57b9f894c4c1.jpg">
 </h1>


 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>비엔나 소시지 초밥 만들기 : 비엔나소시지를 모양틀로 모양을 내준후 뜨거운물에 삶아줍니다.</li>
 <li>뜨거운물에 삶아내니 모양이 더 뚜렷하게 나오네요. 삶아낸 비엔나소시지를 반으로 잘라주세요</li>
 <li>뜨거운 흰밥에 참기름과 밥이랑야채,치즈를 넣고 잘 섞어주세요</li>
 <li>골고루 잘 섞어줍니다.</li>
 <li>한입모양으로 잘 뭉쳐줍니다.</li>
 <li>달걀지단과 치즈를 밥크기대로 잘라 올려줍니다.</li>
 <li>밥-치즈 또는 달걀지단 - 비엔나소시지 순서대로 올려줍니다.</li>

 <li>김을 길게 잘라서 밥과햄이 잘 고정될수있도록 돌돌 말아주세요</li>
 <li>비엔나 초밥이 완성되었어요 이뿌죠~아이들과 함께 만들면 더 재미있고 좋아한답니다.</li>
 <li>후랑크소시지 김밥 만들기 : 후랑크소시지를 물에 삶아줍니다.</li>
 <li>김을 반으로 자르고 밥을 골고루 펴 줍니다.</li>
 <li>물에 삶아준 소시지를 올려줍니다.</li>
 <li>소시지를 넣고 돌돌 말아 김밥을 완성합니다.</li>
 <li>만들어놓은 지단에 김밥을 올려 잘 말아줍니다. 한입크기로 잘 썰어줍니다.</li>
 <li>소시지로 만든 초밥과 김밥 완성되었어요~</li>
 <li>모양이 너무 이뿌죠~ 맛도 최고 만드는 재미도 최고~</li>

 <h1><b>12.참치김밥 황금레시피 참치김밥 만드는 법 </b>
 <span>by.꽃청춘이주부</span>
 <div class="recipe_info">
 <b>분량 :</b>3인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>아무나 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/09/07/3b06dbe785155c6bffd0626538d13cbc1.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>쌀 맥주컵2컵, 김밥햄, 단무지, 맛살, 깻잎 12장, 오이 1개, 당근 1/2개, 참치캔, 계란 2개, 양파 1/4개</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>참기름, 소금, 기름, 깨</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>​재료:쌀(백주컵2컵)김밥햄,단무지,맛살,깻잎12장,오이1개,당근1/2개,참치캔,계란2개,양파1/4개

​

참기름,소금,기름,깨​</li>
 <li>​밥을 먼저 할건데요~^^

맥주컵2컵의 양을 준비해서 깨끗하게 씻어줍니다.

쌀을 씻는 물의 온도가 40도로 해야 밥이 찰​치고 시간이 단축된다고 하네요~

​

물은 손등마디까지 해서

비율을 따지면 물1:쌀1:비율이예요~^^​

​</li>
 <li>참치김밥의 밥이 되는동안 재료를 손질 할께요!

황금 레시피에는 햄과 단무지 맛살이 들어가지 않지만 

전 넣도록할게요!

김밥용햄을 잘라주고, 맛살도 반을 잘라서 준비합니다.

 

오이1개와 당근1/2개를 채썰어줍니다.</li>
 <li>당근1/2개를 채를 썰어서 기름을 살짝 두른 후라이팬에 소금을 조금 넣고 볶아주세요.</li>
 <li>​오이1개를 채썰어서 준비하신뒤 소금에 버무려서 바로 사용합니다.

​

나중에 보니까 물이 생기더라구요~

​

오이를 채썰어서 소금에 절이는것은 제일 나중에 하는걸 추천합니다.</li>
 <li>김밥용햄을 기름을 두르지 않은 후라이팬에 살짝 구워주세요.

 

여기서 황금레시피에는 우엉이 들어가는데 저는 우엉을 싫어해서 안넣었어요!

 

혹시 우엉을 넣으실분들은

우엉을 물에 담구어 식초를 넣어 20분간 두었다가

 

잘 헹구어 팬에 물+흑설탕+간장+물엿을 넣고 조립니다.

 

30분간 중불에서 조리면 우엉조림 완성</li>
 <li>계란을 2개 준비하신뒤에 잘풀어서 소금으로 살짝 간을 하시고 얇게 부쳐줍니다.</li>
 <li>잘 부쳐진 계란을 둥굴게 말아서 채썰어주세요

계란은 두껍게 썰지 마시고, 가늘게 채를 썰어야 다른재료와 조화를 이루어서 단백한 맛을 낸다고 해요~^^</li>
 <li>참치캔을 한개 준비하셔서 채에 걸러 기름을 빼주세요.</li>
 <li>양파1/4개를 준비하신뒤에 잘게 다져서 준비해주세요

양파를 다져서 함께 넣어주면 담백하고, 비린맛이 없어져서 상큼하게 드실수 있다고 해요^^</li>
 <li>양파 다진것+참치캔+마요네즈3스푼+후추(두번톡톡)해서 잘섞어서 참치김밥의 속을 준비해주세요.</li>
 <li>재료준비가 끝났을때쯤 밥이 완성이 되었을거예요

 

밥을 꺼내서 깨(2스푼정도)와 참기름(1스푼반)을 넣고 잘섞어줍니다.</li>
 <li>참치김밥 황금레시피 재료 준비 끝!

이제 본격적으로 말아줄께요!</li>
 <li>보통은 김밥용 김을 사시면 김이 10장인데요~

그중에3장을 반을 잘라줍니다.</li>
 <li>김발위에 김밥용김을 한장을 깔아줍니다.</li>
 <li>김위에 밥을 얇게 잘펴서 올려줍니다.</li>
 <li>밥위에 김1/2를 잘랐던것을 올려줍니다.

이렇게 김을 올려주면 채소의 수분들이 스며들지 않는다고 해요^^</li>
 <li>깻잎을 두장을 김위에 올린다음 참치속을 올려줍니다.</li>
 <li>그리고 단무지1개,맛살1개,햄1개를 올려주세요.

여기서 참치와 깻잎은 살짝 둥굴게 말아줍니다.</li>
 <li>채썰어놓았던 당근+계란+오이를 올려줍니다.</li>
 <li>쌀때 몸쪽으로 당기면서 단단히 눌러주세요

 

그리고 반정도 접히면 다시 앞쪽으로 당기어서 동글게 말아주시면 완성입니다.</li>
 </ol>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
