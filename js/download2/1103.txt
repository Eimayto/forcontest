<meta property="og:description" content="추위에 몸이 움추러들 때에는 매운 양념의 닭볶음탕을 먹는 게 제격이다. 닭볶음탕만 생각해도 특유의 얼큰달달한 맛이 생각나서 군침이 절로 난다. 끓일수록 닭에 양념이 베어들어 두끼 정도 반찬걱정 안생기는 효자반찬이다. 닭볶음탕 레시피가 너무 다양하고, 닭갈비로 변질되기도 하는 은근히 어려운 닭볶음탕을 만레에서 소개한다. 1.[집밥백선생레시피]백종원 닭볶음탕 레시피,집밥 닭볶음탕 by.꽃청춘이주부 분량 :3인분 조리시간 :60분 이내 난이도 :아무나 [주재료] 닭 10호 (1kg) 1마리, 감자 2개, 양파 1개, 당근 1/2개, 청 고추 3개, 홍 고추 1개, 대파 1개, 새송이버섯 2개, 표고버섯 4개 [양념 및 소스재료] 설탕 3T, 마늘 1T, 간장 1컵 (살짝 모자르게), 고춧가루 1/2컵, 후추 조금 [요리순서] 재료:닭10호(1kg),감자2개,양파1개,당근1/2개,고추3개,홍고추1개,대파1개,새송이버섯2개,표고버섯4개 양념재료:설탕3스푼,마늘1스푼,간장1컵(살짝 모자르게),고추가루반컵,후추조금 우선 닭을 손질해줄께요. 닭내장이 닭의 등뼈쪽에 붙어 있는 내장을 깨끗하게 손질해줍니다. 닭내장이 잡내와 지저분한 거품을 유발한다고 하네요.. 저는 여기서 닭가슴살을 우유를넣고,허브솔트조금,다진마늘1스푼을 넣고 20분정도 재워둔걸 사용했습니다. (재워둔뒤에는 따로 우유를 헹구거나 하지 않고 바로 사용했어요) 여기서 가위를 이용해서 넓게 펴주는 방식으로 하면 간이 골구로 배어서 맛있다고 하네요. 표고버섯4개,새송이버섯2개,감자2개,양파1개,대파1개,고추3개,홍고추1개를 닭크기와 비례하도록 손질합니다. 닭이 작으면 채소도 작게~ 닭이 크면 채소도 큼직하게~ 냄비에 물을 3컵을 준비한뒤 손질된 닭을 넣어줍니다. 여기서팁! 설탕3스푼을 넣어줍니다. 손질한 닭과 물을 넣고 끓일때 설탕을 먼저 넣어주는게 포인트라고 하네요~ 닭에 설탕맛이 먼저베면 나중에 다른 양념을 해도 양념이 잘배고 첫맛이 설탕맛이라고 맛있게 느껴진다고 합니다. 중간에 올라오는 나쁜 거품은 틈틈히 건져서 버려주세요. 1차로 감자,양파,당근을 넣어줍니다. 뽀얀 국물이 나올때까지 야채들이 다 익기직전까지 끓여줍니다. 여기에 마늘 한스푼을 넣어줍니다. 간마늘을 넣어야 잡내가 없어지고~ 잡내가 날아가도록 뚜껑을 덮지 않고 끓여주세요. 간장1컵(살짝 모자르게 )준비하여 끓고 있는 닭볶음탕에 넣어줍니다. 짠맛이 충분히 스며들게 끓여줍니다. ​어느정도 끓으면 썰어두었던 새송이와, 표고버섯을 넣어줍니다. 한소큼 끓이다가, 고추가루 반컵을 준비하여 넣어줍니다. ​고추가루는 넣어가면서 양을 조절하여 색깔을 보면서 맞추어주는게 좋다고 하네요. ​ 저는 반컵을 넣으라고 해서 다넣었더니 너무 뻘건색이 나와버려서 ㅎㅎㅎㅎ​ 마무리로 대파와,고추를 넣고 한번더 끓이다가~ 후추를 넣어서 마무리 하면 백종원님표 닭볶음탕 완성! 레시피 상세보기 2.묵은지 닭볶음탕 만들기 by.밍밍뿡 조리시간 :30분 이내 난이도 :초급 [주재료] 닭 1마리(중), 묵은지 1/4포기, 감자 3개, 양파 1/2개, 청양고추 3개, 다진마늘 1스푼, 대파 1뿌리, 소금 조금 [양념 및 소스재료] 고추장 2스푼, 고추가루 3스푼, 올리고당 2스푼, 간장 2스푼, 후추 조금 [요리순서] 닭볶음탕에 들어갈 야채를 큼직하게 듬성듬성 썰어 주세요. 닭은 깨끗이 씻어 물을 넣고, 소금도 조금 넣어 10분 정도 삶아 주세요. 살짝 삶아진 닭은 깨끗이 한번 더 씻어 물기를 빼 주시고, 양념을 분량 대로 넣어 주세요. 묵은지가 들어가니 간을 약하게 해 주세요. 넣은 양념을 조물조물 섞어 주세요. 물을 닭이 잠길 정도로 자작히 부어 주세요. (육수를 만들어서 넣으시면 더 좋아요~) 닭과 물을 넣어 끓여 주다가 묵은지를 넣어 지글지글 한번 더 끓여 주세요. 썰어 놓은 야채를 모두 넣고 끓여 주세요. 청양고추는 양을 조절해서 넣어 주시면 됩니다~ 레시피 상세보기 3.치킨카레 NoNo 카레닭볶음탕! by.폭탄공주 분량 :2인분 조리시간 :60분 이내 난이도 :초급 [주재료] 닭한마리 600g, 감자 1개, 양파 1개, 당근 조금 [양념 및 소스재료] 우유 2컵, 카레가루 1봉지, 고추장 1T, 케첩 1T [요리순서] 닭은 찬물에 깨끗하게 씻어서 우유에 재웁니다. 육질을 부드럽게 해주고 비린내를 없애줘요. 닭을 재우는 동안 감자, 양파, 당근도 잘라둡니다. 닭은 끓는 물에 한번 데쳐내어 불순물을 제거합니다. 채소와 닭고기를 볶다가 감자가 익을무렵 물을 넣고 끓입니다. 카레는 강한맛이 나니 따로 육수는 사용하지 않았어요. 끓기를 기다리며 양념을 만듭니다. 카레가루 한봉지를 물에 잘 갠 뒤 고추장 1T를 넣어 섞어주면 끝! 카레가루 한봉지에 물은 총 3컵(종이컵 기준) 정도가 적당하지만 농도는 취향에 맞게 만드는게 좋은 듯해요. 닭이 익었다 싶으면 양념을 넣고 끓입니다. 화르륵 끓인 뒤 토마토 케첩을 1T 넣어줍니다. 풍미가 살아요. 그리고 양념의 맛이 닭에 쏙 배도록 약한 불에서 졸여주면 완성입니다. 레시피 상세보기 " />

var _blogIsSubmit = false;
function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);
        }
     } else {
        getBlogContentsList();
     }
}
function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}
function setBlogContents(q_link) {
    if (typeof q_link == 'undefined' || q_link == '') {

        if (!q_link) {
            alert('블로그 컨텐츠를 선택하세요.');
            return;
        }
    }
    $("#btnSetBlogContents").html('<span class="fa fa-spinner fa-spin" style="width:' + $("#btnSetBlogContents").width() + 'px"></span>');
    if(fromOther == 'n') document.location.href = '/inbox/insert.html?ref_blog_url=' + encodeURIComponent(q_link);
    else if(fromOther == 'review') document.location.href = '/event/ins_review.html?seq=1103&review_seq=&ref_blog_url=' + encodeURIComponent(q_link);
}

}

function doCommentSubmit()
{
    var txtVal = $("#cmtTextarea").val();
    if(!txtVal)
    {
        alert('의견을 입력해주세요.');
        return ;
    }
    if(txtVal.length < 4)
    {
        alert('의견은 4자 이상으로 작성해 주세요.');
        return ;
    }
    $.ajax({
        type: "POST",
        url: "/bbs/ajax.html",
        data : $("#insCmtFrm").serialize(),
        success: function(result) {
            if(result == 'SUCCESS')
            {
                $("#cmtTextarea").val('');
                getCommentList();
                var ccnt = parseInt($("#recipeCommentListCount").html(),10);
                $("#recipeCommentListCount").html(ccnt+1);
            }
            else if(result == 'NO_PARAM')
            {
                alert('파라미터 부족. 다시 확인해 주세요.')
            }
            else
            {
                alert('등록 실패! 잠시후 다시 시도해 주세요.')
            }
        },
        error: function () {
            alert('의견 조회 실패! 잠시후 다시 시도해 주세요.')
        }
    });
}

}

 <h1><b>1.[집밥백선생레시피]백종원 닭볶음탕 레시피,집밥 닭볶음탕</b>

 <div class="recipe_info">
 <b>분량 :</b>3인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>아무나 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/12/22/d0502fc2ee7267c1499bfb1267d050461.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>닭 10호 (1kg) 1마리, 감자 2개, 양파 1개, 당근 1/2개, 청 고추 3개, 홍 고추 1개, 대파 1개, 새송이버섯 2개, 표고버섯 4개</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>설탕 3T, 마늘 1T, 간장 1컵 (살짝 모자르게), 고춧가루 1/2컵, 후추 조금</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>재료:닭10호(1kg),감자2개,양파1개,당근1/2개,고추3개,홍고추1개,대파1개,새송이버섯2개,표고버섯4개

 

양념재료:설탕3스푼,마늘1스푼,간장1컵(살짝 모자르게),고추가루반컵,후추조금</li>
 
 <li>우선 닭을 손질해줄께요.

 

닭내장이 닭의 등뼈쪽에 붙어 있는 내장을 깨끗하게 손질해줍니다.

 

닭내장이 잡내와 지저분한 거품을 유발한다고 하네요..

 

 

저는 여기서 닭가슴살을 우유를넣고,허브솔트조금,다진마늘1스푼을 넣고 20분정도 재워둔걸 사용했습니다.

(재워둔뒤에는 따로 우유를 헹구거나 하지 않고 바로 사용했어요)

 

 

여기서 가위를 이용해서 넓게 펴주는 방식으로 하면 간이 골구로 배어서 맛있다고 하네요.</li>
 
 <li>표고버섯4개,새송이버섯2개,감자2개,양파1개,대파1개,고추3개,홍고추1개를 닭크기와 비례하도록 손질합니다.

 

닭이 작으면 채소도 작게~

닭이 크면 채소도 큼직하게~</li>
 
 <li>냄비에 물을 3컵을 준비한뒤 손질된 닭을 넣어줍니다.</li>
 
 <li>여기서팁! 

 

설탕3스푼을 넣어줍니다.

 

손질한 닭과 물을 넣고 끓일때 설탕을 먼저 넣어주는게 포인트라고 하네요~

닭에 설탕맛이 먼저베면 나중에 다른 양념을 해도 양념이 잘배고 첫맛이 설탕맛이라고 맛있게 느껴진다고 합니다.</li>
 
 <li>중간에 올라오는 나쁜 거품은 틈틈히 건져서 버려주세요.</li>
 
 <li>1차로 감자,양파,당근을 넣어줍니다.

 

 

뽀얀 국물이 나올때까지 야채들이 다 익기직전까지 끓여줍니다.</li>
 
 <li>여기에 마늘 한스푼을 넣어줍니다.

 

간마늘을 넣어야 잡내가 없어지고~ 잡내가 날아가도록 뚜껑을 덮지 않고 끓여주세요.</li>
 
 <li>간장1컵(살짝 모자르게 )준비하여 끓고 있는 닭볶음탕에 넣어줍니다.</li>
 
 <li>짠맛이 충분히 스며들게 끓여줍니다.</li>
 
 <li>​어느정도 끓으면 썰어두었던 새송이와, 표고버섯을 넣어줍니다.</li>
 
 <li>한소큼 끓이다가,</li>
 
 <li>고추가루 반컵을 준비하여 넣어줍니다.</li>
 
 <li>​고추가루는 넣어가면서 양을 조절하여 색깔을 보면서 맞추어주는게 좋다고 하네요.

 <li>마무리로 대파와,고추를 넣고 한번더 끓이다가~</li>
 
 <li>후추를 넣어서 마무리 하면 백종원님표 닭볶음탕 완성!</li>
 </ol>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
