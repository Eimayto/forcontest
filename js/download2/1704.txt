<meta property="og:description" content="평소에 기름지고 자극적인 음식을 드신다면, 머릿결 상태를 살펴볼 필요가 있는데요.머릿결이 나빠 속상한 분들을 위해 머릿결 개선에 도움이 되는 요리를 소개합니다. 1.석류청, 석류과일샐러드, 석류주스 만들기 by.가시장미 분량 :3인분 조리시간 :90분 이내 난이도 :아무나 [주재료] 석류청만들기 : 석류3개, 설탕(석류양의 80%), 열탕소독한 유리병... 석류과일샐러드 : 석류알, 각종 과일(귤, 배, 사과, 노랑파프리카, 빨강파프리카, 무순등등), 석류청, 레몬즙... 석류주스 : 석류알적당량 [요리순서] 알알이 떼어낸 석류알은 물에 씻어 체반에 받쳐 물기를 제거해주세요. 물기제거한 석류알에 설탕을 넣고 섞어주세요. 이때, 설탕은 유리병에 넣었을때 윗부분에 올릴 설탕은 남겨두세요. 설탕에 버무린 석류알을 열탕소독한 유리병에 담고, 윗부분에는 미리 남겨두었던 설탕을 올려주세요. 석류청은 설탕이 다 녹을때까지 실온에 두었다가 설탕을 다 녹으면 2~3달정도 숙성한뒤 청만 걸러 냉장고에 보관하심 되요. 요 석류청은 만든지 5일정도 된 석류청이에요. 빛깔이 참 곱지요? 맛있는 과일들과 석류청으로 석류과일샐러드를 만들어볼께요. 과일들은 취향에 맞게 썰어주심 되요. 샐러드접시에 얇게 슬라이스한 귤을 담고 그 위에 채썰어 둔 과일들을 올려주세요. 무순은 서로 교차해서 올려주시면 더 이뻐요~ ^ ^ 마지막으로 석류알을 올려주세요. 석류드레싱으로 사용할 석류청이에요.요기에다 레몬즙만 살짝 섞어주기만해도 간단 석류청드레싱 완성. 요건 기호에 맞게 만드시면 될듯해요~ 석류알, 각종 과일, 무순을 이쁘게 올린 과일샐러드접시에 석류청 드레싱을 먹기직전에 뿌려주세요. 석류알을 알알이 다 분리해서 깨끗하게 씻어 물기를 제거한뒤 원액기로 착즙하심 되요. 고운 빛깔만큼이나 맛도 넘 좋아요. 레시피 상세보기 2.생생정보 오이미역냉국 황금레시피 by.세미짱 분량 :2인분 조리시간 :15분 이내 난이도 :아무나 [주재료] 오이 1개, 마른미역 한줌 20g, 물 600ml, 통깨 조금, 얼음 [양념 및 소스재료] 소금 1T, 설탕 4T, 식초 6T [요리순서] 마른미역을 찬물에 담가 10분간 불려주세요. 오이미역냉국 황금레시피 6146 대로 국물을 만들어주세요. 물 600ml에 소금 1T, 설탕 4T, 식초 6T를 넣고 저어주세요. 국물을 냉장고에 넣어 놓고 먹기 바로 전에 꺼내서 오이랑 미역을 넣고 먹으면 된답니다. 오이는 소금에 문질러 씻은 후 채썰어놓고 불린 미역은 물기를 꼭 짠 후 송송 썰어주세요. 냉장고에 넣어 놨던 오이미역냉국 국물을 유리볼에 담고 오이채, 미역을 한줌씩 넣어주세요. 시원하게 얼음도 몇개 넣어주고 통깨 솔솔 뿌려주면 아삭아삭하고 새콤달콤 시원한 오이미역냉국 완성이예요. 레시피 상세보기 3.두부야채 계란말이 by.돌하우스 분량 :2인분 조리시간 :15분 이내 난이도 :초급 [주재료] 두부 1/4모, 계란 3개, 양파(중) 1/2개, 당근, 게맛살, 파 [양념 및 소스재료] 소금 1/3스푼, 통깨 1/2스푼 [요리순서] 두부 1/4모를 으깬후 볼에 담아줍니다. 모든야채는 동량씩 잘게 다져서 사용합니다. 모든재료를 함께 담아서 계란 3개와 소금, 통깨를 넣고 섞어줍니다. 식용유를 약간 두른 팬에 넓게 펴서 부칩니다. 돌돌 말아주면 완성입니다. 레시피 상세보기 4.생꿀로 만든 호두조림 by.서현마미 난이도 :아무나 [주재료] 호두 100g, 은행 15개, 잣 1Ts [양념 및 소스재료] 간장 2ts, 맛술 1Ts, 참기름 1ts, 물 1Ts, 꿀 2Ts, 통깨 약간 [요리순서] 은행의 딱딱한 껍질을 쉽게 까는 방법~ 다 먹고 깨끗이 씻어 말린 우유팩을 준비합니다. 거기에 은행을 넣어서 전자레인지에 1분 30초~ 2분정도 돌려주세요. (은행을 넣고는 뚜껑을 잘 봉해주셔야 해요.) 돌리다보면 따딱~ 하면서 은행이 튀고 껍질이 벌어지는 소리가 날꺼예요. 조금 있다가 꺼내면 은행이 조금씩 벌어져 있어요. 그럼 손으로 쉽게 겉껍질을 깔수 있답니다. 은행도 다 익은 상태라서 바로 먹어도 되구요~ 겉껍질을 까낸 다음, 물을 살짝 묻힌 키친타올에 은행을 올려서 살살 문질러가면서 속껍질을 벗겨주세요. 호두는 전처리를 해주면 좋아요. 끓는 물에 호두를 넣어서 2~3분정도 끓여주면, 물이 뿌옇게 되면서 불순물들이 나와요. 불에서 내려서 체에 걸러주고 흐르는 물에서 여러번 헹구어주세요. 키친타올로 물기를 닦아주고, 기름없이 달군 팬에 넣어서 볶아서 준비해주시면 되어요. 이렇게하면 깔끔하고 더 고소한 호두를 맛보실수가 있답니다. 꿀과 통깨를 제외한 양념재료들을 팬에 넣어서 한번 끓여주세요. 끓으면 호두, 은행, 잣을 넣어서 중약불에서 섞어가며 졸여줍니다. 어느정도 졸아서 간장국물이 거의 없어지면 꿀을 넣어서 고루 섞어가며 볶아주세요. 내용물에 윤기가 돌면 불에서 내려 통깨를 뿌려 섞어주면 완성이예요. 레시피 상세보기 " />

var _blogIsSubmit = false;
function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);
        }
     } else {
        getBlogContentsList();
     }
}
function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}
function setBlogContents(q_link) {
    if (typeof q_link == 'undefined' || q_link == '') {
        q_link = $("#listBlog [id^=q_link_]:checked").val();

        if (!q_link) {
            alert('블로그 컨텐츠를 선택하세요.');
            return;
        }
    }
    $("#btnSetBlogContents").html('<span class="fa fa-spinner fa-spin" style="width:' + $("#btnSetBlogContents").width() + 'px"></span>');
    if(fromOther == 'n') document.location.href = '/inbox/insert.html?ref_blog_url=' + encodeURIComponent(q_link);
    else if(fromOther == 'review') document.location.href = '/event/ins_review.html?seq=1704&review_seq=&ref_blog_url=' + encodeURIComponent(q_link);
}

        $("#top_blog_select_id").show();
    }

}

function doCommentSubmit()
{
    var txtVal = $("#cmtTextarea").val();
    if(!txtVal)
    {
        alert('의견을 입력해주세요.');
        return ;
    }
    if(txtVal.length < 4)
    {
        alert('의견은 4자 이상으로 작성해 주세요.');
        return ;
    }
    $.ajax({
        type: "POST",
        url: "/bbs/ajax.html",
        data : $("#insCmtFrm").serialize(),
        success: function(result) {
            if(result == 'SUCCESS')
            {
                $("#cmtTextarea").val('');
                getCommentList();
                var ccnt = parseInt($("#recipeCommentListCount").html(),10);
                $("#recipeCommentListCount").html(ccnt+1);
            }
            else if(result == 'NO_PARAM')
            {
                alert('파라미터 부족. 다시 확인해 주세요.')
            }
            else
            {
                alert('등록 실패! 잠시후 다시 시도해 주세요.')
            }
        },
        error: function () {
            alert('의견 조회 실패! 잠시후 다시 시도해 주세요.')
        }
    });
}

}

 <h1><b>1.석류청, 석류과일샐러드, 석류주스 만들기</b>
 <span>by.가시장미</span>
 <div class="recipe_info">
 <b>분량 :</b>3인분 <b>조리시간 :</b>90분 이내 <b>난이도 :</b>아무나 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/12/07/224a17d6445eb19e287a4c8e103a81a01.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>석류청만들기 : 석류3개, 설탕(석류양의 80%), 열탕소독한 유리병... 석류과일샐러드 : 석류알, 각종 과일(귤, 배, 사과, 노랑파프리카, 빨강파프리카, 무순등등), 석류청, 레몬즙... 석류주스 : 석류알적당량</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>알알이 떼어낸 석류알은 물에 씻어 체반에 받쳐 물기를 제거해주세요.</li>
 <li>물기제거한 석류알에 설탕을 넣고 섞어주세요. 이때, 설탕은 유리병에 넣었을때 윗부분에 올릴 설탕은 남겨두세요.</li>
 <li>설탕에 버무린 석류알을 열탕소독한 유리병에 담고,</li>
 <li>윗부분에는 미리 남겨두었던 설탕을 올려주세요.</li>
 <li>석류청은 설탕이 다 녹을때까지 실온에 두었다가 설탕을 다 녹으면 2~3달정도 숙성한뒤 청만 걸러 냉장고에 보관하심 되요.</li>
 <li>요 석류청은 만든지 5일정도 된 석류청이에요. 빛깔이 참 곱지요? 맛있는 과일들과 석류청으로 석류과일샐러드를 만들어볼께요.</li>
 <li>과일들은 취향에 맞게 썰어주심 되요.</li>
 <li>샐러드접시에 얇게 슬라이스한 귤을 담고 그 위에 채썰어 둔 과일들을 올려주세요.</li>
 <li>무순은 서로 교차해서 올려주시면 더 이뻐요~ ^ ^</li>
 <li>마지막으로 석류알을 올려주세요.</li>
 <li>석류드레싱으로 사용할 석류청이에요.요기에다 레몬즙만 살짝 섞어주기만해도 간단 석류청드레싱 완성. 요건 기호에 맞게 만드시면 될듯해요~</li>
 <li>석류알, 각종 과일, 무순을 이쁘게 올린 과일샐러드접시에 석류청 드레싱을 먹기직전에 뿌려주세요.</li>
 <li>석류알을 알알이 다 분리해서 깨끗하게 씻어 물기를 제거한뒤 원액기로 착즙하심 되요. 고운 빛깔만큼이나 맛도 넘 좋아요.</li>
 </ol>
 </dd>
 </dl>

 <h1><b>2.생생정보 오이미역냉국 황금레시피 </b>
 <span>by.세미짱</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>15분 이내 <b>난이도 :</b>아무나 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/06/25/4bea879075f7f862c30070a1affd9f611.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>오이 1개, 마른미역 한줌 20g, 물 600ml, 통깨 조금, 얼음</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>소금 1T, 설탕 4T, 식초 6T</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>마른미역을 찬물에 담가 10분간 불려주세요.</li>
 <li>오이미역냉국 황금레시피 6146 대로 국물을 만들어주세요. 물 600ml에 소금 1T, 설탕 4T, 식초 6T를 넣고 저어주세요.</li>
 <li>국물을 냉장고에 넣어 놓고 먹기 바로 전에 꺼내서 오이랑 미역을 넣고 먹으면 된답니다.</li>
 <li>오이는 소금에 문질러 씻은 후 채썰어놓고 불린 미역은 물기를 꼭 짠 후 송송 썰어주세요.</li>
 <li>냉장고에 넣어 놨던 오이미역냉국 국물을 유리볼에 담고 오이채, 미역을 한줌씩 넣어주세요.</li>
 <li>시원하게 얼음도 몇개 넣어주고 통깨 솔솔 뿌려주면 아삭아삭하고 새콤달콤 시원한 오이미역냉국 완성이예요.</li>
 </ol>
 </dd>

 <h1><b>4.생꿀로 만든 호두조림</b>
 <span>by.서현마미</span>
 <div class="recipe_info">
 <b>난이도 :</b>아무나 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/04/23/7955ee11cc5785d18ebcb1f4c8c941ce1.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>호두 100g, 은행 15개, 잣 1Ts</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>간장 2ts, 맛술 1Ts, 참기름 1ts, 물 1Ts, 꿀 2Ts, 통깨 약간</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>은행의 딱딱한 껍질을 쉽게 까는 방법~ 다 먹고 깨끗이 씻어 말린 우유팩을 준비합니다. 거기에 은행을 넣어서 전자레인지에 1분 30초~ 2분정도 돌려주세요. (은행을 넣고는 뚜껑을 잘 봉해주셔야 해요.) 돌리다보면 따딱~ 하면서 은행이 튀고 껍질이 벌어지는 소리가 날꺼예요. 조금 있다가 꺼내면 은행이 조금씩 벌어져 있어요. 그럼 손으로 쉽게 겉껍질을 깔수 있답니다. 은행도 다 익은 상태라서 바로 먹어도 되구요~ 겉껍질을 까낸 다음, 물을 살짝 묻힌 키친타올에 은행을 올려서 살살 문질러가면서 속껍질을 벗겨주세요.</li>
 <li>호두는 전처리를 해주면 좋아요. 끓는 물에 호두를 넣어서 2~3분정도 끓여주면, 물이 뿌옇게 되면서 불순물들이 나와요. 불에서 내려서 체에 걸러주고 흐르는 물에서 여러번 헹구어주세요. 키친타올로 물기를 닦아주고, 기름없이 달군 팬에 넣어서 볶아서 준비해주시면 되어요. 이렇게하면 깔끔하고 더 고소한 호두를 맛보실수가 있답니다.</li>
 <li>꿀과 통깨를 제외한 양념재료들을 팬에 넣어서 한번 끓여주세요. 끓으면 호두, 은행, 잣을 넣어서 중약불에서 섞어가며 졸여줍니다.</li>
 <li>어느정도 졸아서 간장국물이 거의 없어지면 꿀을 넣어서 고루 섞어가며 볶아주세요. 내용물에 윤기가 돌면 불에서 내려 통깨를 뿌려 섞어주면 완성이예요.</li>
 </ol>
 </dd>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
