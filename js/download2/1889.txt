function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {

        } else {

function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}

 <h1><b>8.봄동골뱅이무침&amp;소면</b>
 <span>by.밥심은국력</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>15분 이내 <b>난이도 :</b>초급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2016/03/27/898c1204725ebd0ab1bffccd651229871.jpg">
 </h1>


 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>봄동은 한 포기인데, 반 포기만 사용하면 됩니다.</li>
 <li>가닥가닥 떼어서 찬물을 넉넉하게 붓고 식초 1큰술을 넣고 10분 정도 담갔다가 여러번 헹궈줍니다. 봄동을 뒤집어 주면 물기가 더 잘 빠집니다. 그리고 봄동을 세로로 2등분하면 줄기와 잎사귀를 골고루 드실 수 있어 좋답니다.</li>
 <li>청고추와 홍고추는 어슷썰어주시구요. 양파는 곱게 채를 썰어주세요.</li>
 <li>골뱅이는 2등분하고, 골뱅이캔 국물은 양념장에 쓸 것이므로 버리지 말고 남겨주세요.</li>
 <li>양념장은 미리 만들어 두면 양념이 고루 배여서 좋습니다. 고추장 1큰술, 고추가루 2큰술, 까나리 액젓 반큰술, 골뱅이캔 국물 2큰술, 매실청 1큰술, 레몬즙 반큰술, 다진마늘 반큰술, 참기름 반큰술, 통깨 반큰술을 넣고 고루 섞어줍니다.</li>
 <li>소면은 2인분으로 국수를 모아 세워서 둥근 부분이 5백원 동전 크기만큼이면 됩니다.</li>
 <li>1.5리터의 팔팔 끓는 물에 소면을 삶아주는데요. 방사형으로 소면을 넣어주면 저절로 냄비 속으로 국수가 들어갑니다. 긴 젓가락을 냄비 중앙에 올려두면 물이 끓어도 넘치지 않아 좋답니다. 참 신기하니 꼭 해보세요. 쇠젓가락은 뜨거울 수 있으니 주의하시구요. 긴 나무 젓가락이면 더욱 좋겠네요. 갑자기 찾을 수가 없어서 소면을 넣은 뒤 1차로 물이 다시 끓어오르면 찬물 1컵을 부어줍니다. 찬물을 붓는 이유는 온도차로 인해서 국수에 긴장감이 생겨서 국수가 쫄깃해지는 원리입니다. 이 과정을 2번 정도 반복하시면 더욱 좋답니다. 국수가 익은 걸 확인하려면 한 가닥 먹어보는 방법이 제일 편하지요.</li>
 <li>찬물을 틀어 놓고 양손으로 소면을 비벼 가며 전분기를 제거합니다.</li>
 <li>큰 볼에 골뱅이와 양념장을 넣고 먼저 버무려주고 봄동을 버무려줍니다. 그리고 양파채와 고추 등을 넣고 가볍게 버무려줍니다.</li>
 <li>​ 맛있게 무쳐졌네요.</li>
 <li>봄동 골뱅이무침과 소면이 완성되었습니다. 쫄깃하게 잘 삶아진 소면. 막 삶은 소면의 쫄깃함이 느껴지지 않나요? 봄동의 연하고 고소한 맛이 정말 안 먹어보면 몰라요. 쫄깃한 골뱅이 맛은 말할 것도 없구요. ​색다른 골뱅이무침을 찾으신다면 봄동을 넣어 한번 만들어 보세요.</li>
 </ol>

 <h1><b>9.쫄깃함이 2배 골뱅이무침 쫄면</b>
 <span>by.별바라기</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>30분 이내 <b>난이도 :</b>초급 </div>

 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>쫄면사리 2인분, 골뱅이 小 1캔, 오이 1/2개, 당근 조금, 양배추 조금, 열무김치 약간, 삶은 계란 1개, 크래미맛살 1개</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>[양념장] 고추장 2숟갈, 고추가루 1숟갈, 양파즙 3숟갈, 간장 1숟갈, 사이다 1/3컵, 요리당 3숟갈, 다진마늘 0.3숟갈, 식초 4숟갈, 참기름 한숟갈, 통깨 약간</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>가장 먼저 할일이 양념장 만드는 일이에요. 만들어서 냉장고에 넣어뒀다 사용하세요. 고추장 2숟갈, 고추가루 밥수저로 깍아서 한숟갈, 양파즙 3숟갈, 간장 1숟갈, 사이다 1/3컵, 요리당 3숟갈, 다진마늘 0.3숟갈, 식초 4숟갈 넣고 고루고루 잘 섞어주세요. 골뱅이는 캔을 반쯤열어 안에든 충전물을 쪽 따라내시고 골뱅이살만 꺼내 준비해 주세요. 흐르는물에 한번 씻으셔도 되는데 구슬골뱅이의 충전물은 뭉글뭉글하지않고 물같이 걸쭉해서 그냥 씻지않고 사용했어요. 취향껏 준비하세요.</li>
 <li>오이, 당근은 채썰어주시고 양배추는 곱게 채썰어 얼음물에 담궜다가 탱탱하고 신선하게 살려서 물기빼서 준비해주세요.</li>
 <li>쫄면은 끓는 물에 삶아 흐르는물에 치대어 씻어서 물기 쪽 빼서 준비해주세요.</li>
 <li>냉장고에 넣어뒀던 양념장 꺼내 큰 볼에 넣고 볼에 골뱅이 넣고 버무리시다가 쫄면과 야채넣고 같이 버무려 주세요. 버무리시면서 부족한 간은 비비시면서 약간씩 추가시키면 될꺼 같아요. 통깨 약간 뿌리고 참기름 한숟갈정도 넣고 고루고루 버무려 그릇에 담아 맛있게 드심 됩니다.</li>
 </ol>
