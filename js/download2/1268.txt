그리고 너무 격하게 힘을 줘서 하시면 굴의 살이 찢어지지 조물조물 살살 해주셔요. 1~2분정도 해주시면 이렇게 검은물이 나오는데요.물은 버려주시고 다시 찬물에서 2~3번정도 헹궈주시면 된답니다.​ 부침옷에 들어갈 재료는 당근과 대파에요. 색감을 예쁘게 만들어주기 위해서이니 재료는 잘게잘게 썰어서 준비해주세요. 이제 부침옷을 입혀주는 과정인데요. 가루채 하나하나씩 뭍혀서 하시기 참 번거롭더라구요. 그래서 비닐팩을 이용해서 부침옷을 입혀보기로 했어요. ​카레가루 1/2스푼정도 를 넣어서 하시면 은은한 카레향과 간이 살짝 되어서 비린냄새도 나지 않고 맛있어요. 그리고 간은 허브솔트로 해주셔도 되지만, 굳이 간을 하지 않아도 삼삼하니 먹기에 괜찮더라구요. ​ ​ 비닐팩에 부침가루 약 5스푼, 카레가루 1/2스푼을 넣고 잘 섞어주신후에 손질한 굴을 넣고 봉지채 흔들어서 고루게 가루옷을 입혀주세요. 이렇게 가루옷을 입히면 손으로 뭍혔을때보다 부침옷이 두껍게입혀져요. 멍울을 풀은 달걀물에 야채를 넣고 잘 섞어주신후에 부침가루가 잘 입혀진 굴에 계란물을 잘 입혀주세요. 달군팬에 적당량의 식용유를 두르신후에 전을 넣고 앞뒤로 노릇하게 구워주시면 완성이에요. ​ 레시피 상세보기 5.고소한 두부쌈장으로 양배추쌈 싸먹기 by.v쭈양v 분량 :4인분 조리시간 :30분 이내 난이도 :초급 [주재료] 애호박 1/3개, 양파 1/2개, 두부 반모 [양념 및 소스재료] 된장 2큰술, 고추장 1큰술, 쌈장 1/2큰술, 들기름 1큰술 [요리순서] 애호박과 양파는 잘게 다져주시고 들기름1큰술 둘러 팬에 볶아주세요 양파가 투명해지면 물한컵을 붓고 분량의 된장, 고추장, 쌈장을 넣은뒤 다진 두부도 넣어 으깨면서 끓여주세요 국물이 자작해질정도로 끓여내면 완성 레시피 상세보기 " />

function getBlogContentsList(page,added_params) {

    }

    }

        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;

    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);

function getViewRecipeList(page)
{

        $("#rRecipContDivPage_"+page).show();

        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
