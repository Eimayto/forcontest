 <li>감자, 양파, 청피망, 홍파프리카는 잘게 썰어 준비해 줍니다.</li>
 <li>냄비에 물, 메추리알, 소금, 식초를 넣고 10분만 삶아 껍질을 벗겨 줍니다. 냄비에 삶은 메추리알과 간장 1/4컵, 설탕 1큰술, 청주 1큰술, 후추를 넣고 조려 줍니다.</li>
 <li>국물이 1/3쯤 남아 있을때 물엿과 설탕을 넣어 짭조름하면서 달콤하게 만들어 줍니다. (물엿과 설탕의 양은 입맛에 맛게 조절해 주세요.) 다시 자작하게 조려지면 참기름 넣고 불을 꺼 줍니다. (밥볼에 넣는거라 짭조름하게 만들어 보았답니다. 짭조름한게 싫으시면 간장의 양을 줄여 주세요.)</li>
 <li>팬에 기름을을 두르고 양파, 감자를 넣고 볶아 줍니다.</li>
 <li>감자와 양파가 다 익어가면 청피망, 홍파프리카를 넣고 소금으로 간을 맞추어 불을 꺼 줍니다. 밥볼을 만들거라 청, 홍파프리카가 숨이 죽을때까지 볶아 주세요.</li>
 <li>계란도 소금간 하여 스크램블에그를 만들어 줍니다.</li>
 <li>밥에 볶은 재료들을 넣고 비벼 주세요. 랩을 깔고 밥을 놓고 그 위에 메추리알을 놓은 다음 동그랗게 감싸 줍니다. 힘있게 랩을 꽉 돌린 다음 동그랗게 만들어 주시면 완성된답니다. (밥볼을 만들때 메추리알에 간장이 많이 묻어 있으면 밥볼이 만들어지지 않는답니다. 간장은 깨끗한 마른 면보나 키친타월로 살짝 닦은 다음 밥볼을 만들어 주세요.)</li>
 <li>밥볼 위에 삶아 둔 깻잎을 감싸 줍니다. 밥 위에 계란을 놓고 랩으로 감싼 다음 밥과 계란이 밀착될 수 있게 랩을 꽉꽉 돌려 동그랗게 만들어 줍니다. 포를 떤 게맛살도 밥위에 올린 다음 랩으로 감싸 꽉꽉 돌려 줍니다.</li>
 </ol>
 </dd>

 <h1><b>4.백종원 스팸주먹밥, 어떤 속재료를 넣어도 OK</b>
 <span>by.요리가좋아in독일</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>15분 이내 <b>난이도 :</b>아무나 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/09/23/7e6d52c8aeee3f6bc595b2bb8ae848101.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>밥 2공기, 스팸 1/3, 계란 2개, 참치 1캔 (작은), 샐러드 야채 한 줌, 김치 혹은 볶은김치 반컵(종이컵기준), 도시락 김 1개, 치즈 2장</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>고추장 1t, 후추 1t, 참기름 혹은 들기름 1T, 소금 조금, 파 조금 (생략가능)</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>재료를 준비해주세요~
스팸을 도톰하게 썰어 준비합니다.</li>
 <li>달걀 2개를 먼저 볼에 풀어줍니다.</li>
 <li>소금 한 꼬집과 채썬 파를 넣고 섞어줍니다.</li>
 <li>약불로 오일을 두른 팬에 달걀을 풀고 앞 뒤로 뒤집어가며 부칩니다.</li>
 <li>참치 작은 캔 하나에 후추 1t, 고추장 1t을 넣고</li>
 <li>섞어 준비합니다.</li>
 <li>달걀부침이 다 되면, 꺼내어 놓고 그대로 스팸을 앞뒤로 노릇하게 볶습니다.</li>
 <li>스팸을 다 익힌 후, 그대로 잘게 자른 김치를 팬에 올리고 참기름 1t 과 설탕 1t, 물 1t 을 넣고 다글다글 볶습니다. 물기가 사라지고 살짝 건조해지면 그릇에 옮겨담아 놓습니다.</li>
 <li>부쳐둔 달걀을 스팸 통에 들어가도록 알맞게 자릅니다.</li>
 <li>모든 속 재료 준비는 끝났습니다.</li>
 <li>밥 두공기를 볼에 담고 참기름 1t, 소금 두꼬집, 밥이랑(생략가능) 넣고 젓가락으로 잘 섞습니다.</li>
 <li>스팸 통 안으로 랩을 꼭 맞게 넣어줍니다.</li>
 <li>김 한장을 넣고 (부숴넣으시는게 먹을 때 더 편할것 같습니다)</li>
 <li>그 위로 밥을 올리고 (1센티정도)</li>
 <li>김치를 올리고</li>
 <li>스팸도 올려줍니다.</li>
 <li>그 위로 야채를 올리고</li>
 <li>밥을 올립니다.</li>
 <li>그리고 김을 넣고 랩을 닫고 다시한번 눌러준뒤 뒤집어 빼시면 됩니다! 
이렇게 세 번 더 하시면 양이 맞습니다. 
고추창 참치 속도 같은 방법으로 하시면 됩니다.</li>
 </ol>
 </dd>
 </dl>

 <h1><b>5.골라먹는 재미~ 주먹밥 삼종세트</b>
 <span>by.서현마미</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>초급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/06/24/9f2df432fb3818c923d5e1e6ac5ec987.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>[재료] 꼬들꼬들한 밥 2공기, 들기름 약간, 가는 소금 2꼬집</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>[주먹밥 속재료] 마른 표고버섯 3장, 당근 1/5개, 무장아찌 다진것 3Ts, 들기름 2Ts, 국간장 1/2ts, 소금 1꼬집, 베이컨 2장, 달걀 2개, 청주 1/2ts, 우유 1ts, 소금 1/4ts, 칵테일새우 6마리, 청주 약간, 후춧가루 약간, 소금 1꼬집, 카레가루 1/3ts</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>무장아찌는 잘게 다져서 준비해둡니다. 마른 표고버섯은 미리 연한 설탕물에 불려서 물기를 꾹 짠 다음 잘게 다져서 준비하고, 당근도 곱게 다져서 준비해두세요.</li>
 <li>달군 팬에 들기름을 부어주고 당근과 표고버섯을 넣어서 볶아주세요. 국간장을 넣어서 볶아주고, 소금으로 간을 맞춰줍니다.</li>
 <li>꼬들꼬들한 밥에 들기름과 가는 소금을 넣어 섞어주면서 간을 맞춰주고, 볶아놓은 재료들과 무장아찌 다진것을 넣어서 살살 섞어주세요. 위에 비닐을 덮어 이대로 잠시 놓아두고,</li>
 <li>달걀을 풀어서 청주, 우유, 소금을 넣어 고루 섞어준 다음, 고운 체에 한번 내려줍니다. 팬에 기름을 살짝 두르고 달걀물을 부어주고, 달걀말이처럼 굴려가며 말아서 달걀 지단을 만들어주세요.</li>
 <li>칵테일새우는 청주를 뿌려 잠시 두었다가 꼬리부분을 손질해줍니다. 물기를 닦아주고,, 달군 팬에 기름을 살짝 둘러서 새우를 넣고 소금, 후춧가루를 뿌려서 익혀주세요. 여기에 카레가루를 넣어서 잘 볶아줍니다. 베이컨은 꼬들꼬들하게 구워서 키친타올에 올려서 기름을 빼주면 되요.</li>
 <li>달걀 지단과 베이컨은 적당한 크기로 잘라주세요. 주먹밥에 올릴꺼니까 한입 크기로~</li>
 <li>만들어둔 밥을 한입 크기로 길쭉하게 만들어서 손으로 단단히 눌러 풀어지지 않게 해주세요. 여기에 고명들을 하나씩 올려주면 주먹밥 삼종세트가 완성되어요. 김을 얇게 잘라서 묶어주면 더 좋아요~</li>
 </ol>
 </dd>
 </dl>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}

    getViewRecipeList(1);
