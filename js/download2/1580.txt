너무 예쁘져? 하트 모양이 너무 이쁘죠? 맛도 계란에 맛살 붙인거니 당연히 맛있습니다. 아이들이 너무 좋아해요. 만들기도 쉽고 모양도 예쁘고 재료도 어디서나 구할 수 있는 손쉬운 재료고 누구나 좋아하는 반찬이고. 꼭 한번 만들어 보세요. 식탁이 화사해 진답니다. 레시피 상세보기 3.든든한 브런치 식빵카나페 by.서현마미 분량 :2인분 조리시간 :15분 이내 난이도 :초급 [주재료] 식빵 3장, 달걀 3개, 슬라이스치즈 3장, 토마토 1/2개, 오이 1/3개 [양념 및 소스재료] 소금 약간, 마요네즈 3Ts, 디종 머스터드 1/3ts, 소금 2꼬집, 후춧가루 약간 [요리순서] 냄비에 달걀을 넣고 물을 채운다음, 소금 약간을 넣어 불에 올려 삶아주세요. 물이 끓기 시작하면 그 후로 12분정도 삶아주시면 되요. 다 삶아지면 불에서 내려 달걀은 곧바로 찬물에 담가 식혀줍니다. 식빵은 원형쿠키커터로 찍어내어 밀대로 살짝 밀어둡니다. 기름없는 팬에 올려서 앞뒤로 노릇하게 구워 식혀두세요. 오이는 돌려깎기해서 가늘게 채썰어주고, 소금을 두어꼬집 넣어서 절여둡니다. 잠시 두었다가 물이 나오면 면보에 싸서 꾹 짜주세요. 그런다음 잘게 다져서 준비해둡니다. 삶아진 달걀은 껍질을 까서 두께 1cm정도로 썰어주세요. 링모양으로 썰어진 흰자 6조각만 남겨둡니다. 나머지 흰자는 잘게 다져줍니다. 노른자는 포크로 곱게 으깨어 줍니다. 으깬 노른자에 소금에 절여 다져둔 오이와 마요네즈, 디종머스터드, 후춧가루, 소금을 넣어서 골고루 섞어주세요. 식빵 위에 원형커터로 찍어낸 치즈를 한장 올려주고, 슬라이스해서 반 자른 토마토를 얹어줍니다. 링모양의 달걀 흰자를 얹어주고, 노른자 믹스를 수북히 얹어주면 든든한 까나페 완성이예요. 레시피 상세보기 4.게맛살 오이 롤 초밥 by.츄츄맘 분량 :1인분 조리시간 :10분 이내 난이도 :아무나 [요리순서] 오이는 깨끗이 씻어서 양쪽을 칼로 썰어 없애주세요. 감자깎는칼을 이용해 얇고 길게 슬라이스 해 주고, 소금을 3~4꼬집 솔솔 뿌려서 조물조물 오이가 분리되거나 망가지지 않게 해서 절여 놓아 줍니다. 그리고 게맛살 5개를 준비 해 주세요. 준비된 게맛살은 북북 찢어 놓아 주시고 마요네즈 2큰술, 허니머스타드 1큰술 넣어 주는데요. 알싸한맛을 좋아하시는 분들은 연겨자 조금 섞어 넣으시거나 허니머스타드가아닌 일반 머스타드 소스를 넣어 주세요. 후추도 약간 뿌려 잘 버무려 주세요. 그리고 초밥인 만큼 밥도 촛물로 따로 양념을 해 줄건데요. 식초 1큰술, 설탕 1큰술 넣고 깨소금도 1큰술 손바닥으로 비비면서 으깨 넣어 주세요. 초밥에 핵심인 밥이 완성되었어요. 양념된 밥은 손으로 조물조물 해서 먹기 좋은 한입크기 사이즈로 만들어주고, 미리 소금에 절여놓았던 오이로 겉을 돌돌 말아주세요. 그 위에 양념해두었던 게맛살을 얹어 주면 끝. 불을 따로 쓰지 않아도 되고, 요리법이 너무나 간단해서 더 좋은 게맛살 오이 롤 초밥. 오이의 시원한 아삭함과 초밥의 조합. 가끔 오이 요리하다 남는 오이 있음 만들어 줘야겠어요. 레시피 상세보기 5.초간단 연어말이 (타르타르와 환상케미) by.모카젯 분량 :2인분 조리시간 :15분 이내 난이도 :아무나 [요리순서] 오이는 깨끗이 씻어 필러를 이용하여 길이로 얇게 슬라이스하여 준비해두세요. 오이는 연어말이에 산뜻한 아삭함을 더해주는 아주 좋은 궁합이구요! 훈제연어 위에 무순, 새싹, 채소잎 등을 올리고 돌돌돌~떼구르르~ 말아주세요. 기호에 맞는 채소를 사용해보세요 얇게 슬라이스해둔 오이로 한번 더 말아 준비하면 연어말이 완성. 넘 쉽네요 ㅎㅎㅎ 분량의 재료를 모두 섞어 타르타르소스를 만들어두세요. 식초 대신 레몬즙을 살짝 더해도 좋고요. 접시에 보기 좋게 담고 타르타르소스와 함께 서빙하기! 레시피 상세보기 " />

function getBlogContentsList(page,added_params) {

    }

        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    _blogIsSubmit = true;

    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';

    }

    $.ajax({
        type: "POST",

        url: "/common/ajx_common.html",

            $("#divBlogContentsList").html(html);

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';

    for(var i=skip; i<skip+scale; i++)

        if(!rvs) continue;

    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
