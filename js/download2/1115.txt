표고버섯덮밥 by.밤곰이맘 분량 :2인분 조리시간 :60분 이내 난이도 :초급 [주재료] 표고버섯큰거 6개, 양파 반개, 당근 1/3개, 깻잎 5장, 쇠고기 80g [양념 및 소스재료] 쇠고기양념 - 간장 1, 다진마늘 0.5, 맛술 양념 - 간장 2, 굴소스 1, 설탕 0.5, 참기름 1, 후춧가루 (녹말물) 물 300ml (2인분 기준) [요리순서] 가장먼저 소고기에 밑간을 해줄게요. 쇠고기는 불고기감을 이용하셔도 좋고 여기서는 구이용 등심을 이용하였습니다. 다진마늘 0.5와 간장1, 맛술 1 넣고 미리 재워두었어요. 그다음 야채를 준비해 줍니다. 오늘의 주인공인 표고버섯은 넉넉하게 준비해 주세요. 그다음 팬에 양념해둔 소고기를 볶아줍니다. 고기가 절반정도 익으면 당근과 양파를 먼저볶아주세요. 양파가 투명하게 익으면 오늘의 주인공인 표고버섯을 넣어주세요. 표고버벗을 넣고 물을 자작하게 부어주세요. 저는 300ml 넣었더니 딱 맞더라고요. 물이 보글보글 끓기 시작하면 녹말물을 넣어주는데요. 녹말물은 미리 만들어두면 굳으니 만들기 직전에 넣어주세요. 녹말물 넣고 빨리 섞어주셔야 녹말이 덩어리 지지 않는답니다. 녹말물 (전분가루1.5 : 물1.5 ) 녹말물을 넣으면 걸죽한 상태의 버섯덮밥 소스가 완성이 된답니다. 마지막으로 깻잎 채썰어둔 거 넣어주세요. 걸죽하게 완성된 소스입니다. 마지막으로 후춧가루 참기름 넣어주시면 소스 완성입니다. 레시피 상세보기 " />

function doChangePortal(portal) {

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);

     } else {

function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}

}

 <h1><b>1.입맛없을 때~한그릇 뚝딱~!!일본식 쇠고기 덮밥 [규동]</b>
 <span>by.쉐프cookie</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>30분 이내 <b>난이도 :</b>초급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/10/18/75e72e97dc79ed1253db0cf9b9575e831.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>쇠고기 샤브샤브용 100g, 대파 1/2, 양파 1/4, 마늘 슬라이스 2개, 느타리버섯이나 새송이버섯, 팽이버섯, 청량초 1개, 후추</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>물 250ml, 다시마 1쪽 (엄지손가락 정도의 크기), 가쓰오부시 한줌(엄지,검지,중지 세손가락으로 집은 양), 맛술 1/2스픈, 간장 2스픈, 설탕 1스픈, 혼다시 1/2스픈,</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>물 250ml에 다시마를 넣고 한소끔 끓여줍니다.</li>
 
 <li>물이 한소끔 끓으면 불을 끄고 가쓰오부시를 넣어서 3분정도 담궈 두셨다가 체에 걸러주세요.</li>
 
 <li>체에 거른 다시는 다시 불위에 올리고 간장 2스픈 맛술 1스픈 설탕 1스픈 혼다시 1/2스픈을 넣고 한소끔 더 끓입니다 이젠 소스가 다 만들어 졌네요. 가쓰오부시가 없으시면 그냥 혼다시만 쓰셔도 됩니다..</li>
 
 <li>쇠고기는 그냥 생고기를 사용하지만..불고기밖에 없는 관계로 그냥 촬영은 불고기로 밀어 붙입니다. 맛은 절인고기가 더 좋아요.</li>
 
 <li>야채는 미리 준비해 두세요.</li>
 
 <li>팬에 기름을 살짝 두르시고~고기를 먼저 볶습니다.</li>
 
 <li>고기가 익으면~준비해 둔 야채를 모두 넣고 같이 볶아주세요.</li>
 
 <li>야채가 어느정도 볶아지면 준비해둔 가쓰오국물을 부어서 끓여줍니다.</li>
 
 <li>끓이면서 후추도 넣어주세요.</li>
 
 <li>팽이버섯은 좀 뒤에 넣어주시면 좋습니다.</li>
 
 <li>마지막으로 계란 톡 넣어서 약불에 바글바글 익혀주시면 됩니다.</li>
 
 <li>계란이 좀 깨끗하게 익었으면 참 좋았을텐데,,,하는 아쉬움이</li>
 
 <li>그릇에 밥을 담으시고 위에다 올려주시면 끝입니다.</li>
 </ol>
 </dd>

 <h1><b>2.백종원 콩나물볶음밥</b>
 <span>by.러블리다담</span>
 <div class="recipe_info">
 <b>분량 :</b>1인분 <b>조리시간 :</b>30분 이내 <b>난이도 :</b>초급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/08/23/fec10ac631a9e1bb3f65d5d5a0d90bb81.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>콩나물 2줌, 밥 1공기, 당근 조금, 대파 1/2개, 계란후라이 1개, 통깨, 깻잎 4장, 김가루 약간 <br>
</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>고추장 2스푼, 고춧가루 2스푼, 간장 2스푼, 설탕 2스푼, 맛술 2스푼, 다진마늘 2스푼</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>당근,대파, 깻잎를 잘게 썰어서 준비해주세요.</li>
 
 <li>계란후라이도 미리 만들어주세요.</li>
 
 <li>고추장 2스푼, 고춧가루 2스푼, 간장 2스푼, 설탕 2스푼, 맛술 2스푼, 다진마늘 2스푼을 넣고 양념장을 만들어주세요.</li>
 
 <li>끓는물에 미리 손질한 콩나물을 넣고 약 2~3분정도 뒤집어서 데쳐주시면 되세요.</li>
 
 <li>데쳐주신다음 찬물에 바로 헹궈주시구요. 사용하는 양은 2줌이지만, 데칠때에는 한봉지 모두 데쳐주었어요.</li>
 
 <li>예열된 팬에 기름을 두르신후 야채를 먼저 볶아주세요. 야채 볶는 향이 올라오면 밥과 양념장을 넣고 잘 섞어주신후에</li>
 
 <li>콩나물을 수북히 올려주신후 섞어서 함께 볶아주세요.</li>
 
 <li>이과정까지 볶으시면 완성이지만, 전 추가로 깻잎 잘게 썰은것을 넣었기때문에 더 볶아주었어요.</li>
 
 <li>깻잎까지 넣고 통깨가루 솔솔 뿌려서 볶아주면 백종원 콩나물볶음밥 완성.</li>
 
 <li>계란후라이도 올려주고 김가루도 뿌려주면 완성입니다.</li>
 </ol>
 </dd>

 <h1><b>3.건강 한그릇 요리 - 표고버섯덮밥 </b>
 <span>by.밤곰이맘</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>초급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/05/13/ff062a74c85bfb51495a1780f32237171.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>표고버섯큰거 6개, 양파 반개, 당근 1/3개, 깻잎 5장, 쇠고기 80g </dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>쇠고기양념 - 간장 1, 다진마늘 0.5, <br>
<br>
맛술 양념 - 간장 2, 굴소스 1, 설탕 0.5, 참기름 1, 후춧가루 (녹말물) 물 300ml (2인분 기준)</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>가장먼저 소고기에 밑간을 해줄게요.
 
쇠고기는 불고기감을 이용하셔도 좋고 여기서는 구이용 등심을 이용하였습니다.

 다진마늘 0.5와 간장1, 맛술 1 넣고 미리 재워두었어요.</li>
 
 <li>그다음 야채를 준비해 줍니다. 오늘의 주인공인 표고버섯은 넉넉하게 준비해 주세요.</li>
 
 <li>그다음 팬에 양념해둔 소고기를 볶아줍니다.</li>
 
 <li>고기가 절반정도 익으면 당근과 양파를 먼저볶아주세요.</li>
 
 <li>양파가 투명하게 익으면 오늘의 주인공인 표고버섯을 넣어주세요.</li>
 
 <li>표고버벗을 넣고 물을 자작하게 부어주세요. 저는 300ml 넣었더니 딱 맞더라고요.</li>
 
 <li>물이 보글보글 끓기 시작하면 녹말물을 넣어주는데요. 녹말물은 미리 만들어두면 굳으니 만들기 직전에 넣어주세요.
 녹말물 넣고 빨리 섞어주셔야 녹말이 덩어리 지지 않는답니다. 녹말물 (전분가루1.5 : 물1.5 )</li>
 
 <li>녹말물을 넣으면 걸죽한 상태의 버섯덮밥 소스가 완성이 된답니다. 마지막으로 깻잎 채썰어둔 거 넣어주세요.</li>
 
 <li>걸죽하게 완성된 소스입니다. 마지막으로 후춧가루 참기름 넣어주시면 소스 완성입니다.</li>
 </ol>
 </dd>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}

    getViewRecipeList(1);
})
