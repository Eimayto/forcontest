<meta property="og:description" content="질리지 않는 친숙한 재료인 오이로 여름철 대표 김치 오이소박이를 담그는 방법과 다 드신 후 남은 국물까지 활용한 국수 레시피도 함께 소개합니다. 1.아삭하게 오이소박이 담그는 법 by.윤 분량 :6인분 이상 조리시간 :120분 이내 난이도 :초급 [주재료] 오이 8개, 천일염 [양념 및 소스재료] 부추 1/2단, 양파 1/2개, 홍고추 2개(또는 청양고추 2개), 다진 마늘 1큰술, 멸치액젓 3큰술, 새우젓 2-3큰술, 깨 1큰술, 설탕 2큰술, 매실청 1큰술, 고춧가루 7큰술, 다진 생강 [요리순서] 천일염으로 문질러 세척한 오이는 끝부분을 잘라내고, 길이로 4등분 한후 길이의 2/3지점 까지 열십자로 칼집을 넣어줍니다. 칼집을 넣은 오이를 천일염을 살짝 묻혀줍니다. 이때 오이 겉면에 물기가 살짝 있어야 소금이 붙어요. 간편하게 천일염에 묻힌 오이를 위생비닐에 넣고 봉하여 1시간 가량 절여 줍니다, 이때 중간에 두어번 비닐의 위 아래 위치를 바꿔줍니다. 소금에 절인 오이를 끓는 소금물에 넣고, 물이 바글바글 끓으려 하면 불을 끄고 3분 정도 그대로 방치합니다. 뜨거운 상태의 오이를 헹구지 말고, 바로 체를 받쳐 자연스레 식힙니다. 분량대로 소를 만들어 버무려주세요. 조금 간을 보시고 모자라시면 소금과 설탕으로 기호에 맞게 맞춰 주세요. 식힌 오이는 손으로 꾹 한번 눌러주면 약간의 물기가 나옵니다. 요정도만 수분을 제거해 주셔도 됩니다. 미리 버무려둔 오이소박이 소를 오이 사이에 꽉 채워 주시면 됩니다. 소를 채운 오이를 용기에 꾹꾹 눌러 담은 후 요즘같이 더울때는 실온에 반나절만 두었다가 냉장고에 보관하시며 드시면 됩니다. 레시피 상세보기 2.오이소박이 황금레시피 오이소박이 만들기 by.꽃청춘이주부 분량 :3인분 조리시간 :60분 이내 난이도 :아무나 [요리순서] 재료:오이10개,부추반단,당근1개,양파1개,굵은소금2컵,물 ​ 양념재료:​ 고추가루8스푼,새우젓2스푼,멸치액젓3스푼,설탕2스푼,마늘1스푼,생강가루1/3스푼 ​ (계량은 모두 밥숟가락과 종인컵기준입니다)​ 우선 오이를 굵은소금을 이용해서 겉표면을 깨끗하게 세척해준뒤 양꼭지를 모두 잘라줍니다. 그리고 3등분을 해주세요. 오이소박이는 개인적으로 생각했을때 익으면 맛이 없더라구요 저처럼 10개씩 자주해서 먹는게 가장맛있는거 같아요^^ 오이를 십자모양으로 칼집을 내줍니다. 칼집을 내주실때 밑기둥부분을 1.5cm가량 남겨주고 잘라줍니다 오이를 모두 십자모양으로 칼집을 내주세요. 칼집을 너무 작게 하면 속이 많이 안들어가서 맛이 없고, 그렇다고 칼집을 크게 내면 오이가 다떨어져버리니 주의해주세요^^ 굵은 소금을 종이컵 기준2컵을 준비하시고 물도 준비한후 소금물을 만들어줍니다. 저같은 경우 물의 양은 종이컵기준 10컵정도 준비했어요. 소금물에 오이를 넣고 오이가 잠길정도로 해서 절여줍니다. 시간은 30~40분정도가 좋아요. ​ 30분정도 지나서 오이를 조금 잘라먹어봤을때 짭쪼름하면 알맞게 절여진거예요. 심심한 맛이 나면 좀더 졸여주세요.​ 오이를 절이는 동안 속을 준비할께요. 양파1개를 썰어줍니다.(최대한 얇게) 당근1개를 채깔을 이용해서 썰어주세요 손으로 썰면 아무래도 채칼보다는 두껍게 썰리니까 채칼을 이용하는게 좋아요^^ ​부추 반단정도를 깨끗하게 씻은후에 3-5cm크기로 잘라줍니다. ​ 큰볼에 부추,당근,양파 썬것을 모두 넣어줍니다. 고추가루8스푼,새우젓2스푼,멸치액젓3스푼,설탕2스푼,마늘1스푼,생강가루1/3스푼 을 넣어줍니다. 야채와 양념이 잘섞어 줍니다. 오이가 다 절여졌다면 물로 2번정도 헹구어 준뒤 물기를 빼주세요. 물기가 어느정도 빠지면 십자로 자른 오이 안에 부추속을 넉넉하게 넣어줍니다. 통에 차곡차곡담아주면 오이소박이 완성입니다.! 오이는 아삭할때가 가장맛있죠~^^ 저는 바로 냉장고에 넣어서 바로바로 꺼내서 먹었답니다. 레시피 상세보기 3.여름철에 별미 오이소박이물김치 by.초이맘 분량 :6인분 이상 조리시간 :30분 이내 난이도 :아무나 [요리순서] 오이 10개를 깨끗이 씻어서 준비했어요. 부추는 1/2단 정도 준비해주세요. 오이는 한 4토막 정도로 잘라서 열십자로 칼집을 넣어주세요. 굵은소금을 준비해주세요. 칼집을 넣어 놓은 오이 예 굵은소금 2스푼을 뿌려 놓아요. 5분 정도 뒤에 한번 뒤적 거려 줍니다. 굵은소금 2스푼, 다진 마늘 1+1/2스푼, 고춧가루 3스푼, 매실 3스푼, 당근 약간 준비해주세요. 부추를 잘게 썰어서 위에 준비해놓은 양념을 넣고 버물어주세요. 혼자 찍다 보니 흔들린 것이 있네요. 버물여준 오이 소를 절여 놓은 오이에 소를 넣어주세요. 오이소를 넣고 차곡차곡 통에 담아주세요. 그리고 생수를 넣어주세요. 생수는 저는 2리터 조금 안되게 넣었어요. 식성에 따라 넣으시면 돼요. 맛있는 오이소박이 물김치가 완성되었어요 속을 넣고 남은 부추를 속으로 넣어 놨었어요. 하루 있다가 꺼내보았더니 익을라고 해요. 익으면 냉 국수나 냉면에 부어서 드시면 정말 맛이는 물냉면과 오이소박이 국수 말이가 된답니다. 오이소박이가 익으면 또 국수 말이 한번 소개해 드릴게요. 레시피 상세보기 4.오이소박이국수 - 아웅 맛있어~시원하게 한그릇 뚝딱!! function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);
        }
     } else {
        getBlogContentsList();
     }
}
function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}

}

 <h1><b>2.오이소박이 황금레시피 오이소박이 만들기</b>
 <span>by.꽃청춘이주부</span>
 <div class="recipe_info">
 <b>분량 :</b>3인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>아무나 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2016/04/05/5ea32e312e78b6c6864b5e8a118f72281.jpg">
 </h1>


 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>재료:오이10개,부추반단,당근1개,양파1개,굵은소금2컵,물

​

양념재료:​ 고추가루8스푼,새우젓2스푼,멸치액젓3스푼,설탕2스푼,마늘1스푼,생강가루1/3스푼

​

(계량은 모두 밥숟가락과 종인컵기준입니다)​</li>
 <li>우선 오이를 굵은소금을 이용해서 겉표면을 깨끗하게 세척해준뒤

 

양꼭지를 모두 잘라줍니다.</li>
 <li>그리고 3등분을 해주세요.

 

오이소박이는 개인적으로 생각했을때 익으면 맛이 없더라구요

 

저처럼 10개씩 자주해서 먹는게 가장맛있는거 같아요^^</li>
 <li>오이를 십자모양으로 칼집을 내줍니다.

 

칼집을 내주실때 밑기둥부분을 1.5cm가량 남겨주고 잘라줍니다</li>
 <li>오이를 모두 십자모양으로 칼집을 내주세요.

 

칼집을 너무 작게 하면 속이 많이 안들어가서 맛이 없고,

 

그렇다고 칼집을 크게 내면 오이가 다떨어져버리니 주의해주세요^^</li>
 <li>굵은 소금을 종이컵 기준2컵을 준비하시고 물도 준비한후 소금물을 만들어줍니다.

 

저같은 경우 물의 양은 종이컵기준 10컵정도 준비했어요.</li>
 <li>소금물에 오이를 넣고 오이가 잠길정도로 해서 절여줍니다.

 

시간은 30~40분정도가 좋아요.

​

30분정도 지나서 오이를 조금 잘라먹어봤을때 짭쪼름하면 알맞게 절여진거예요.

심심한 맛이 나면 좀더 졸여주세요.​</li>
 <li>오이를 절이는 동안 속을 준비할께요.

 

양파1개를 썰어줍니다.(최대한 얇게)</li>
 <li>당근1개를 채깔을 이용해서 썰어주세요

 

손으로 썰면 아무래도 채칼보다는 두껍게 썰리니까 채칼을 이용하는게 좋아요^^</li>
 <li>​부추 반단정도를 깨끗하게 씻은후에 3-5cm크기로 잘라줍니다.

​</li>
 <li>큰볼에 부추,당근,양파 썬것을 모두 넣어줍니다.</li>
 <li>고추가루8스푼,새우젓2스푼,멸치액젓3스푼,설탕2스푼,마늘1스푼,생강가루1/3스푼 을 넣어줍니다.</li>
 <li>야채와 양념이 잘섞어 줍니다.</li>
 <li>오이가 다 절여졌다면 물로 2번정도 헹구어 준뒤 물기를 빼주세요.</li>
 <li>물기가 어느정도 빠지면 십자로 자른 오이 안에 부추속을 넉넉하게 넣어줍니다.</li>
 <li>통에 차곡차곡담아주면 오이소박이 완성입니다.!

 

오이는 아삭할때가 가장맛있죠~^^

 

저는 바로 냉장고에 넣어서 바로바로 꺼내서 먹었답니다.</li>
 </ol>
 </dd>

 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>오이는 굵은 소금(1줌)과 함께 움켜쥐고 비벼가며 씻는다.</li>
 <li>길게 열십자(+)로 칼집을 넣는다.</li>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
