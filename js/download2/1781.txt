당근과 양파도 채썰어 준비합니다. 채써는게 힘들지만 채칼로 썰면 금방 부스러지고 볶을때 망가지기 쉬워요. 가능한 칼로 썰어주시는게 좋답니다. 감자볶음 황금레시피 포인트는 바로 이거랍니다. 팔팔 끓는물에 천일염(굵은소금) 한스푼 넣어주고 채썰어 놓은걸 넣어주세요. 미리 한번 데쳐주는게 포인트랍니다~ 딱 2분!!!! 약간 설 익은듯 익어야 기름 넣고 볶을때 부서지지 않는답니다. 삶은 다음 체에 걸러 물을 빼주세요. 감자는 전분을 제거해야 볶을때 타지 않기때문에 물에 10분정도 담가서 전분기를 빼주거나, 이렇게 삶아서 전분기를 제거해주셔야 한답니다. 달궈진 궁중팬에 올리브유를 두르고 당근넣어 볶은다음 양파 넣고~ 어느정도 익은 후 데친 감자를 넣어주세요. 한번 데친거라서 금방 익으니깐 다른 야채들은 미리 다 익힌다음 넣어야 부서지지 않고 금방 볶아낼 수 있답니다. 맛소금 반스푼 넣고 통깨 넣고 마무리합니다. 예쁜 그릇에 담아주면 맛있는 감자볶음 완성. 레시피 상세보기 4.카레감자채볶음 by.러블리다담 분량 :1인분 조리시간 :30분 이내 난이도 :초급 [주재료] 감자 3개, 당근 조금, 양파 1/2개 [양념 및 소스재료] 다진마늘 1/3스푼, 소금 1/2스푼, 물 3스푼, 카레가루 1스푼, 통깨 1스푼, 식용유 [요리순서] 감자는 채 썰어서 찬물에 약 10분정도 담가주세요. 전분끼를 빼주셔야 하니까요. 그사이에 양파와 당근을 채썰어서 준비해주시구요. 감자는 끓는물에 센불에서 약 5~10분정도 사이로 데쳐서 찬물로 헹궈서 채에 받쳐서 물기를 빼주시구요. 예열된 팬에 기름을 두른후 식용유를 충분히 넣은 다음 양파와 당근을 넣고 먼저 볶아주세요. 양파의 볶은향이 올라오기 시작하면 감자와 소금 1/2T스푼을 넣고 볶아주세요. 소금은 티스푼으로 넣었구요. 감자는 미리 데쳐져서 익은 상태이기 때문에 양파와 당근, 카레 1스푼과 물 3스푼을 넣고 섞어가며 볶아주시면 되요. 카레가루는 밥수저 계량 기준이에요. 물을 넣지 않고 카레가루만 넣게 되면 다소 뻣뻣함이 느껴져서 식감이 별로이니 물을 넣어주면 볶은 감자의 식감도 촉촉해지면서 카레가루가 잘 베여요. 완성된 볶음에 통깨가루 조금 뿌려주면 끝. 레시피 상세보기 5.베이컨 감자 볶음 by.시크제이맘 분량 :2인분 조리시간 :10분 이내 난이도 :아무나 [요리순서] 감자는 껍질을 깍고 최대한 얇게 썰어주세요. 모양은 원하시는 스탈로 채를 썰어주셔도 상관없어요. 썰어놓은 감자는 찬물에 두세번 정도 헹궈주세요. 그리고 물기를 제거해 주심 된답니다. 베이컨은 먹기 좋은 크기로 잘라주세요. 달걀은 소금간을 한다음 잘 풀어서 준비해 주심 된답니다. 여기에 후추를 조금 넣어도 좋아요. 팬에 기름을 아주 조금 두른후 감자를 볶아주세요. 여기에 소금간 살짝 해주세요. 베이컨이 간간하니깐 소금은 조금만 넣어주시는게 좋아요. 감자가 어느정도 익으면 베이컨을 넣고 함께 볶아주세요. 감자와 베이컨이 다 익으면 팬 한쪽으로 밀어두고 풀어놓은 계란을 부어주세요. 그리고 젓가락으로 휘휘 젓어가며 스크램블을 만들어 주심 된답니다. 계란이 몽글 몽글 해지면 계란이 몽글 몽글 해지면 볶아놓아던 감자와 베이컨이 잘 섞이게 한번더 볶아주심 된답니다. 완성된 베이컨 감자볶음을 그릇에 담고 파슬리가루를 솔솔 뿌려주세요. 간간하니 아이 밥 반찬으로 아주 딱이예요. 케찹까지 뿌려주면 더욱 맛나요. 요건 입맛에 따라 즐겨주세요. 레시피 상세보기 " />

당근과 양파도 채썰어 준비합니다. 채써는게 힘들지만 채칼로 썰면 금방 부스러지고 볶을때 망가지기 쉬워요. 가능한 칼로 썰어주시는게 좋답니다. 감자볶음 황금레시피 포인트는 바로 이거랍니다. 팔팔 끓는물에 천일염(굵은소금) 한스푼 넣어주고 채썰어 놓은걸 넣어주세요. 미리 한번 데쳐주는게 포인트랍니다~ 딱 2분!!!! 약간 설 익은듯 익어야 기름 넣고 볶을때 부서지지 않는답니다. 삶은 다음 체에 걸러 물을 빼주세요. 감자는 전분을 제거해야 볶을때 타지 않기때문에 물에 10분정도 담가서 전분기를 빼주거나, 이렇게 삶아서 전분기를 제거해주셔야 한답니다. 달궈진 궁중팬에 올리브유를 두르고 당근넣어 볶은다음 양파 넣고~ 어느정도 익은 후 데친 감자를 넣어주세요. 한번 데친거라서 금방 익으니깐 다른 야채들은 미리 다 익힌다음 넣어야 부서지지 않고 금방 볶아낼 수 있답니다. 맛소금 반스푼 넣고 통깨 넣고 마무리합니다. 예쁜 그릇에 담아주면 맛있는 감자볶음 완성. 레시피 상세보기 4.카레감자채볶음 by.러블리다담 분량 :1인분 조리시간 :30분 이내 난이도 :초급 [주재료] 감자 3개, 당근 조금, 양파 1/2개 [양념 및 소스재료] 다진마늘 1/3스푼, 소금 1/2스푼, 물 3스푼, 카레가루 1스푼, 통깨 1스푼, 식용유 [요리순서] 감자는 채 썰어서 찬물에 약 10분정도 담가주세요. 전분끼를 빼주셔야 하니까요. 그사이에 양파와 당근을 채썰어서 준비해주시구요. 감자는 끓는물에 센불에서 약 5~10분정도 사이로 데쳐서 찬물로 헹궈서 채에 받쳐서 물기를 빼주시구요. 예열된 팬에 기름을 두른후 식용유를 충분히 넣은 다음 양파와 당근을 넣고 먼저 볶아주세요. 양파의 볶은향이 올라오기 시작하면 감자와 소금 1/2T스푼을 넣고 볶아주세요. 소금은 티스푼으로 넣었구요. 감자는 미리 데쳐져서 익은 상태이기 때문에 양파와 당근, 카레 1스푼과 물 3스푼을 넣고 섞어가며 볶아주시면 되요. 카레가루는 밥수저 계량 기준이에요. 물을 넣지 않고 카레가루만 넣게 되면 다소 뻣뻣함이 느껴져서 식감이 별로이니 물을 넣어주면 볶은 감자의 식감도 촉촉해지면서 카레가루가 잘 베여요. 완성된 볶음에 통깨가루 조금 뿌려주면 끝. 레시피 상세보기 5.베이컨 감자 볶음 by.시크제이맘 분량 :2인분 조리시간 :10분 이내 난이도 :아무나 [요리순서] 감자는 껍질을 깍고 최대한 얇게 썰어주세요. 모양은 원하시는 스탈로 채를 썰어주셔도 상관없어요. 썰어놓은 감자는 찬물에 두세번 정도 헹궈주세요. 그리고 물기를 제거해 주심 된답니다. 베이컨은 먹기 좋은 크기로 잘라주세요. 달걀은 소금간을 한다음 잘 풀어서 준비해 주심 된답니다. 여기에 후추를 조금 넣어도 좋아요. 팬에 기름을 아주 조금 두른후 감자를 볶아주세요. 여기에 소금간 살짝 해주세요. 베이컨이 간간하니깐 소금은 조금만 넣어주시는게 좋아요. 감자가 어느정도 익으면 베이컨을 넣고 함께 볶아주세요. 감자와 베이컨이 다 익으면 팬 한쪽으로 밀어두고 풀어놓은 계란을 부어주세요. 그리고 젓가락으로 휘휘 젓어가며 스크램블을 만들어 주심 된답니다. 계란이 몽글 몽글 해지면 계란이 몽글 몽글 해지면 볶아놓아던 감자와 베이컨이 잘 섞이게 한번더 볶아주심 된답니다. 완성된 베이컨 감자볶음을 그릇에 담고 파슬리가루를 솔솔 뿌려주세요. 간간하니 아이 밥 반찬으로 아주 딱이예요. 케찹까지 뿌려주면 더욱 맛나요. 요건 입맛에 따라 즐겨주세요. 레시피 상세보기 " />

function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

function getViewRecipeList(page)
{

        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;

        return ;
    }
    totalVRcnt = rvArr.length;
