레시피 상세보기 2.매운 잡채덮밥 by. 시연성연맘 분량 :1인분 조리시간 :90분 이내 난이도 :초급 [주재료] 당면 2/3줌, 양파 1/4개, 오이약간, 청양고추 1개, 느타리버섯 약간, 식용유, 참깨, 굴소스 1큰술 [양념 및 소스재료] [잡채양념]고춧가루 1작은술, 간장 1큰술, 설탕 1작은술, 참기름 2/3큰술, 다진파 2/3큰 [요리순서] 오이는 돌려깍기 한 후 채썰어 준비하고 양파와 청양고추 역시 씨를 뺀 후 가늘게 채썰어 준비합니다. 느타리 버섯은 적당한 크기로 찢어 준비해 주세요. 분량의 잡채양념재료를 섞어 미리 준비해주세요. 굴소스도 1큰술 넣어 함께 섞어주세요. 달궈진 팬에 식용유를 두른 후 채썰어둔 재료들을 모두 넣어 달달 센불에서 볶아주세요. 참고로 오이는 입맛에 따라 가첨하시면 되는데요 오이를 넣으니 아삭함이 살아서 더 맛있게 느껴진답니다. 채소들이 모두 볶아졌다면 여기에 미리 준비해 둔 잡채양념을 넣어 중불에서 한 번 더 볶아주세요. 불려둔 당면을 넣어 1분간 볶아주세요. 이미 충분히 불려둔 상태인지라 가볍게만 볶아내도 당면은 충분히 익어요. 매콤해서 더욱 맛있는 매콤 잡채덮밥 완성이네요. 레시피 상세보기 3.맛있게 매운 닭가슴살 볶음 by. 시연성연맘 조리시간 :30분 이내 난이도 :초급 [주재료] 닭가슴살 2조각, 청양고추 3~4개, 양배추 약간, 통마늘 8개, 다진마늘 1작은술, 고추기름 1작은술(또는 식용유) [양념 및 소스재료] 간장 1작은술, 굴소스 1과 1/3큰술, 핫소스 1큰술, 후추가루 약간, 생강가루 약간, 올리고당 1큰술 [요리순서] 달달 볶아 먹을 꺼라 한입에 쏙 들어가는 크기로 썰어 주었어요. 그리고 매콤함을 더해줄 청양고추는 송송~ 통마늘은 그대로 혹은 너무 큰 마늘의 경우에는 절반가량만 썰어주시면 되요. 분량의 양념을 모두 섞어 준비했어요. 달궈진 팬에 고추기름을 두르고 다진마늘 1작은술과 닭가슴살 그리고 통마늘을 넣고 달달 볶아줍니다. 이때는 센불로 볶아주세요. 닭이 2/3정도 익었을즈음 청양고추를 넣어 볶아주시는데요. 이때 불의 세기를 중불로 조절합니다. 살짝 볶아준 후 바로 분량의 양념장을 넣어준후 조려내듯이 볶아주세요. 짠~ 마지막으로 참깨도 솔솔 뿌려 마무리 해주었어요. 레시피 상세보기 4.마리텔, 매운 해물볶음면 레시피ㅋ 분식레시피! by.요리가좋아in독일 분량 :2인분 조리시간 :15분 이내 난이도 :아무나 [주재료] 새우, 오징어 1컵(종이컵), 다진마늘 2T, 만능매운장 2T, 라면 2개, 파 1대, 양파/브로콜리 (방송X 생략가능) 조금씩, 청양고추 조금 [양념 및 소스재료] 간장 1T, 고추가루 조금, 통깨 조금, 참기름 1T [요리순서] 재료를 준비해주세요. (만능매운장은 업로드해둔 지난 레시피를 참고하세요) 냄비에 면을 삶을 물을 넣고 끓입니다. 재료들을 볶을 팬에는 식용유를 두르고 다진마늘을 볶습니다. 마늘 향이 충분히 났으면, 양파와 해산물을 모두 넣고 볶습니다. (레몬 반 쪽이 있어서 추가했습니다) 매운 만능장 2스푼을 추가하고 브로콜리를 넣습니다. (방송레시피X) 간장1/ 설탕1/ 청양고추, 가루 조금씩 넣고 간을 맞춥니다. 양념이 탈것 같으면 물을 조금 붓습니다. 끓는 물에 면을 삶아주세요. 너무 오래 삶으면 볶으면서 퍼지므로 1~2분 후 꺼내서 팬으로 옮깁니다. 팬에 면을 넣고 볶습니다. 라면 수프는 절반만 넣어줍니다. 잘 섞어준 뒤 불을 끕니다. 파와 참기름 1t을 넣고 마무리하면 요리 완성입니다! 레시피 상세보기 5.매운 볶음 우동 만들기 by.밍밍뿡 난이도 :초급 [주재료] 새송이버섯, 청양고추, 양파, 파프리카, 영양부추, 적채 조금씩, 우동면, 칵테일새우, 레몬즙 조금 [양념 및 소스재료] 고추기름 2스푼, 굴소스 2스푼, 두반장소스 1/2스푼, 올리고당 1/2스푼, 간장 1/2스푼 [요리순서] 냉장고 자투리 야채를 길쭉 하게 잘라주세요. 집에 조금씩 남아 있는 야채 사용하시면 되요. 냉장고 깊숙이 있던 소스들도 꺼내봅니다. 고추기름과 굴소스,두반장 소스도 준비. 새우는 냉동상태이니 흐르는 물에 해동 시켜주세요. 요즘엔 내장도 다 제거되어 있어 내장제거 안해도 되네요. 이미 익힌 상태에서 급 냉동된 칵테일 새우니 해동후 레몬즙 살짝 뿌려 냄새만 제거해 주세요. 살짝 달군 팬에 고추 기름을 2스푼 넣어 주세요. 매운맛이 나네요. ㅎ 좋다 매운맛 우선 새우를 먼저 센불에서 볶아 주세요. 야채와 면이 들어가 다시 볶아야 하니 반정도 익게 볶아주세요. 썰어 두었던 자투리 야채도 넣어주세요. 이제 바로 양념을 해요. 굴소스 2스푼을 넣어 주세요. 감칠맛이 나는 굴소스 두반장소스도 있어 넣어 봅니다. 반스푼 정도만 넣어 매운맛을 더해 줍니다. 그리고 분량대로 간장과 올리고당 후추를 조금 넣어 휘릭 휘릭 센불에 볶아 간을 맞춰 주세요. 우동면도 살짝 뜨거운 물에 삶아주세요. 끓는 물에 2분 정도면 스르르 풀려요. 볶아야 하니 살짝 데친다 생각하고 삶아 주세요. 야채가 담긴 팬에 우동면을 넣어주세요. 탱글 탱글 면이 불으니 센불에서 볶아 주시면 되요. 휘릭 휘릭~ 양념이 베이게 골고루 빠르게 볶아주세요. 야채가 가득 들어가 아삭이면서 매콤한 볶음 우동 완성^^ 레시피 상세보기 " />

function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);
        }
     } else {
        getBlogContentsList();
     }
}
function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}

        if (!q_link) {

}

 <h1><b>2.매운 잡채덮밥</b>
 <span>by. 시연성연맘</span>
 <div class="recipe_info">
 <b>분량 :</b>1인분 <b>조리시간 :</b>90분 이내 <b>난이도 :</b>초급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/06/10/3f8c86e7676e0748a84cebf13e92e7b81.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>당면 2/3줌, 양파 1/4개, 오이약간, 청양고추 1개, 느타리버섯 약간, 식용유, 참깨, 굴소스 1큰술</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>[잡채양념]고춧가루 1작은술, 간장 1큰술, 설탕 1작은술, 참기름 2/3큰술, 다진파 2/3큰</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>오이는 돌려깍기 한 후 채썰어 준비하고 양파와 청양고추 역시 씨를 뺀 후 가늘게 채썰어 준비합니다. 느타리 버섯은 적당한 크기로 찢어 준비해 주세요.</li>
 <li>분량의 잡채양념재료를 섞어 미리 준비해주세요.</li>
 <li>굴소스도 1큰술 넣어 함께 섞어주세요.</li>
 <li>달궈진 팬에 식용유를 두른 후 채썰어둔 재료들을 모두 넣어 달달 센불에서 볶아주세요. 참고로 오이는 입맛에 따라 가첨하시면 되는데요 오이를 넣으니 아삭함이 살아서 더 맛있게 느껴진답니다.</li>
 <li>채소들이 모두 볶아졌다면 여기에 미리 준비해 둔 잡채양념을 넣어 중불에서 한 번 더 볶아주세요.</li>
 <li>불려둔 당면을 넣어 1분간 볶아주세요. 이미 충분히 불려둔 상태인지라 가볍게만 볶아내도 당면은 충분히 익어요.</li>
 <li>매콤해서 더욱 맛있는 매콤 잡채덮밥 완성이네요.</li>
 </ol>
 </dd>

 <h1><b>3.맛있게 매운 닭가슴살 볶음</b>
 <span>by. 시연성연맘</span>
 <div class="recipe_info">
 <b>조리시간 :</b>30분 이내 <b>난이도 :</b>초급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/04/04/5a9bddaf0ace4008358799687d0109811.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>닭가슴살 2조각, 청양고추 3~4개, 양배추 약간, 통마늘 8개, 다진마늘 1작은술, 고추기름 1작은술(또는 식용유)</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>간장 1작은술, 굴소스 1과 1/3큰술, 핫소스 1큰술, 후추가루 약간, 생강가루 약간, 올리고당 1큰술<br>
</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>달달 볶아 먹을 꺼라 한입에 쏙 들어가는 크기로 썰어 주었어요.</li>
 <li>그리고 매콤함을 더해줄 청양고추는 송송~ 통마늘은 그대로 혹은 너무 큰 마늘의 경우에는 절반가량만 썰어주시면 되요.</li>
 <li>분량의 양념을 모두 섞어 준비했어요.</li>
 <li>달궈진 팬에 고추기름을 두르고 다진마늘 1작은술과 닭가슴살 그리고 통마늘을 넣고 달달 볶아줍니다. 이때는 센불로 볶아주세요.</li>
 <li>닭이 2/3정도 익었을즈음 청양고추를 넣어 볶아주시는데요. 
이때 불의 세기를 중불로 조절합니다.</li>
 <li>살짝 볶아준 후 바로 분량의 양념장을 넣어준후 조려내듯이 볶아주세요.</li>
 <li>짠~ 마지막으로 참깨도 솔솔 뿌려 마무리 해주었어요.</li>
 </ol>
 </dd>

 <h1><b>4.마리텔, 매운 해물볶음면 레시피ㅋ 분식레시피!</b>
 <span>by.요리가좋아in독일</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>15분 이내 <b>난이도 :</b>아무나 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/12/01/ebf9db0b91346c620644e198e6dc36751.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>새우, 오징어 1컵(종이컵), 다진마늘 2T, 만능매운장 2T, 라면 2개, 파 1대, 양파/브로콜리 (방송X 생략가능) 조금씩, 청양고추 조금</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>간장 1T, 고추가루 조금, 통깨 조금, 참기름 1T</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>재료를 준비해주세요. (만능매운장은 업로드해둔 지난 레시피를 참고하세요)</li>
 <li>냄비에 면을 삶을 물을 넣고 끓입니다.</li>
 <li>재료들을 볶을 팬에는 식용유를 두르고 다진마늘을 볶습니다.</li>
 <li>마늘 향이 충분히 났으면, 양파와 해산물을 모두 넣고 볶습니다. (레몬 반 쪽이 있어서 추가했습니다)</li>
 <li>매운 만능장 2스푼을 추가하고</li>
 <li>브로콜리를 넣습니다. (방송레시피X) 간장1/ 설탕1/ 청양고추, 가루 조금씩 넣고 간을 맞춥니다. 양념이 탈것 같으면 물을 조금 붓습니다.</li>
 <li>끓는 물에 면을 삶아주세요. 너무 오래 삶으면 볶으면서 퍼지므로 1~2분 후 꺼내서 팬으로 옮깁니다.</li>
 <li>팬에 면을 넣고 볶습니다. 라면 수프는 절반만 넣어줍니다.</li>
 <li>잘 섞어준 뒤 불을 끕니다. 파와 참기름 1t을 넣고 마무리하면 요리 완성입니다!</li>
 </ol>
 </dd>
 </dl>

 <h1><b>5.매운 볶음 우동 만들기</b>
 <span>by.밍밍뿡</span>
 <div class="recipe_info">
 <b>난이도 :</b>초급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/04/18/ac1a4415e20c048b3449bca5aaa2c7961.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>새송이버섯, 청양고추, 양파, 파프리카, 영양부추, 적채 조금씩, 우동면, 칵테일새우, 레몬즙 조금</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>고추기름 2스푼, 굴소스 2스푼, 두반장소스 1/2스푼, 올리고당 1/2스푼, 간장 1/2스푼</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>냉장고 자투리 야채를 길쭉 하게 잘라주세요. 집에 조금씩 남아 있는 야채 사용하시면 되요.</li>
 <li>냉장고 깊숙이 있던 소스들도 꺼내봅니다. 고추기름과 굴소스,두반장 소스도 준비. 새우는 냉동상태이니 흐르는 물에 해동 시켜주세요. 요즘엔 내장도 다 제거되어 있어 내장제거 안해도 되네요.</li>
 <li>이미 익힌 상태에서 급 냉동된 칵테일 새우니 해동후 레몬즙 살짝 뿌려 냄새만 제거해 주세요.</li>
 <li>살짝 달군 팬에 고추 기름을 2스푼 넣어 주세요. 매운맛이 나네요. ㅎ 좋다 매운맛</li>
 <li>우선 새우를 먼저 센불에서 볶아 주세요. 야채와 면이 들어가 다시 볶아야 하니 반정도 익게 볶아주세요.</li>
 <li>썰어 두었던 자투리 야채도 넣어주세요. 이제 바로 양념을 해요.</li>
 <li>굴소스 2스푼을 넣어 주세요. 감칠맛이 나는 굴소스</li>
 <li>두반장소스도 있어 넣어 봅니다. 반스푼 정도만 넣어 매운맛을 더해 줍니다.</li>
 <li>그리고 분량대로 간장과 올리고당 후추를 조금 넣어 휘릭 휘릭 센불에 볶아 간을 맞춰 주세요.</li>
 <li>우동면도 살짝 뜨거운 물에 삶아주세요. 끓는 물에 2분 정도면 스르르 풀려요. 볶아야 하니 살짝 데친다 생각하고 삶아 주세요.</li>
 <li>야채가 담긴 팬에 우동면을 넣어주세요. 탱글 탱글 면이 불으니 센불에서 볶아 주시면 되요.</li>
 <li>휘릭 휘릭~ 양념이 베이게 골고루 빠르게 볶아주세요.</li>
 <li>야채가 가득 들어가 아삭이면서 매콤한 볶음 우동 완성^^</li>
 </ol>
 </dd>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
