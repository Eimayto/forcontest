 <div class="recipe_info">

 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>무염버터 85 g, 달걀 1/2개, 유기농 설탕 80 g, 바닐라 익스트랙 1 ts, 우리밀가루 160 g, 베이킹 파우더 1/2 ts, 소금 1/8 ts, 다크 초콜릿 15 g</dd>

 <li>바닐라 익스트랙을 넣고 휘핑한 후 두어번 체친 밀가루와 베이킹파우더를 넣고 주걱으로 #자를 그리며 잘 섞습니다.</li>

 <li>흰색 반죽과 초콜릿 반죽을 이용하여 동글동글 손으로 빚어 곰돌이를 만들어주세요. 190도로 예열된 오븐에서 7~12분 정도 바닥이 노릇할 정도로 구워주세요~</li>
 </ol>
 </dd>

 <h1><b>5.곰돌이 찜케익 만들기 </b>
 <span>by.초록바람N</span>
 <div class="recipe_info">
 <b>분량 :</b>3인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>초급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2016/05/01/c5d46ba38106e5e7e2e4cc8e23fd7f581.jpg">
 </h1>


 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>볼에 계란을 푼 후 설탕을 넣고 잘 섞은 후, 우유를 넣고 잘 섞어주세요.</li>
 <li>바닐라 익스트랙을 넣고 섞은 후, 체친 밀가루와 베이킹 파우더를 넣고 거품기로 잘 섞어줍니다. 바닐라 익스트랙은 없으면 생략 가능합니다.</li>
 <li>전자레인지에 잠깐 돌려 녹인 버터를 미지근하게 식혀 넣고, 완성된 반죽 한큰술만 덜어 짤주머니에 넣어주세요. 따로 뗀 반죽으로 곰돌이 코부분을 만들어줄꺼에요~</li>
 <li>볼에 있는 반죽에는 코코아를 넣고 잘 섞어주세요.. 그리고 코코아 반죽 한큰술을 덜어 짤주머니에 넣어주세요. 이 반죽으론 곰돌이 귀를 만들어줄겁니다~</li>
 <li>머핀컵에 완성된 반죽을 팬닝해주세요. 사진상으론 팬닝이 많이 됐는데, 여러분들은 머핀컵의 60~70프로만 팬닝해주세요. 많이 팬닝하면 너무 부풀어 모양이 예쁘지 않습니다. 머핀컵은 모양이 단단하게 유지되는 걸로 하시면 좋아요~ 머핀컵이 단단하지 않거나 잘 풀어지는 머핀컵이라면, 부푸는 중에 모양이 다 찌그러집니다~ 그리고 하얀 반죽으로 코부분을 그려주세요. 원모양을 작게 그려주시는게 좋아요~ 크면 부푸는 중에 하얀원이 너무 크게 퍼져버립니다~</li>
 <li>귀부분은 따로 종이호일 깐 팬에 작은 원형으로 만들어주세요. 그리고 팔팔끓어 김오른 찜통에서 20~25분 정도 쪄주세요. 반죽에 물이 떨어지지 않게 뚜껑에 면보 씌워 쪄주세요. 찜통은 좀 큰게 좋습니다~ 꼬치로 찔러보아 깨끗하면 다 익은거에요~</li>
 </ol>
 </dd>
 </dl>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
