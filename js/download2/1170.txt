(10초 정도?) 통째로 조리할 시 다리부분만 가위로 잘라 먼저 꺼내고 머리는 완전히 익혀준다. 데쳐낸 쭈꾸미는 찬물에 헹구어 탱글탱글한 식감을 살려준다. 헹군 쭈꾸미는 먹기좋은 크기로 잘라 물기를 빼준다. 머리를 통째로 조리시 완전히 익힌 후 찬물에 헹구어 준비한다. 팬에 들기름 2스푼 , 고춧가루 2티스푼 ,다진마늘1티스푼을 넣어 볶는다. 마늘과 고춧가루가 익어가는 냄새가 날때까지 볶다가 준비해둔 쭈꾸미와 양념장을 2~3스푼 듬뿍 넣어 센불에서 1분가량 볶아준다. 1분을 볶은 쭈꾸미에 부추를 제외한 채소를 넣어 센불에서 1분을 더 볶다가 불을 줄여 30초 정도 볶아준다. 끝으로 부추와 깨를 넣고 후다닥 한번 볶아내면 완성 ! 소면과 함께 낸다. 레시피 상세보기 3.바지락, 마늘, 올리브유만 있으면 만들 수 있는 봉골레파스타!! 쉽게 만들어서 맛있게 먹어요~ by.야모 분량 :1인분 조리시간 :60분 이내 난이도 :초급 [주재료] 바지락 5~8개, 양파 1/4개, 마늘 2톨, 스파게티면 1인분 [양념 및 소스재료] 올리브유 3T, 소금 1.5T, 후추 약간 [요리순서] 굵은소금(1t)넣은 물에 스파게티면을 삶아줍니다. 이 때 주의사항!! 요구되는 시간보다 2분 정도 덜 삶아줍니다. 올리브유를 살짝 떨궈 삶은 면에 코팅을 시켜주면, 면끼리 들러붙거나 퍼지지 않아요. 해감한 바지락을 물(2C)을 넣고 1.5C 정도로 줄어들만큼 끓여줍니다. 바지락이 입을 쫙!! 국물은 뽀얗게!! 양파는 잘게 잘라주고, 마늘은 편썰어줍니다. 달군 팬에 올리브유(1T)를 넣고 마늘을 먼저 볶아 향을 내고 양파를 넣어 볶아줍니다. 바지락만 걷어서 넣고, 뽀얗게 우러나온 국물은 체에 걸러 넣어줍니다. 살짝 덜 익은 면을 넣고 익혀줍니다. 국물이 자작하게 졸아들면 올리브유(1.5T)를 넣고 소금간 살짝 후추도 살짝 뿌려 간을 합니다. 그릇에 담아 주면 끝!! 파슬리가루 살짝 뿌려서 완성했습니다, 봉골레스파게티!! 레시피 상세보기 4.춘곤증에 좋은 봄나물! 달래무침만드는법 by.돌하우스 분량 :1인분 조리시간 :15분 이내 난이도 :초급 [주재료] 달래 한줌 [양념 및 소스재료] 진간장 1.5스푼, 고춧가루 1/2스푼, 설탕 1/3스푼, 통깨 1/3스푼, 들기름 1/2스푼, 매실원액 1스푼 [요리순서] 달래는 보통 고무줄에 한줌정도 묶어서 팔더라구요... 그렇게 한줌정도되는 양을 사용했습니다. 흐르는 물에 여러번 헹궈서 흙과 잡풀들을 씻어낸후 적당한 길이로 썰어줍니다... 분량의 진간장, 고춧가루, 설탕, 통깨, 들기름, 매실원액을 넣고 잘 섞어서 양념장을 만듭니다. 적당히 썰어논 달래에 양념장을 넣어가면서 무쳐내면 완성입니다... 레시피 상세보기 " />

function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}
