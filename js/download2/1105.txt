<meta property="og:description" content="가지 특유의 물컹한 식감과 가지향으로 인해 가지 안 좋아하는 분들 꽤 있으시죠? 그렇지만, 가지는 칼로리가 낮고 영양만점이며 특히 해독능력이 아주 뛰어난 식품이다. 독소가 많이 쌓이는 현대인들에게 더할 나위 없이 좋은 식품이기 때문에 섭취를 해주는 것이 좋다. 가지를 맛있게 먹을 수 있는 방법으로 가지요리 매력에 빠져보자. 1.토마토 가지 스파게티 by.아내의식탁 분량 :2인분 조리시간 :15분 이내 난이도 :초급 [주재료] 스파게티 면 300g, 소금 2큰술, 물 3L, 가지 1개, 올리브유 약간, 파마산 치즈 약간, 소금 약간, 후추 약간 [양념 및 소스재료] 홀토마토 2캔, 올리브유 4큰술, 다진마늘 2큰술, 다진양파 200g, 생 바질 잎 6잎, 이탈리안 시즈닝 1큰술(생략 가능), 소금 1~2작은술, 설탕 1~2큰술, 후추 약간 [요리순서] 홀토마토는 꼭지와 껍질을 제거하여 손으로 으깨주시고 양파는 잘게 다져주세요. 가지는 어슷하게 썰어서 소금 후추를 뿌려 두었다가 팬에 올리브유를 살짝 두르고 구워주세요. 깊이가 있는 팬에 올리브유를 두르고 다진 마늘과 양파를 넣어 달달한 향이 올라올 때까지 볶아주세요. 양파가 익으면 토마토, 바질, 올리브유, 이탈리안 시즈닝을 넣고 20~30분간 저어가며 끓여주세요. 소스의 양이 2/3정도로 줄어들면 소금, 설탕, 후추로 간을 하고 구워둔 가지를 넣어 섞어주세요. 냄비에 물을 끓이고 소금과 스파게티 면을 넣고 익혀주세요.(스파게티면 봉지에 적힌 조리 시간보다 1~2분 정도 적게 삶아주세요) 면이 익으면 체에 건져 물기를 빼고 올리브유를 약간 뿌린 후 고루 섞어주세요. 접시에 면을 얹고 소스를 뿌린 후 파마산 치즈를 뿌려서 완성해주세요. 레시피 상세보기 2.가지만두 만들기 by.시크제이맘 분량 :2인분 난이도 :아무나 [주재료] 가지, 다진돼지고기 1컵, 대파 컵, 부추 1/2컵 [양념 및 소스재료] 소금 1/2컵, 참기름 1/2컵, 생강, 후추, 튀김가루 ,식용유 [요리순서] 가지만두에 들어갈 재료를 준비해주세요. 그릇에 다진 돼지고기 1컵 + 파1컵 + 부추1/2컵을 일단 준비해주세요. 그리고 생강을 검지 한마디 정도 넣어주세요. 전 생강가루를 사용했습니다. 소금은 숟가락으로 1/2정도 넣어주세요. 참기름은 소주컵으로 1/2 넣어주세요. 튀김가루 소주컵으로 1컵 넣어주세요. 후추 살짝 넣어 반죽해 주심 된답니다. 잘 버무려주세요. 가지는 굵은걸로 준비해 주시는게 좋아요. 하지만 제가 준비한 가지는 보통인듯합니다. 가지에 소를 집어 넣어주기 위해 가지를 잘라줘야합니다. 사진처럼 나무젓가락을 가지아랫쪽에 나란히 깔고 일정한 두께로 잘라주세요. 잘라 놓은 가지는 짝수번째마디만 끝까지 잘라주면 됩니다. 캐스터네츠 모양으로 완성되요. 가지 입을 벌리고 그 사이에 소를 넣어주면 완성 적당이 넣어야지 너무 많이 넣으면 지저분하게 흘러나온답니다. 튀김가루 1컵 + 물 1/2를 넣어 잘 섞어주세요. 농도조절은 물로 해주심 되요. 만들어 놓은 튀김옷에 가지만두를 잘 묻혀주세요. 식용유를 넉넉하게 부어 기름을 달궈주세요. 기름 온도를 위해 전 튀김반죽을 떨어 트려서 확인했네요. 반죽이 위로 올라오면 적당한 온도랍니다. 기름속으로 가지만두 퐁당~~ 노오랗게 튀겨지면 완성~ 튀기는 시간은 생각보다 길지 않아요. 키친타올을 이용해 기름을 제거한 후 맛있게 드시면 됩니다. 레시피 상세보기 3.백종원표 매콤한 가지볶음과 안매운 가지볶음 황금레시피 by.인생은소풍 분량 :2인분 조리시간 :30분 이내 난이도 :초급 [주재료] 가지 3개 [양념 및 소스재료] 매운가지볶음양념 : 간장 5스푼, 국간장 1스푼, 설탕 1스푼, 깨소금 1스푼, 참기름 1스푼, 고추 2개, 다진마늘 반스푼, 고춧가루 1스푼 안매운가지볶음양념 : 간장 1스푼, 깨소금 1스푼, 참기름 1스푼, 다진마늘 반스푼 [요리순서] 가지는 어슷썰기로 잘라서 준비해줍니다. 약 5센티로 잘라주면 될거같아요. 후라이펜에 기름을 두르고 가지를 올려줍니다. 약불에 익혀주는게 포인트에요. 쎈불에 하면 가지가 흑색으로 변함. 요리조리 알맞게 잘 익혀줍니다. 건조시킨 가지로 변신. 양념장을 만들어볼께요. 가장 중요한 순서죠 다진마늘 반스푼, 고추도 잘게 썰어줍니다. 고추대신 파를 넣어주면 더 좋을거 같아요. 간장 5스푼, 국간장 1스푼, 참기름 1스푼, 깨소금 1스푼, 고춧가루 1스푼, 설탕 1스푼 넣고 잘 섞어주면 양념장이 제대로 만들어졌어요. 양념장과 가지를 넣고 볶아주면 완성입니다. 안매운 가지볶음도 만들어 봅니다. 가지를 삼등분해서 잘라줍니다. 팔팔 끓인 찜기에 가지를 넣고 5분간 쪄주는거에요. 두껑을 닫고 찌는게 포인트. 준비한 다진마늘, 간장넣고 참기름, 깨소금 넣으면 양념은 끝. 5분간 잘 쪄진 가지를 먹기좋게 썰어줍니다. 준비한 양념과 버무려주면 완성이에요. 스피드하게 반찬 두개가 뚝딱 만들어졌네요. 레시피 상세보기 " />

var _blogIsSubmit = false;
function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);
        }
     } else {
        getBlogContentsList();
     }
}
function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}
function setBlogContents(q_link) {
    if (typeof q_link == 'undefined' || q_link == '') {
        q_link = $("#listBlog [id^=q_link_]:checked").val();

        if (!q_link) {
            alert('블로그 컨텐츠를 선택하세요.');
            return;
        }
    }
    $("#btnSetBlogContents").html('<span class="fa fa-spinner fa-spin" style="width:' + $("#btnSetBlogContents").width() + 'px"></span>');
    if(fromOther == 'n') document.location.href = '/inbox/insert.html?ref_blog_url=' + encodeURIComponent(q_link);
    else if(fromOther == 'review') document.location.href = '/event/ins_review.html?seq=1105&review_seq=&ref_blog_url=' + encodeURIComponent(q_link);
}

    }

}

function doCommentSubmit()
{
    var txtVal = $("#cmtTextarea").val();
    if(!txtVal)
    {
        alert('의견을 입력해주세요.');
        return ;
    }
    if(txtVal.length < 4)
    {
        alert('의견은 4자 이상으로 작성해 주세요.');
        return ;
    }
    $.ajax({
        type: "POST",
        url: "/bbs/ajax.html",
        data : $("#insCmtFrm").serialize(),
        success: function(result) {
            if(result == 'SUCCESS')
            {
                $("#cmtTextarea").val('');
                getCommentList();
                var ccnt = parseInt($("#recipeCommentListCount").html(),10);
                $("#recipeCommentListCount").html(ccnt+1);
            }
            else if(result == 'NO_PARAM')
            {
                alert('파라미터 부족. 다시 확인해 주세요.')
            }
            else
            {
                alert('등록 실패! 잠시후 다시 시도해 주세요.')
            }
        },
        error: function () {
            alert('의견 조회 실패! 잠시후 다시 시도해 주세요.')
        }
    });
}

        });
    }
}

 <h1><b>1.토마토 가지 스파게티 </b>
 <span>by.아내의식탁</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>15분 이내 <b>난이도 :</b>초급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/10/30/17a3058ad61ddd1bbae7e5d454ccfe871.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>스파게티 면 300g, 소금 2큰술, 물 3L, 가지 1개, 올리브유 약간, 파마산 치즈 약간, 소금 약간, 후추 약간</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>홀토마토 2캔, 올리브유 4큰술, 다진마늘 2큰술, 다진양파 200g, 생 바질 잎 6잎, 이탈리안 시즈닝 1큰술(생략 가능), 소금 1~2작은술, 설탕 1~2큰술, 후추 약간</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>홀토마토는 꼭지와 껍질을 제거하여 손으로 으깨주시고 양파는 잘게 다져주세요.</li>
 
 <li>가지는 어슷하게 썰어서 소금 후추를 뿌려 두었다가 팬에 올리브유를 살짝 두르고 구워주세요. 깊이가 있는 팬에 올리브유를 두르고 다진 마늘과 양파를 넣어 달달한 향이 올라올 때까지 볶아주세요. 양파가 익으면 토마토, 바질, 올리브유, 이탈리안 시즈닝을 넣고 20~30분간 저어가며 끓여주세요. 소스의 양이 2/3정도로 줄어들면 소금, 설탕, 후추로 간을 하고 구워둔 가지를 넣어 섞어주세요. 냄비에 물을 끓이고 소금과 스파게티 면을 넣고 익혀주세요.(스파게티면 봉지에 적힌 조리 시간보다 1~2분 정도 적게 삶아주세요) 면이 익으면 체에 건져 물기를 빼고 올리브유를 약간 뿌린 후 고루 섞어주세요.</li>
 
 <li>접시에 면을 얹고 소스를 뿌린 후 파마산 치즈를 뿌려서 완성해주세요.</li>
 </ol>
 </dd>

 <h1><b>2.가지만두 만들기</b>
 <span>by.시크제이맘</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>난이도 :</b>아무나 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/12/14/dbc462f929778b83d57afd70206349641.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>가지, 다진돼지고기 1컵, 대파 컵, 부추 1/2컵</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>소금 1/2컵, 참기름 1/2컵, 생강, 후추, 튀김가루 ,식용유</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>가지만두에 들어갈 재료를 준비해주세요. 그릇에 다진 돼지고기 1컵 + 파1컵 + 부추1/2컵을 일단 준비해주세요.</li>
 
 <li>그리고 생강을 검지 한마디 정도 넣어주세요. 전 생강가루를 사용했습니다.</li>
 
 <li>소금은 숟가락으로 1/2정도 넣어주세요.</li>
 
 <li>참기름은 소주컵으로 1/2 넣어주세요.</li>
 
 <li>튀김가루 소주컵으로 1컵 넣어주세요. 후추 살짝 넣어 반죽해 주심 된답니다.</li>
 
 <li>잘 버무려주세요.</li>
 
 <li>가지는 굵은걸로 준비해 주시는게 좋아요. 하지만 제가 준비한 가지는 보통인듯합니다. 가지에 소를 집어 넣어주기 위해 가지를 잘라줘야합니다. 사진처럼 나무젓가락을 가지아랫쪽에 나란히 깔고 일정한 두께로 잘라주세요.</li>
 
 <li>잘라 놓은 가지는 짝수번째마디만 끝까지 잘라주면 됩니다. 캐스터네츠 모양으로 완성되요.</li>
 
 <li>가지 입을 벌리고 그 사이에 소를 넣어주면 완성 적당이 넣어야지 너무 많이 넣으면 지저분하게 흘러나온답니다.</li>
 
 <li>튀김가루 1컵 + 물 1/2를 넣어 잘 섞어주세요. 농도조절은 물로 해주심 되요.</li>
 
 <li>만들어 놓은 튀김옷에 가지만두를 잘 묻혀주세요.</li>
 
 <li>식용유를 넉넉하게 부어 기름을 달궈주세요. 기름 온도를 위해 전 튀김반죽을 떨어 트려서 확인했네요. 반죽이 위로 올라오면 적당한 온도랍니다.</li>
 
 <li>기름속으로 가지만두 퐁당~~ 노오랗게 튀겨지면 완성~ 튀기는 시간은 생각보다 길지 않아요.</li>
 
 <li>키친타올을 이용해 기름을 제거한 후 맛있게 드시면 됩니다.</li>
 </ol>
 </dd>
 </dl>

 <h1><b>3.백종원표 매콤한 가지볶음과 안매운 가지볶음 황금레시피</b>
 <span>by.인생은소풍</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>30분 이내 <b>난이도 :</b>초급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/09/10/3ac02cef41697db5c59eaeae5494229f1.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>가지 3개</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>매운가지볶음양념 : 간장 5스푼, 국간장 1스푼, 설탕 1스푼, 깨소금 1스푼, 참기름 1스푼, 고추 2개, 다진마늘 반스푼, 고춧가루 1스푼 <br>
안매운가지볶음양념 : 간장 1스푼, 깨소금 1스푼, 참기름 1스푼, 다진마늘 반스푼</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>가지는 어슷썰기로 잘라서 준비해줍니다. 약 5센티로 잘라주면 될거같아요.</li>
 
 <li>후라이펜에 기름을 두르고 가지를 올려줍니다. 약불에 익혀주는게 포인트에요. 쎈불에 하면 가지가 흑색으로 변함.</li>
 
 <li>요리조리 알맞게 잘 익혀줍니다.</li>
 
 <li>건조시킨 가지로 변신.</li>
 
 <li>양념장을 만들어볼께요. 가장 중요한 순서죠 다진마늘 반스푼, 고추도 잘게 썰어줍니다. 고추대신 파를 넣어주면 더 좋을거 같아요.</li>
 
 <li>간장 5스푼, 국간장 1스푼, 참기름 1스푼, 깨소금 1스푼, 고춧가루 1스푼, 설탕 1스푼 넣고</li>
 
 <li>잘 섞어주면 양념장이 제대로 만들어졌어요.</li>
 
 <li>양념장과 가지를 넣고 볶아주면 완성입니다.</li>
 
 <li>안매운 가지볶음도 만들어 봅니다. 가지를 삼등분해서 잘라줍니다.</li>
 
 <li>팔팔 끓인 찜기에 가지를 넣고 5분간 쪄주는거에요.</li>
 
 <li>두껑을 닫고 찌는게 포인트.</li>
 
 <li>준비한 다진마늘, 간장넣고</li>
 
 <li>참기름, 깨소금 넣으면 양념은 끝.</li>
 
 <li>5분간 잘 쪄진 가지를 먹기좋게 썰어줍니다.</li>
 
 <li>준비한 양념과 버무려주면 완성이에요. 스피드하게 반찬 두개가 뚝딱 만들어졌네요.</li>
 </ol>
 </dd>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
