function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();

function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();

            return html;
        },

            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}

 <h1><b>2.오늘뭐먹지, 스페인 요리사가 알려준 정통 '파에야' 레시피!</b>
 <span>by.요리가좋아in독일</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>중급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/11/30/6e2815d00f2ca572120e805c5d6c9c491.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>해산물 1컵(종이컵), 토마토 2개, 마늘 2~3쪽, 닭가슴살 (두부사용) 1덩이, 양파 1개 (소), 파프리카 조금, 레몬 1/2개, 애호박(생략가능) 조금, 샤프란 (강황가루로 대체) 조금</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>치킨스톡 1T, 후추 / 소금 조금씩</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>재료를 준비해주세요. 저는 닭가슴살 대신 두부를 이용해봤어요 ^ ^</li>
 <li>재료들을 잘게 썰어서 준비합니다.</li>
 <li>닭가슴살 (저는 두부)을 깍뚝썰어 소금과 후추로 밑간을 합니다.</li>
 <li>토마토는 절반으로 갈라 강판에 갈아냅니다.</li>
 <li>파에야의 기본 소스가 되는 소프리토를 만듭니다. 먼저, 냄비에 올리브오일을 두르고 파프리카/ 양파/ 애호박을 넣습니다. 소금 2~3꼬집으로 간을 합니다.</li>
 <li>양파 색이 변하면 강판으로 갈아둔 토마토를 모두 넣고 볶습니다.</li>
 <li>레몬즙을 짜서 넣고, 다진 마늘로 마무리합니다. 소프리토 완성입니다!</li>
 <li>팬에 올리브 오일을 두르고, 닭가슴살(두부)을 먼저 볶고, 해산물과 소프리토를 넣고 함께 볶습니다. (새우는 마지막에 올릴 3~4마리를 제외하고 다 넣습니다)</li>
 <li>쌀과 함께 끓일 육수를 만듭니다. 물 3~4컵과 치킨스톡 1스푼, 샤프란(강황가루로 대체)를 넣고 끓여줍니다.</li>
 <li>팬에 생 쌀 한컵을 넣습니다.</li>
 <li>쌀의 두 배가 되는 양의 육수를 넣고 자작하게 끓입니다.</li>
 <li>졸아들면 육수를 조금씩 추가하면서 볶습니다. 내용물을 젓지않고 팬을 흔들면서 15~20분가량 볶습니다.</li>
 <li>간을 보시고, 쌀의 중심부분이 살짝 설익었을 때!</li>
 <li>남겨둔 새우들을 올려줍니다.</li>
 <li>그리고 포일로 팬을 싸고 오븐에 익히듯, 바짝 익혀냅니다.</li>
 <li>3~4분 가량 기다렸다가 오픈해, 잘 익었는지 확인하시면 됩니다!</li>
 </ol>
 </dd>

 <h1><b>3.올리브쇼 스페인요리 타파스만들기</b>
 <span>by.강철새잎</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>15분 이내 <b>난이도 :</b>아무나 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2016/04/23/c90874e09b8f642e963c37a330359c1d1.jpg">
 </h1>


 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>먼저 소스를 만들어 주는데요. 일종의 마늘마요네즈라고 생각하시면 됩니다.

달걀 노른자 2개에</li>
 <li>여기에 다진마늘 한큰술을 넣어주고</li>
 <li>레몬즙도 한두방울 넣어 주세요. 올리브유도 한두방울 떨어 뜨려 주세요. 
그리고 믹서기로 갈아 주시면 마늘마요네즈가 완성됩니다. ​</li>
 <li>고소하면서도 마늘의 향이 나는 독특한 마늘마요네즈가 완성되었습니다.
소스는 잠시 냉장고에 보관해 놓고요.</li>
 <li>버섯 파타스에 들어가는 버섯은 물기가 없는 버섯을 선택 해주시면 되는데
저는 느타리버섯을 선택했습니다. 소금, 후추, 올리브유 넣고 볶아주시는데

물기가 완전히 날라갈때까지 볶아주시는 것이 포인트입니다.</li>
 <li>물기가 날라가서 다 볶아졌습니다.
이제 따로 그릇에 담아 내 놓고요.
사실 이게 다이죠</li>
 <li>타파스나 판 콘 토마테에 쓰이는 빵은
겉은 좀 단단하고 안쪽은 거친 느낌이 빵이 좋습니다. 
저는 블랙올리브빵을 선택했는데 속이 아주 거칠죠~~ 겉은 질기면서 딱딱한 편이고요. 
반으로 자른 후에 이것이 빵 속이 보이도록 뒤집어 놓고</li>
 <li>이와 같은 방식으로 썰어 줍니다.
이렇게 썰면 부드러운 빵 안쪽 부분이 앞뒤로 나오겠죠.

모양은 사각형으로</li>
 <li>이제 달궈진 팬에 올리브오일 두르고는
썰어 놓은 빵의 안쪽부분만 앞뒤로 구워 주세요.</li>
 <li>구우실 때에는 이렇게 뚜껑을 닫고 구워주면
앞뒤로 바삭바삭하게 구워 집니다.</li>
 <li>판 콘 토마테를 만들려고 하는데요.
구운 빵과 여기 완숙된 토마토와 통마늘만 있으면 완성됩니다. 
별거 아닌데 그 맛이 참 독특했어요.</li>
 <li>그리고 미리 준비를 해 놓은 볶은 버섯과 마늘마요네즈도 준비해 놓습니다. 
요건 판 콘 토마테를 이용해서 버섯 타파스를 만들어 줄려고요.</li>
 <li>빵이 겉표면이 바삭바삭하게 잘 구워 졌습니다. 
이제 초스피드로 판 콘 토마테를 만들어 보죠.</li>
 <li>구운 빵 표변에 완숙 토마토를 썰어서는
빵에 골고루 뭍혀 줍니다. 
완숙이라서 토마토의 즙이 잘 나오겠죠. 골고루 빵의 표면에 토마토의 즙을 입혀 줍니다.</li>
 <li>저민 마늘도 똑같은 방법으로 빵에 발라 줍니다. 
참으로 신기한 방법이지만
스페인사람들이 아주 즐겨 먹는 판 콘 토마테라고 하네요. 판 콘 토마테는 완성입니다. 완전히 간단하죠~~~</li>
 <li>완성된 판 콘 토마테에 올리브유를 뿌려 주시고</li>
 <li>그 위에 볶아낸 버섯을 올려 줍니다.</li>
 <li>그리고 집에 반숙된 삶은 달걀이 있어서

제법 잘 어울릴 것 같고 더욱 든든할 것 같아서 살짝 위에 올려 놓았습니다.

뭐 없으시면 그냥 버섯볶음만 올리셔도 충분히 맛있습니다.</li>
 <li>마지막으로 파슬리가루 솔솔 뿌려주고 
만들어 놓았던 마늘마네요즈 소스를 살짝 뿌려 주시면
버섯볶음 타파스 완성입니다.</li>
 </ol>
 </dd>
 </dl>

 <h1><b>4.스페인요리 칼라마리 샐러드 만들기 </b>
 <span>by.희재맘</span>
 <div class="recipe_info">
 <b>분량 :</b>4인분 <b>조리시간 :</b>30분 이내 <b>난이도 :</b>중급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/04/10/e8c40fbda5b7b1d682fa2603a7b690891.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>오징어 1마리, 사과 반쪽, 미니 파프리카 3개, 그린 샐러드 1팩, 얼음 두컵, 파슬리 약간, 튀김가루 3컵, 물</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>백설 안달루시아산 올리브유 1큰술, 간장 1큰술, 매실청 2큰술, 식초 반수저, 물 2컵, 갈색설탕 1큰술 ​</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>믹싱볼에 튀김가루를 붓어 주세요! 찬물에 얼음을 넣어 사용을 튀김옷을 입히면 더욱 바삭하게 튀김을 만들 수 있어요. 너무 세게 휘젖지 마세요. 그렇게 하면 반죽이 오히려 딱딱해져요.</li>
 <li>그냥 파프리카 보다 미니 파프리카가 더 영양이 많다고 하더라고요,. 그래서 씻은 후.. 물기는 키친타월로 제거를 한 후.. 튀김옷을 입힌 후 중간불에 튀겨냈어요. 중간에 들어주면 공기가 들어가 더욱 바삭하게 튀길 수 있답니다.</li>
 <li>집에 치즈가 있으면 튀김옷을 입혀서 튀겨 보세요!! 특히 와인과 함께 곁들어서 먹으면 좋더군요.</li>
 <li>오징어 대신 한치를 사용하면 좋았을텐데. 한치 제철이 아니라고 대신 물 좋은 갑오징어를 사용 했어요.</li>
 <li>오징어 튀김옷을 입힐 때 썰어 주신 후.. 바로 튀김 옷을 입히면 홀라당 튀김옷이 벗겨지거든요. 키친타월로 물기를 제거하신 후 튀김가루룰 묻힌 후 튀김옷을 입히세요.</li>
 <li>이때 파슬리를 썰어 튀기면 더욱 보기고 좋고 맛있답니다.</li>
 <li>마트가면 그린샐러드 담아서 판매하고 있어요~~ 더욱 아삭하게 드시고 싶다면 얼음물에 담갔다가 드시기전에 체에 받쳐 물기를 빼주신 후 샐러드 드시면 더욱 신선하게 드실 수 있어요.</li>
 <li>아무래도 튀김 요리다보니 .. 자칫 느끼할 수 있거든요~~ 이럴 때는 사과를 함께 드시면 더욱 프레쉬하게 즐길 수 있어요. 무채 썰듯 사과를 썰어 갈변을 막기 위해서 물에 갈색설탕 한수저를 넣고 담가 놓으세요.</li>
 <li>자 그럼 올리브유 드레싱 만들어 볼께요. 집에 간장 다들 있으시죠. 
간장 1큰술에 식초 반수저, 그리고 집에서 담은 매실청 2큰술, 그리고 백설 안달루시아산 올리브유 1큰술을 넣어서 올리브유 드레싱를 만들어 주시면 됩니다!! 화끈한맛을 원하시면 청양고추 반개 썰어서 넣어주시면 더욱 맛있게 드실 수 있어요.</li>
 <li>튀김도 다 만들어졌으면 자 접시에 샐러드 담아주시고요.</li>
 <li>튀김 파프리카와 오징어튀김도 올리세요! 그리고 가운데 사이사이에 사과를 꽂아 주세요!!</li>
 </ol>
 </dd>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
