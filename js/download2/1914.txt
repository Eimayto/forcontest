잘 삶은 콩나물에 당근은 얇게 자르고 파는 쫑쫑 썰어줍니다. 소금이랑 통깨도 넣어주세요. 마늘을 넣으셔도 되겠죠. 익히는것이 아니라 먹고나면 마늘냄새가 나서 전 빼고 했어요. 그리고 들기름을 팍팍 넣어요. 비닐장갑을 끼고 조물조물 무쳐주세요. 방금 먹을거면 간을 딱 맞추면 되는데 냉장고에 넣었다가 드실거라면 소금을 약간 더 넣으세요. 나중에 드시면 싱거워지더라구요. 레시피 상세보기 5.아삭아삭 콩나물무침(천원밥상) by.환갑을꿈꾸는아줌마 분량 :4인분 조리시간 :30분 이내 난이도 :아무나 [요리순서] 콩나물을 다듬어 깨끗이 3~4번 씻어요.. 바구니에 담아 물기를 제거해요. 물을 조금 넣고 콩나물을 넣어 데쳐 주어요..(이때 뚜껑을 닫아요) 센불에서... 김이 나기시작하면 중불로 (비린내가 나다가 콩나물익은냄새가 나면 불을 꺼 주어요.) 데친 콩나물을 볼에 얼른 담아두고.. 엷게 펼쳐 열기에 익지 않도록 해요.. 소금과 고춧가루, 마늘을 넣고 살살살 버리고 간을 보아 조절 하세요. 마지막에 들기름(참기름)을 주르륵~~ 참깨 솔솔...(깨소금을 넣으면 더 고소하답니다.) 쪽파는 넣지 않고, 고춧가루 넣지 않은 무침에만 넣었어요.. * 쪽파 송송 같이 넣어도 괜찮아요. 매콤한게 싫으면 담백한 콩나물무침으로.. 고춧가루만 넣지 않고.. 매운것 못 먹는 아이들을 위해서 한접시 만들었어요. 레시피 상세보기 ">

<meta property="og:description" content="비타민C가 풍부한 콩나물! 더운 여름에 기력 보충하기에 딱 좋죠~ 아삭아삭하게 씹히는 그 식감을 느끼다 보면 절로 건강해지는 느낌이 드는데요, 그냥 밥반찬으로 먹어도 좋지만 다른 반찬들 싹 긁어모아 비비면 맛있는 비빔밥으로 재탄생~ 한 끼 간단히 해결할 수 있는 콩나물무침, 오늘 반찬으로 어떠신가요? 1.아삭한 콩나물무침 만들기 by.민a지a맘 분량 :4인분 조리시간 :30분 이내 난이도 :아무나 [주재료] 콩나물, 실파 2개, 당근 약간, 양파 1개, 마늘 2개, 소금 5ml, 통깨 조금, 들기름 5ml, 고춧가루 20ml [요리순서] 먼저 콩나물을 깨끗이 씻어서 삶아야겠지요? 당근과 양파는 채를 썰어 놓아요 맛소금과 통깨 마늘과 고춧가루 들기름을 준비했습니다. 냄비에 콩나물을 넣고 15분 동안 삶아줍니다. 냄비 뚜껑을 덮어준 상태로 삶아줍니다. 삶아준 콩나물이네요. 콩나물을 바구니에 붓고 차가운 물에 헹궈줍니다. 차가운 물에 헹궈주시면 콩나물이 더 아삭하고 맛있답니다. 차가운 물에 헹궈준 콩나물을 물기를 빼주고 함박에 넣어서 미리 준비해둔 재료를 다 넣습니다. 마늘과 들기름, 고춧가루, 통깨, 소금을 넣어줍니다. 이제 콩나물을 맛있게 버무려줍니다. 그럼 아삭한 콩나물무침 만들기가 완성된다는 거죠~^^ 레시피 상세보기 2.백종원 콩나물무침 by.또바기S 분량 :3인분 조리시간 :15분 이내 난이도 :초급 [주재료] 콩나물 300g(소금 1/2큰술), 쪽파 25g, 당근 25g [양념 및 소스재료] 국간장 1/2큰술, 다진마늘 1/2큰술, 꽃소금 2/3큰술, 설탕 1/2작은술, 참기름 1과1/2큰술, 통깨 1작은술 [요리순서] 콩나물은 뿌리 끝을 자르고. 깨끗하게 씻기. 쪽파와 당근은 4~5cm 정도로 채 썰기. 물을 넉넉히 넣고 끓인 후, 소금 1/2큰술과 콩나물을 넣고 4~5분 삶기. 삶은 콩나물 찬물 헹군 뒤, 체에서 물기 빼기. 손으로 짜내면 질겨질 수 있다고 하네요. 국간장 1/2큰술, 마늘 1/2큰술, 소금 2/3큰술, 설탕 1/2작은술을 넣고 섞기. 소금은 기호에 맞게 조금씩 넣어가며 섞으면 좋을 것 같습니다. 참기름을 넣고 섞기. 통깨를 넣고 섞기. 레시피 상세보기 3.콩나물무침 아삭하게 무치기 by.마리안느40 분량 :4인분 조리시간 :15분 이내 난이도 :초급 [주재료] 콩나물 400g, 쪽파 2뿌리 [양념 및 소스재료] 소금 1/2T, 고추가루 1T, 마늘 1/3T, 깨소금 1T, 참기름 1T [요리순서] 끓는 물에 콩나물 넣고 콩나물 비린내 나지 않게 처음부터 뚜껑을 열고 3~4분정도 삶아 주시는데요. 중간에 집게로 뒤집어 주면서 골고루 삶아주세요 아삭아삭한 콩나물무침을 무치는 비법이라면 이 얼음물이 아닐까 싶네요. 콩나물 삶고 바로 물에 헹궈서 김을 날려 주시구요. 마지막에 얼음 동동 띄운물에 담궈서 헹궈주시면 콩나물이 아삭아삭해지는게 촉감으로도 느껴져요. 대파 대신 집에 쪽파가 있어서 쪽파 2뿌리 쫑쫑 썰어준비했어요. 분량의 양념 한꺼번에 넣고 썰어놓은 쪽파도 넣고 맛있게 무쳐주면 아삭아삭맛있는 콩나물무침 완성입니다. 간보고 싱거우면 소금 한꼬집씩 추가해서 맞춰주심 되세요 레시피 상세보기 4.콩나물무침 by.의정부바겨사 분량 :3인분 조리시간 :30분 이내 난이도 :초급 [주재료] 콩나물, 당근, 파 [양념 및 소스재료] 들기름, 소금, 통깨 [요리순서] 콩나물은 깨끗하게 씻어줍니다. 콩나물을 끓는 물에 삶아주는데 처음부터 뚜껑을 닫았으면 그렇게 열었으면 열어놓은 상태로 하는건 아시죠? 잘 삶은 콩나물에 당근은 얇게 자르고 파는 쫑쫑 썰어줍니다. 소금이랑 통깨도 넣어주세요. 마늘을 넣으셔도 되겠죠. 익히는것이 아니라 먹고나면 마늘냄새가 나서 전 빼고 했어요. 그리고 들기름을 팍팍 넣어요. 비닐장갑을 끼고 조물조물 무쳐주세요. 방금 먹을거면 간을 딱 맞추면 되는데 냉장고에 넣었다가 드실거라면 소금을 약간 더 넣으세요. 나중에 드시면 싱거워지더라구요. 레시피 상세보기 5.아삭아삭 콩나물무침(천원밥상) by.환갑을꿈꾸는아줌마 분량 :4인분 조리시간 :30분 이내 난이도 :아무나 [요리순서] 콩나물을 다듬어 깨끗이 3~4번 씻어요.. 바구니에 담아 물기를 제거해요. 물을 조금 넣고 콩나물을 넣어 데쳐 주어요..(이때 뚜껑을 닫아요) 센불에서... 김이 나기시작하면 중불로 (비린내가 나다가 콩나물익은냄새가 나면 불을 꺼 주어요.) 데친 콩나물을 볼에 얼른 담아두고.. 엷게 펼쳐 열기에 익지 않도록 해요.. 소금과 고춧가루, 마늘을 넣고 살살살 버리고 간을 보아 조절 하세요. 마지막에 들기름(참기름)을 주르륵~~ 참깨 솔솔...(깨소금을 넣으면 더 고소하답니다.) 쪽파는 넣지 않고, 고춧가루 넣지 않은 무침에만 넣었어요.. * 쪽파 송송 같이 넣어도 괜찮아요. 매콤한게 싫으면 담백한 콩나물무침으로.. 고춧가루만 넣지 않고.. 매운것 못 먹는 아이들을 위해서 한접시 만들었어요. 레시피 상세보기 " />

var _blogIsSubmit = false;
function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);
        }
     } else {
        getBlogContentsList();
     }
}
function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}
function setBlogContents(q_link) {
    if (typeof q_link == 'undefined' || q_link == '') {

        if (!q_link) {
            alert('블로그 컨텐츠를 선택하세요.');
            return;
        }
    }
    $("#btnSetBlogContents").html('<span class="fa fa-spinner fa-spin" style="width:' + $("#btnSetBlogContents").width() + 'px"></span>');
    if(fromOther == 'n') document.location.href = '/inbox/insert.html?ref_blog_url=' + encodeURIComponent(q_link);
    else if(fromOther == 'review') document.location.href = '/event/ins_review.html?seq=1914&review_seq=&ref_blog_url=' + encodeURIComponent(q_link);
}

    }

}

function doCommentSubmit()
{
    var txtVal = $("#cmtTextarea").val();
    if(!txtVal)
    {
        alert('의견을 입력해주세요.');
        return ;
    }
    if(txtVal.length < 4)
    {
        alert('의견은 4자 이상으로 작성해 주세요.');
        return ;
    }
    $.ajax({
        type: "POST",
        url: "/bbs/ajax.html",
        data : $("#insCmtFrm").serialize(),
        success: function(result) {
            if(result == 'SUCCESS')
            {
                $("#cmtTextarea").val('');
                getCommentList();
                var ccnt = parseInt($("#recipeCommentListCount").html(),10);
                $("#recipeCommentListCount").html(ccnt+1);
            }
            else if(result == 'NO_PARAM')
            {
                alert('파라미터 부족. 다시 확인해 주세요.')
            }
            else
            {
                alert('등록 실패! 잠시후 다시 시도해 주세요.')
            }
        },
        error: function () {
            alert('의견 조회 실패! 잠시후 다시 시도해 주세요.')
        }
    });
}

        });
    }
}

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}

    getViewRecipeList(1);
})
