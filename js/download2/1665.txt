​ 양파와 부추에 뿌릴 소스를 만들께요. ​ 간장1:식초1:물1:설탕0.5 비율로 넣어서 만들어줍니다.​ ​설탕이 잘녹을때까시 섞어서 준비합니다. 후라이팬에 목살을 구워줍니다. 양념이 묻어 있어 쉽게 탈수 있으니 약불에 구워서 자주 뒤집어 주세요. 그릇에 양파를 올리고, 그위에 썰어두었던 부추도 올립니다. 그리고 간장 소스도 골고루 뿌려줍니다. 부추위에 고추장 목살구이를 오려주면 완성!!! 레시피 상세보기 3.목살스테이크 카레 by.세미짱 분량 :4인분 조리시간 :30분 이내 난이도 :초급 [주재료] 목살 500g, 감자 3개, 양파 1개, 당근 1개, 파프리카, 물 1ℓ [양념 및 소스재료] 카레가루 100g [요리순서] 돼지고기에 소금후추로 밑간을 해주세요. 양파만 채썰고 나머지 감자, 당근, 파프리카는 큼직하게 썰어주세요. 후라이팬에 기름을 약간 두르고 목살을 노릇하게 구워주세요. 양파, 감자, 당근을 넣고 뒤적뒤적 볶아주세요. 물을 넣고 감자가 익을때까지 끓여주세요. 카레가루를 넣고 저어서 끓여주세요. 마지막에 파프리카를 넣어주세요. 레시피 상세보기 4.[간단한요리]목살숙주볶음, 술안주로 제격! by.정쿡쿡 분량 :3인분 조리시간 :30분 이내 난이도 :초급 [주재료] 돼지고기 1인 200g 기준, 파프리카 1개, 마늘 반큰술, 양파 1개, 숙주 [양념 및 소스재료] 후추 및 소금, 굴소스 [요리순서] 인원수에 맞게 목살을 준비하시고 후추와 소금으로 밑간을 해줍니다 (집에 허브솔트가 있다면 굿굿) 양념된 고기를 튀기듯 구워줍니다 (백종원 아저씨의 팁) 기름 살짝 두르고 강한불에 구웠습니다 고기가 구워질때 후추를 조금 더 뿌려줍니다. 타지 않을 정도록 구워주세요 야채는 미리 손질하여 준비해주세요! 고기와 마늘을 먼저 한번 볶아주시고 마늘이 반쯤 익었을 때 야채와 굴소스를 넣고 볶아줍니다! (야채의 숨이 반정도 죽었을 때 불을끄세요!) 짜잔 ! 완성이 되었습니다 ! 야채를 숨이 반쯤 죽었을때 불을 끄라는 것을 고기의 열로 야채의 숨이 충분히 죽기 때문이지요^^ 맛있게 쌈 한입에 소주한잔! 맥주한잔 기울이면 금상첨화 입니다 ! 다음번에 즐거운 요리로 또 찾아오겠습니다!! 행복하세요! 레시피 상세보기 5.목살엔 블랙올리브 by.루비아수아베 분량 :2인분 조리시간 :30분 이내 난이도 :초급 [주재료] 돼지고기 목살 400g, 블랙 올리브 8~10개, 할라피뇨 2~3개, 다진 마늘 1큰술, 올리브 오일 1큰술, 파슬리 1작은술 [양념 및 소스재료] 소금 약간, 후춧가루 약간 [요리순서] 돼지고기에는 소금과 후추가루로 간을 해 줍니다. 블랙 올리브는 2~3조각으로 자르고, 할라피뇨는 크게크게 다져줍니다. 팬이 달궈지기 전에 올리브 오일을 두르고 마늘을 넣고 갈색이 되도록 볶아 준다. 블랙 올리브와 할라피뇨를 넣고 함께 볶다가 파슬리 가루를 넣고 잘 섞어 준 후에 빼 둔다. 돼지고기 목살을 달군 팬에 올려 구워 준다. 반대쪽까지 노릇하게 굽는다. 접시에 잘 구워진 목살을 올리고 만들어둔 블랙 올리브 소스를 올려 준다. 먹기 좋은 크기로 잘라 블랙 올리브 소스를 올려도 좋다. 레시피 상세보기 " />

function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');

        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');

    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
