당신이 늦거나 서비스 준비에 늦었다면 완벽한 음식을 제공하기 어려워진다. 모든 요리사는 각자의 위치에서 자기의 능력을 제대로 발휘해야 한다. 한 명만 늦어도 매우 불안정한 상태로 그날 서비스가 진행될 것이다.4.당신이 마련한 미장 플라스가 완벽한지 확인해야 한다주방 조직원 간의 신뢰는 매우 중요하다. 매일 서비스가 시작되기 전 완벽한 준비를 마쳐야 한다. 호흡이 잘 맞는 동료와 일을 하면 기대 이상의 매출을 올릴 수도 있다. 하루의 첫 주문서가 주방에 도달하기 전 모든 준비를 마쳐야 한다. 만약 어떤 한 파트라도 준비를 마치지 못한다면, 연쇄적으로 다른 파트에도 영향을 미치고, 결국 주방은 멈춰버릴 것이다.5.다른 요리사의 미쟝 플라스에 손대지 말 것사전 준비인 미장 플라스는 체계적인 조리를 위해 계획대로 완성되어야 한다. 만약 자신이 맡은 준비가 미흡하다고 다른 사람이 준비해 놓은 것을 빌려오려고 생각하면 안 된다. 차라리 도움이 필요하다고 주변에 알리는 것이 더 나은 방법이다. 이미 말했지만, 주방에서의 신뢰는 굉장히 소중하다.6.큰소리로 말하는 것은 개인적인 감정 때문이 아니다물론 듣는 입장에서는 불편할 수 있다. 하지만 현실을 인정해야 한다. 바쁜 주방에서는 때때로 누군가의 주의를 끌고 상황을 알릴 필요가 있다. 물론 집중력을 높이기 위해 큰소리를 낼 때도 있지만, 개인적인 감정을 담아 그러는 것이 아니다.일부 요리사들은 이런 환경이 서로 적대적인 감정을 부추길 것이라고 한다. 하지만 큰소리로 말한 뒤에 미소나 격려, 칭찬 또는 사과가 따라온다면 굉장히 효율적인 주방이 될 것이다. 대부분 주방장들은 이 방법을 알고 있다.7.무릎을 이용해서 들어라주방에서 생기는 부상 대부분은 무언가를 들어 올리거나 몸을 비틀 때 생긴다. 25kg의 양파 자루나 밀가루 포대를 들 때 허리를 구부려 드는 모습을 상상하면 된다. 무거운 물건을 들거나 힘을 써야 할 때는 도움을 요청하던가, 무릎을 이용해 일어서듯 들어야 한다. 또한, 어깨높이에 있는 찬장에 무거운 것은 올려놓지 않도록 주의한다. 건강한 신체는 요리사의 기본 덕목이다.8.항상 깨끗이 일하라영리한 요리사들은 아무리 바쁜 서비스 시간이라도 작업대를 깨끗하게 유지한다. 또한 소독제, 젖은 행주, 마른행주 등을 갖춰놓고 청결함에 신경을 쓴다. 초보 요리사가 반드시 체득해야 할 습관이다.9.노크하라주방에선 무수히 많은 사고가 생길 수 있다. 만약 워크인 냉장고에 창문이 없어 밖을 볼 수 없다면 문을 열기 전에 항상 노크하는 습관을 들여야 한다. 이러한 행동은 주방에서 일하는 모든 직원이 당연히 지켜야 할 사항이다.10.‘뒤에 있다’고 알려라당신이 어디에 있는지 사람들에게 알려라. 요리사들은 항상 시간에 쫓기는 동시에 항상 손에 무언가를 들고 움직인다. 모든 주방에는 사각지대가 존재하고 충돌할 수 있는 여지가 많다는 것을 잊지 말자. 당신이 누군가의 뒤를 지나가거나, 모퉁이를 돌 때, 그리고 문을 열고 어딘가를 들어갈 때마다 자신의 상황을 알려야 한다.11.디테일에 집착하라모든 것이 중요하다. 만약 당신이 스토브 위에 팬을 올려놓고 잠시 자리를 비운다면 팬의 손잡이를 돌려놓기를 바란다. 통로에 툭 튀어나온 손잡이는 이동에 방해되기 때문이다. 전자 조리 기구를 사용했다면 사용 후 바로 설거지를 해 놓아라. 콘소메 수프를 만들 때 모든 재료를 브뤼누아즈brunoise 로 썰었는지, 소스를 만들기 위해 송아지 육수를 만들 계획이라면 미르포아Mire Poix의 카라멜라이징을 완벽하게 했는지 확인하라.말하자면 끝이 없다. 모든 디테일에 집중하라.12.제시간에 높은 수준의 요리를 해야 한다물론 모든 요리사는 요리를 대충 그리고 빨리 끝내는 방법을 알고 있다. 그러나 음식의 수준과 시간 중 하나만 고르라면, 음식의 수준을 선택하라. 절대로 지름길을 선택하지 말아라. 사소한 타협이 모여 평범한 레스토랑을 만들게 된다. 성공적인 레스토랑의 주방에서 일하고 싶다면 두 마리 토끼를 잡는 방법을 연구할 필요가 있다. 요리사는 속도와 실력 모두 필요하다.13.젖은 수건으로 뜨거운 팬을 잡지 말 나타나지 않으면 동료들은 몹시 화가 날 것이고 빈자리를 채우기 위해 고군분투할 것이다. 하지만 설거지 담당자가 나타나지 않으면 주방은 멈추고 만다. 함께 일하는 사람이라면 응당 존중할 줄 알아야 한다.16.공급받는 식재료의 수준을 점검하여라자신이 원하는 수준의 식재료를 꾸준하게 받기 위해서는 거래처에 수시로 이의를 제기하고 의심해야 한다. 그들은 당신의 function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {

function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();

            _blogIsSubmit = false;
            return html;

            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
