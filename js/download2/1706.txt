(밥은 다 익은 상태이기에 오래 굽지 않으셔도 된답니다) 기호에 따라 돈까스 소스나 스테이크 소스를 뿌려 제철과일이나 샐러드등을 올려 함께 셋팅해 주시면 전문점 못지 않은 고급스러운 밥스테이크가 완성된답니다. 레시피 상세보기 5.찬밥 식빵스틱 by.돌하우스 분량 :2인분 조리시간 :30분 이내 난이도 :초급 [주재료] 김치, 햄, 식빵, 카레가루, 밥, 계란, 빵가루, 통깨약간, 식용류 [요리순서] 치는 국물을 꼬옥 짜준후 송송 썰어줍니다. 햄은 잘게 썰어 볶아서 사용합니다. 찬밥은 렌지에 데워 햄, 김치, 통깨, 카레가루를 넣어 섞어줍니다. 계란물과 빵가루를 준비해주세요. 식빵은 가장자리를 자르고 밀대로 밀어둡니다. 식빵 가장자리에 계란물을 발라줍니다. 밥을 올려 돌돌 말고 가장자리를 꾹꾹 눌러줍니다. 완성된 스틱은 계란옷을 입혀줍니다. 빵가루까지 입힌 식빵스틱은 기름에 노릇하게 튀겨주세요. 레시피 상세보기 6.찬밥 게살 크림 크로켓 만들기 by.MJ후다닥레시피 조리시간 :30분 이내 난이도 :중급 [주재료] 찬밥 한공기, 우유 200ml, 밀가루 3스푼가득, 버터 30g, 다진양파, 파프리카, 버섯, 크레미 7개, 허브쏠트 (소금), 식물성기름 약간, 밀가루, 달걀, 빵가루, 토핑용 돈까스 소스 [요리순서] 양파와 버섯, 파프리카는 다져서 준비해 주세요. 식물성 기름을 두른 다음 양파를 넣고 잘 볶아 줍니다. 양파가 노릇할 정도로 볶아지면 버터를 넣고 녹여 주세요. 이어서 밀가루 3스푼 정도 넣어 함께 저어 주세요. 우유를 한컵 부어 주세요. 곧이어 밀가루가 우유를 빨아들이면서 되직한 반죽이 될거에요. '루'를 속성으로 만들어 본거랍니다 . 불을 약하게 줄이고 파프리카, 버섯, 크래미를 넣어 잠시만 뒤적뒤적하며 익혀 주세요. 오늘의 주인공 찬밥을 넣어 주걱으로 으깨가며 잘 섞이게 해 줍니다. 불을 끈 후 허브솔트(소금)을 넣어 간을 맞춥니다. 볼에 담아 식혀준 후, 요렇게 동그랗게 모양을 잡아 만들어 주세요. 밀가루를 뿌려 주고 달걀물이 잘 뭍을 수 있도록 굴려준 후, 달걀에 넣어 줍니다. 빵가루를 묻혀 주고, 기름에 튀겨 주세요. 불이 쎄면 타기 쉬우니 잘 조절해 주시고 오래 튀기실 필요는 없어요~ 고루고루 노릇하게 한 번씩만 튀기면 돼요~ 어린잎 채소에 토핑견과 올리고 오리엔탈 소스를 곁들인 샐러드와 크로켓 위에 돈까스 소스를 올려 장식했어요. 레시피 상세보기 " />

function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",

            $("#divBlogContentsList").html(html);

}

 <h1><b>6.찬밥 게살 크림 크로켓 만들기</b>
 <span>by.MJ후다닥레시피</span>

 <b>조리시간 :</b>30분 이내 <b>난이도 :</b>중급 </div>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>찬밥 한공기, 우유 200ml, 밀가루 3스푼가득, 버터 30g, 다진양파, 파프리카, 버섯, 크레미 7개, 허브쏠트 (소금), 식물성기름 약간, 밀가루, 달걀, 빵가루, 토핑용 돈까스 소스 </dd>

 <li>곧이어 밀가루가 우유를 빨아들이면서 되직한 반죽이 될거에요. '루'를 속성으로 만들어 본거랍니다 .</li>
 <li>불을 약하게 줄이고 파프리카, 버섯, 크래미를 넣어 잠시만 뒤적뒤적하며 익혀 주세요.</li>
 <li>오늘의 주인공 찬밥을 넣어 주걱으로 으깨가며 잘 섞이게 해 줍니다. 불을 끈 후 허브솔트(소금)을 넣어 간을 맞춥니다.</li>
 <li>볼에 담아 식혀준 후, 요렇게 동그랗게 모양을 잡아 만들어 주세요.</li>
 <li>밀가루를 뿌려 주고 달걀물이 잘 뭍을 수 있도록 굴려준 후, 달걀에 넣어 줍니다.</li>
 <li>빵가루를 묻혀 주고, 기름에 튀겨 주세요. 불이 쎄면 타기 쉬우니 잘 조절해 주시고 오래 튀기실 필요는 없어요~ 고루고루 노릇하게 한 번씩만 튀기면 돼요~</li>
 <li>어린잎 채소에 토핑견과 올리고 오리엔탈 소스를 곁들인 샐러드와 크로켓 위에 돈까스 소스를 올려 장식했어요.</li>
 </ol>

        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
