 <span>by.eponger</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>초급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/09/09/6fea41459c39806c3db447917db4e2a81.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>닭가슴살 3쪽, 채소들(오이, 파프리카, 양파, 양배추 등 원하는 채소로), 메밀국수 500원동전 양만큼 </dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>육수 : 시중에 파는 냉면육수 한 봉지, 식초 1-2숟갈, 겨자 1/2숟갈, 설탕 약간, 소금 약간</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>시중에 손질된 닭가슴살 부위를 사서 해도 되고 아님 저처럼 집에 닭볶음탕하려고 사 놓은 닭에서 가슴살만 가져와서 써도 돼요~ 닭을 깨끗이 씻어주고, 시간이 있으면 20분정도 물에 담궈서 핏물을 제거해주세요~ 안해도 무방</li>
 <li>​집에 있는 채소를 예쁘게 얇게 썰어주세요. 초계탕에는 양배추를 많이 넣던데 저는 없어서 그냥 보이는 파프리카로~</li>
 <li>냄비에 잘 씻은 닭과 물 4컵정도를 부어서 약 2.30분정도 끓여주세요. ​</li>
 <li>닭은 건져놓고 식혀주시고~​</li>
 <li>육수는 버리지 말고 대접에 잘 넣어주세요~</li>
 <li>좀 식은 닭은 잘 찢어주세요~</li>
 <li>냉면 용 그릇에 삶은 메밀국수를 젤 밑에 깔아주세요. 메밀국수는 약 6-7분정도 삶아주세용. 면을 별로 안 좋아하면 빼도 괜찮아요~</li>
 <li>​면 위에 잘 썰어놓은 채소를 예쁘게 담공~ ​​</li>
 <li>그 가운데 찢어놓은 닭가슴살을 살포시 올려주세용~</li>
 <li>육수는 닭 우린 육수 한 컵과 냉면육수 한봉지, 식초 1-2숟갈, 겨자 1/2숟갈, 설탕 약간, 소금 약간 넣고 잘 섞어주세요~ 맛 보시면서 개인 취향대로 약간씩 더 넣어주시면 돼용~ 다 만들어진 육수는 냉동실에 넣어서 얼려서 먹는게 더 맛있답니다~채소, 닭을 잘 올린 그릇에 모양이 흐트러지지 않게 육수를 넣고 깨뿌리고 얼음을 넣어주면 끝~</li>
 </ol>
 </dd>
 </dl>

 <h1><b>8.초계탕 만들기</b>
 <span>by.피피꽁주</span>
 <div class="recipe_info">
 <b>분량 :</b>1인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>중급 </div>
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/05/11/92d2dfc74a55db6e53124ab526876bcb1.jpg">
 </h1>

 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>토종닭 1마리, 동봉된 한약재, 당근 1개, 송고버섯 1팩, 오이 2개, 달걀 5개, 일반 국수, 메밀국수 한 줌씩, 볶음용 포도씨유 1큰술</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>절임용 소금 2큰술, 참기름 1큰술, 연두 1큰술, 닭 육수 2컵, 설탕 2큰술, 겨자 1큰술, 소금 1큰술 반, 잣 4큰술, 참깨 3큰술</dd>
 </dl>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 <li>토종닭은 기름기 많은 부분을 가위로 잘라내주세요. 동봉된 한약재를 흐르는 물에 가볍게 씻은 뒤 토종닭과 함께 푹 삶아요.</li>
 <li>잘 삶아진 토종닭은 24cm 편수 멀티웍에 부어주고, 닭 육수는 아래로 토종닭만 남겨지면 가슴살은 분리하고 닭 다리는 즉석에서 뜯어먹어요!</li>
 <li>닭 육수는 충분히 식혀 냉동실로 직행해요!</li>
 <li>송고 버섯은 미지근한 물에 충분히 불려 두세 번 씻어준 뒤 물기를 꼭 짜서 분량의 양념을 넣고 달달 볶아주어요!</li>
 <li>오이는 4등분 해서 돌려 깎기 한 뒤 (씨부분은 제외) 얇게 채 썰고 굵은소금을 뿌려 잠시 절여주세요.</li>
 <li>당근도 오이와 같은 길이로 채 썰어 소금에 절여 깨끗하게 씻어 물기를 빼주어요!</li>
 <li>달걀은 황백으로 나눠 소금을 약간 넣고 풀어 얇게 지단을 부쳐냅니다. 달걀지단 충분히 식혀 채 썰어야 예쁜 모양으로 잘려요~</li>
 <li>당근과 오이는 바닥에 눌어붙지 않고 미끄러지듯 잘 볶아주세요.</li>
 <li>초계탕 고명으로 올라갈 재료들 모두 완성해놓았어요.</li>
 <li>잣, 볶은 참깨 그리고 분량의 설탕, 식초, 소금, 연겨자를 계량해요! 지금까지는 닭 육수 냉동실에서 얼리는 중인지라 이 상태로 놓고 국수 먼저 삶기 시작해요!</li>
 <li>잘 삶아진 국수는 헹구어주세요.</li>
 <li>닭 육수는 촘촘한 삼베망에 걸러주세요.</li>
 <li>잣, 볶은 참깨 그리고 분량의 설탕, 식초, 소금, 연겨자를 계량 한것을 넣고 믹서기로 갈아줘요!</li>
 <li>뽀얀 초계탕 육수 완성이랍니다. 이때 맛을 보고 취향에 맞게 소금이나 연겨자를 추가해주면 좋겠어요.</li>
 <li>초계탕 위에 검정깨를 뿌려주면 훨씬 더 식감을 자극하겠네요~</li>
 </ol>
 </dd>
 </dl>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
