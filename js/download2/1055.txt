function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

 <h1><b>3.팬케이크로 만든 크리스마스 케이크</b>
 <span>by.밀키</span>
 <div class="recipe_info">
 <b>분량 :</b>4인분 <b>조리시간 :</b>30분 이내 <b>난이도 :</b>초급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/09/08/27321f7a0475b003e76f21f423b7506e.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>핫케익가루 500g, 우유 250g, 계란 2개, 생크림 1컵, 설탕 2큰술, 딸기조금, 식용유 조금</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>핫케익가루 봉투에 레시피를 참고로 만드시면 되는데 저는 위의 레시피를 반으로 줄여서 커다란 핫케익 한장을 구워 만들었어요. 먼저 볼에 우유와 계란을 넣고 거품기로 섞어 주고</li>
 
 <li>핫케익가루를 넣고 섞어서 반죽을 만들어 줍니다.</li>
 
 <li>주르룩 흐르면서 리본모양으로 떨어지면 잘된 반죽 입니다.</li>
 
 <li>팬을 가스불(중불로 조절)에 올려 달군후 식용유를 몇방울 뿌려주고 식용유를 키친타올로 닦아 줍니다. 반죽을 팬에 부워주고 불을 중불에서 약불로 조절하고 핫케익을 구워줍니다. 사진처럼 기포가 생기다가 톡톡 터지면서 맛있는 냄새가 날때 팬을 살살 흔들어 준후 뒤집개로 뒤집어 주세요.</li>
 
 <li>예쁜 황금색으로 잘구워 졌죠?</li>
 
 <li>구워진 핫케익은 잠시 식혀 주세요.</li>
 
 <li>쿠키커터(쿠키커터가 없다면 소주잔으로) 적당한 사이즈의 도구로 핫케익을 잘라 줍니다. 저는 10개정도 잘라 주었어요.</li>
 
 <li>생크림에 설탕을 넣고 거품기로 휘핑해 주세요. 사진처럼 볼을 기울여서 거품기를 탁탁 쳐주면서 돌려가며 한쪽 방향으로 4분간 휘핑해주면 됩니다.</li>
 
 <li>크림이 딱딱하게 휘핑이 되었다면 생크림을 조금 더넣고 부드러운 휘핑크림을 만들어 주세요. 그래야 크림이 촛농처럼 주르룩 흘러 내린답니다.</li>
 
 <li>잘라둔 핫케익에 생크림을 수저로 발라주고 샌드해 줍니다. 중간중간 과일을 얹어 샌드하면 더 맛있는 케익이 된답니다.</li>
 
 <li>맨위에 수저로 생크림을 듬뿍 올려 주면 자연스럽게 크림이 주르룩 흘려 내려 촛농처럼 보인답니다.</li>
 
 <li>딸기를 올려서 촛불모양을 완성해 주면 케이크가 완성. 저는 식용은구슬(초코과자류) 아라잔을 뿌려 주었어. 슈가파우더를 뿌려도 예쁠것 같아요.</li>
 </ol>
