계속 섞어주다 보면 이렇게 겉면이 하얗게 굳어요. 이때 초콜릿을 다시 부어주시면 되지요. 10번에 걸쳐서 초콜릿을 입힌 모습이예요. 울퉁불퉁한 모습도 많이 사라지고, 두툼하게 초콜릿 옷을 입은 아몬드가 된것 같지요? 이제 마지막 작업입니다. 아망드 쇼콜라에 체친 코코아파우더를 넣어서 주걱으로 고루 섞어주세요. 그런다음 체에 살살 내려주면 아망드 쇼콜라 완성이예요~ 룰루~ 레시피 상세보기 2.달콤함에 퐁당~ 핫초코라떼 by.서현마미 조리시간 :15분 이내 난이도 :아무나 [주재료] 우유 200cc, 다크초콜릿(코코아함량 55%) 45g, 비정제황설탕 1/2ts, 소금 반꼬집, 계핏가루 약간, 거품낼 우유 80cc [요리순서] 우유를 두툼한 냄비에 부어 뜨뜻하게 데워서 불에서 내려주고, 잘게 자른 초콜릿, 설탕, 소금, 계핏가루를 넣어서 잘 저어주세요. 초콜릿이 다 녹으면 불에 다시 올려서 뜨끈하게 데워주고, 우유는 거품을 풍성하고 보드랍게 내어서 준비해요. 우유거품기가 없으면 냄비에 우유를 넣고 데워가면서 거품기로 힘껏 휘저어 주시면 되어요. 컵에 핫초코를 담아주고, 그 위에 우유거품을 가득 얹어주시면 매력적인 핫초코라떼 완성이예요. 코코아가루도 솔솔 뿌려줍니다. 레시피 상세보기 3.오레오 브라우니 만들기 by.잼양 분량 :2인분 조리시간 :60분 이내 난이도 :초급 [주재료] 박력분 1T, 버터 90g, 다크초콜릿 180g, 계란 3개, 코코아가루 10g, 오레오쿠키 100g, 설탕 80g [요리순서] 먼저 초콜릿과 버터를 중탕시켜주세요 뜨거운 물을 밑에 받쳐준 뒤 그 열기로 녹이는 게 중탕입니다 계란 3개를 볼에 풀어 준 뒤 설탕을 나눠가며 거품기로 휘핑해주세요 이렇게 풍성하고 뽀얀 거품을 만들어주세요 중탕한 초콜릿을 반죽에 넣어 섞어주세요 미리 체쳐준 박력분과 코코아가루를 넣고 주걱으로 자르듯이 섞어주세요 거품이 죽지 않도록 살살 섞어주세요 분량의 오레오를 살짝 부숴준 다음 반죽에 넣고 섞어주세요 유산지를 깔아준 팬에 반죽을 팬닝하고 남은 오레오로 장식해주세요 180도로 예열한 오븐에서 15~20분간 구워주세요 레시피 상세보기 " />

계속 섞어주다 보면 이렇게 겉면이 하얗게 굳어요. 이때 초콜릿을 다시 부어주시면 되지요. 10번에 걸쳐서 초콜릿을 입힌 모습이예요. 울퉁불퉁한 모습도 많이 사라지고, 두툼하게 초콜릿 옷을 입은 아몬드가 된것 같지요? 이제 마지막 작업입니다. 아망드 쇼콜라에 체친 코코아파우더를 넣어서 주걱으로 고루 섞어주세요. 그런다음 체에 살살 내려주면 아망드 쇼콜라 완성이예요~ 룰루~ 레시피 상세보기 2.달콤함에 퐁당~ 핫초코라떼 by.서현마미 조리시간 :15분 이내 난이도 :아무나 [주재료] 우유 200cc, 다크초콜릿(코코아함량 55%) 45g, 비정제황설탕 1/2ts, 소금 반꼬집, 계핏가루 약간, 거품낼 우유 80cc [요리순서] 우유를 두툼한 냄비에 부어 뜨뜻하게 데워서 불에서 내려주고, 잘게 자른 초콜릿, 설탕, 소금, 계핏가루를 넣어서 잘 저어주세요. 초콜릿이 다 녹으면 불에 다시 올려서 뜨끈하게 데워주고, 우유는 거품을 풍성하고 보드랍게 내어서 준비해요. 우유거품기가 없으면 냄비에 우유를 넣고 데워가면서 거품기로 힘껏 휘저어 주시면 되어요. 컵에 핫초코를 담아주고, 그 위에 우유거품을 가득 얹어주시면 매력적인 핫초코라떼 완성이예요. 코코아가루도 솔솔 뿌려줍니다. 레시피 상세보기 3.오레오 브라우니 만들기 by.잼양 분량 :2인분 조리시간 :60분 이내 난이도 :초급 [주재료] 박력분 1T, 버터 90g, 다크초콜릿 180g, 계란 3개, 코코아가루 10g, 오레오쿠키 100g, 설탕 80g [요리순서] 먼저 초콜릿과 버터를 중탕시켜주세요 뜨거운 물을 밑에 받쳐준 뒤 그 열기로 녹이는 게 중탕입니다 계란 3개를 볼에 풀어 준 뒤 설탕을 나눠가며 거품기로 휘핑해주세요 이렇게 풍성하고 뽀얀 거품을 만들어주세요 중탕한 초콜릿을 반죽에 넣어 섞어주세요 미리 체쳐준 박력분과 코코아가루를 넣고 주걱으로 자르듯이 섞어주세요 거품이 죽지 않도록 살살 섞어주세요 분량의 오레오를 살짝 부숴준 다음 반죽에 넣고 섞어주세요 유산지를 깔아준 팬에 반죽을 팬닝하고 남은 오레오로 장식해주세요 180도로 예열한 오븐에서 15~20분간 구워주세요 레시피 상세보기 " />

var curVRPage = 1;

function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';

    for(var i=skip; i<skip+scale; i++)

        var rvs = rvArr[i];
        if(!rvs) continue;

    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
