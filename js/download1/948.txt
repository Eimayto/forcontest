<meta property="og:description" content="한입 베어 물면 패티의 육즙이 촤르르~!패티만 만들어서 냉동실에 얼려두면 간편하게 아이 간식으로 만들어주기 좋겠죠?그동안 느낄 수 없었던 수제 햄버거의 매력으로 빠져볼까요? 1.초간단 수제 햄버거 패티 만드는 방법 by.요리가좋아in독일 분량 :3인분 조리시간 :10분 이내 난이도 :아무나 [주재료] 다진고기 500g(돼지50:소50), 달걀 1개, 양파 1개, 마늘 4쪽 [양념 및 소스재료] 통후추 1T, 소금 1T [요리순서] 다진고기를 볼에 담아주세요. 재료들을 섞어야하니 좀 움푹 패인 볼이 좋겠습니다. 양파와 마늘을 잘게 다져 넣습니다. 믹서에 살짝 갈아도 무방할것 같습니다. 달걀 하나를 깨서 넣습니다. 통후추와 소금도 기호에 맞게 넣어주세요. 그리고 마구 치대주세요! 그리고 나서 하나하나 패티로 사용하기 좋을 크기로 떼어내어 지퍼백에 담아주면 끝! 냉동실에 얼려두었다가 먹을때마다 하나씩 해동하면 문제 없어요! 레시피 상세보기 2.고기 패티 두툼하게 홈메이드 햄버거 by.예쁜포비 분량 :3인분 난이도 :중급 [주재료] 잡곡빵 6조각, 양파 1/2개, 피망 1/2개 토마토 슬라이스 3조각, 슬라이스 치즈 3장, 어린잎채소 약간, 다진 소고기 200g, 다진 돼지고기 200g, 다진 양파 4큰술, 다진 당근 4큰술 [양념 및 소스재료] 식용유 약간, 발사믹소스 약간, 마요네즈 3큰술, 홀그레인머스타드 1큰술, 빵가루 2큰술, 다진 파 2큰술, 다진마늘 1큰술, 달걀 1개, 간장 1큰술, 소금, 후춧가루 약간씩 [요리순서] 다진 돼지고기와 소고기를 준비해 주세요. 햄버거 고기 패티 재료를 분량대로 볼에 담고 끈기가 생기도록 치대 주세요. (다진 소고기 200g, 다진 돼지고기 200g, 다진 양파 4큰술, 다진 당근 4큰술, 빵가루 2큰술, 다진 파 2큰술, 다진마늘 1큰술, 달걀 1개, 간장 1큰술, 소금, 후춧가루 약간씩) 고기 패티를 둥글 납작하게 두툼하게 빚어 주세요. 전 고기가 두툼한 것을 좋아해서 3개만 만들었는데 4개로 만드셔도 됩니다. 잡곡빵과 토마토, 어린잎 채소, 슬라이스 치즈를 준비해 주세요. 달군 팬에 식용유를 두른 후 중간 불에서 겉면만 노릇노릇하게 앞뒤로 구워서 색을 내 주세요. 고기가 얇으면 프라이팬에서 익혀도 되지만 전 고기가 두꺼워서 오븐에서 다시 익혀 주었습니다. 일단 육즙이 빠져나가지 못하게 겉면만 프라이팬에서 익혀 주었습니다. 180℃ 예열된 오븐에서 앞뒤로 5분씩 구워 주세요. 고기를 구운 팬에 양파와 파프리카 슬라이스 한 것을 볶아 주세요. 이 때 소금간 약간 해 주세요. 요즘 재료에 각자 간을 해 주셔야 모든 음식이 맛이 좋습니다. 적당한 크기로 자른 곡물빵을 프라이팬 또는 토스트기에 구워 주세요. 마요네즈와 홀그레인머스타드 섞어 빵에 발라 주세요. 그래야 빵에 수분이 스며드는 것을 방지해 줍니다. 빵 - 어린잎 채소 - 고기 패티 - 슬라이스 치즈 볶은 양파와 파프리카를 얹은 후 발사믹 소스와 치즈가루를 더 뿌려 주었습니다. 진짜 두툼한 고기 패티 햄버거 완성 레시피 상세보기 3.데리야끼버거~ 버거보다 맛있는 엄마표 햄버거 by.봉자바리 조리시간 :30분 이내 난이도 :초급 [주재료] 햄버거빵, 치즈, 양상추, 피클, 마요네즈, 데리야끼소스 [양념 및 소스재료] 갈은소고기 220g, 갈은돼지고기 200g, 양파 1/3개, 감자(소) 1개, 계란 1개, 빵가루, 후추가루, 소금 약간 [요리순서] 갈은소고기220g, 갈은돼지고기200g 분량의 고기를 준비합니다. 꼭 소고기가 20그람 많아야하진 않구요.. 전 집에있는 고기를 사용한거라서... 똑같은 비율로 하셔도 되고... 소고기가 좀더 들어가면 기름기가 좀 적은 담백한 패티가 된답니다. 양파 1/3개는 다져서 후라이팬에 한번 슬쩍 볶아 줍니다. 그래야 나중에 패티에 수분이 생기지 않는답니다. 감자 작은거 한개는 쪄서 으깨어 놓습니다. 패티에 감자 으깬걸 넣어줄껀데요. 이렇게 감자를 쪄서 으깨넣으면 패티가 식어도 촉촉하고 부드러움을 유지한답니다~ 갈은 고기, 볶은양파, 으깬감자를 몽땅넣고.. 계란한개와 빵가루도 넣고...후추와 소금으로 간을하며 열씨미 치대줍니다. 빵가루의 양은 너무 퍽퍽하지 않게 질기를 보면서 넣어줍니다. 전..집에서 직접 갈은 빵가루를 넣었더니 더 부드럽고 고소하더라구요~ 간은 좀 약하게... 데리야끼 소스를 뿌려줄꺼라서 간은 약하게 만들었답니다. 둥글납작하게 성형해서 랩으로 하나하나 싸주었답니다. 이렇게해서 냉동보관했다 비상식량으로...ㅋㅋ 하나하나 잘 떼어지고 좋아요~ 햄버거의 재료들.. 요즘 토마토도 맛있으니 토마토 넣어도 좋구요~ 전..다진피클이 있어서 넣어주었답니다. 양상추나 상추는 필수~ 하랑이와 먹을꺼라서 모닝빵으로 준비했답니다. 빵을 반갈라 마른후라이팬에 살짝 구워줍니다. 만들어 놓은 패티를 노릇노릇 약한불에서 속까지 완벽히 익혀주세요~ 구워놓은 빵에 마요네즈 살짝 바르고 상추한장 올리고 패티올리고 마요네즈와 데리야끼소스 지그재그뿌리고 다진피클 올리고 치즈한장 올리고 햄버거 뚜껑 덮어주기~~ 레시피 상세보기 " />

var _blogIsSubmit = false;
function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);
        }
     } else {
        getBlogContentsList();
     }
}
function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}
function setBlogContents(q_link) {

        if (!q_link) {
            alert('블로그 컨텐츠를 선택하세요.');
            return;
        }
    }
    $("#btnSetBlogContents").html('<span class="fa fa-spinner fa-spin" style="width:' + $("#btnSetBlogContents").width() + 'px"></span>');
    if(fromOther == 'n') document.location.href = '/inbox/insert.html?ref_blog_url=' + encodeURIComponent(q_link);
    else if(fromOther == 'review') document.location.href = '/event/ins_review.html?seq=948&review_seq=&ref_blog_url=' + encodeURIComponent(q_link);
}

}

function doCommentSubmit()
{
    var txtVal = $("#cmtTextarea").val();
    if(!txtVal)
    {
        alert('의견을 입력해주세요.');
        return ;
    }
    if(txtVal.length < 4)
    {
        alert('의견은 4자 이상으로 작성해 주세요.');
        return ;
    }
    $.ajax({
        type: "POST",
        url: "/bbs/ajax.html",
        data : $("#insCmtFrm").serialize(),
        success: function(result) {
            if(result == 'SUCCESS')
            {
                $("#cmtTextarea").val('');
                getCommentList();
                var ccnt = parseInt($("#recipeCommentListCount").html(),10);
                $("#recipeCommentListCount").html(ccnt+1);
            }
            else if(result == 'NO_PARAM')
            {
                alert('파라미터 부족. 다시 확인해 주세요.')
            }
            else
            {
                alert('등록 실패! 잠시후 다시 시도해 주세요.')
            }
        },
        error: function () {
            alert('의견 조회 실패! 잠시후 다시 시도해 주세요.')
        }
    });
}

            }
        });
    }
}
