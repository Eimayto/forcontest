function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');

}

 <dd>닭다리살 500g, 생강가루 1t, 후추 1t, 소금 1t, 청주 소주잔한컵, 새송이버섯, 양파, 떡, 쪽파나부추, 피자치즈, 통깨</dd>

 <dd>다진마늘 1큰술, 고춧가루 2큰술, 고추장 2큰술, 간장 2큰술, 설탕 1큰술, 다진파 1큰술, 카레가루 1큰술, 후추 1t, 맛술 1큰술, 쥐똥고추 기호에따라</dd>

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>닭다리살은 미리 청주, 생강가루, 후추, 소금에 재워 둔다.</li>
 
 <li>닭갈비에 같이 들어갈 재료들을 손질해주시고 우선 버섯, 양파, 양배추, 부추를 떡크기나 닭갈비 크기로 썰어서 준비합니다.</li>
 
 <li>올리브쇼 김호윤셰프의 양념장을 준비하는데 고춧가루, 후추, 다진마늘, 파, 쥐똥고추를 일단 넣고 저는 천연조미료도 넣어 주었습니다.</li>
 
 <li>그리고 간장, 맛술이나 청주, 카레가루를 넣어 주고 잘 섞어 줍니다.</li>
 
 <li>마지막으로 간을 보면서 설탕을 넣어 줍니다.</li>
 
 <li>이제 오늘뭐먹지에서 권우중셰프가 보여준 불맛을 내는 방법 첫번째를 해야하겠지요. 각종 야채들과 재워둔 고기를 준비대기시켜줍니다.</li>
 
 <li>그리고는 연기가 날 정도로 팬을 달궈 줍니다. 기름을 전혀 두르지 않습니다. 절대로 기름을 두르지 않고 그냥 팬 자체로 달궈줍니다. 연기가 날때까지</li>
 
 <li>이 팬은 닭을 구워줄 팬이고 위의 팬은 야채를 볶아 줄 팬입니다. 둘다 기름없이 열심히 달궈지고 있습니다.</li>
 
 <li>충분히 달궈지고 팬에서 연기가 나게 되면 기름없이 야채를 넣고 볶아 줍니다. 일종에 태워준다고 하는 것이 맞겠죠. 여기에서 바로 불맛을 내어주는 것입니다.</li>
 
 <li>다른 팬에 닭갈비는 껍질쪽부터 구워주기 시작합니다. 역시나 기름을 두르지 않았지요. 그렇지만 닭에서 나오는 기름이 있기 때문에 중간중간에 기름을 종이타월로 닦아 내면서 구워 줍니다.</li>
 
 <li>각각의 야채들은 따로 따로 기름없이 팬에 볶아 줍니다. 그리고 다 볶아진 야채들은 따로 그릇에 담아서 준비대기 시켜주시구요.</li>
 
 <li>껍질쪽이 노릇하게 완전히 익었다면 뒤집어 구워 줍니다.</li>
 
 <li>이렇게 팬에 기름이 없이 볶아내면 정말 탄듯하게 불맛이 입혀지고 그을린 듯한 야채들이 됩니다. 우와~~~~~ 냄새에서부터 벌써 불맛이 나네요.</li>
 
 <li>이제 나머지 야채들을 볶아 줍니다. 기름없이</li>
 
 <li>그리고 특이하게 떡을 닭다리살을 구워주었던 팬에 구워줍니다. 바삭바삭하게~~~~~~~</li>
 
 <li>이렇게 모든 재료들이 다 볶아 졌습니다. 불맛을 입고서는</li>
 
 <li>그렇다면 이번에 오늘뭐먹지에서 소개한 불맛의 오삼불고기 권우중셰프의 최고의 불맛을 내는 방법을 할 차례가 왔습니다요~~ ㅋ

바로 뜨겁게 달궈진 팬에 역시나 아까처럼 연기가 날정도의 팬에 설탕을 쭈욱 두르고</li>
 
 <li>거기에 올리브오일을 두르고</li>
 
 <li>설탕이 끓어 오르고 마치 강정을 만들때 처럼 설탕을 태워 줍니다.</li>
 
 <li>이제 설탕이 쫄려졌다면 불에 그을리듯이 볶아 주었던 각종 야채들과 닭다리살을 함께 넣고 잘 섞어주면서 볶아 줍니다.</li>
 
 <li>마지막으로 양념장을 넣고 기름없이 볶아 줍니다. 기름은 전혀 넣지 않아요. 설탕을 볶을때에만 조금 넣어주었지요~~</li>
 
 <li>이건 올리브쇼 김호윤셰프의 방법을 이용했지요. 바로 피자치즈를 위에 올려 주고 원래는 토치를 사용해서 구워주지만 토치를 사용하지 않기 때문에 그대로 잘 섞어서 치즈맛이 나게 했지요.</li>
 
 <li>이렇게 불맛나는 닭갈비가 모두 완성되었습니다요. 권우중셰프의 오삼불고기를 응용하고 올리브쇼의 김호윤셰프님의 양념으로 완성된 불맛나는 닭갈비입니다요~</li>
 </ol>
