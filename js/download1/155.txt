Jon Shook(오른쪽)매 순간을 즐겨라. 노트를 가지고 다니면서 많이 귀담아들어라. 그리고 여행에서 당신이 가진 돈을 먹고 즐기는 데 모두 써라. 100배 이상 값어치를 할 것이다.많은 것을 먹으러 돌아다녀라. 다른 사람들이 요리하는 모습을 보는 것도 아주 중요하다. 이건 당신의 미각을 훈련 시키고 발전시킬 것이고 영감도 줄 것이다. 이 조언은 나의 첫 번째 멘토인 미셸 번스타인Michelle Bernstein으로부터 받은 것이다. 타이슨 콜 (Tyson Cole / Uchi/Uchiko / Austin, TX)요리사가 하고 일은 특별하다. 그리고 우리는 충분히 이 일에 즐거움을 느끼고 있다. 하지만 가끔은 요리 자체에 파묻힌다. 우리는 돈을 내고 음식을 먹는 손님이나, 우리 음식을 먹을 사람들이 있다는 것을 잊어서는 안 된다.조쉬 케이폰 (Josh Capon / Lure Fishbar / New York, NY)좋아하는 것뿐만 아니라 어떤 것에서 겪었던 경험을 최대화시켜 하루빨리 당신의 경력으로 만들길 바란다. 흡수하고 기록해라, 계속 여행을 다니고 체험해라.사소한 경험도 경력으로 만들어라. 만약 당신이 미슐랭 3개의 별을 받은 주방에서 하루를 보냈다면, 당장 집에 가서 노트를 펴라. 당신이 눈으로 보고, 냄새를 맡고, 직접 만졌던 것들과 재료를 준비하고 요리하고 먹었던 것을 통틀어 기록해라. 나중에 기록한 노트를 다시 펴보았을 때, 모든 것을 맛이 날 수 있도록. 브라이언 말라키 (Brian Malarkey / Searsucker / San Diego, CA)현명한 셰프는 말한다. ‘당신이 저축한 돈을 세지 말고 지금 벌고 있는 돈을 세라’. 일관성이 최고다. 전력 질주 하지 말고 속도를 유지해라.다니앨 홀즈먼 (Daniel Holzman / The Meatball Shop / New York, NY)항상 스스로에게 물어라. 요리를 하는데 왜 특정한 방법으로 하는지. 주방에서는 모든 방법을 배울 수 있다. 그 방법에 대해 이유를 설명할 수 있다면 남들보다 앞서 나가고 있는 것이다.그리고 기억해라. 맛이 최우선이다. 맛이 없다면 다른 문제는 문제도 아니다.찰리 파머 (Charlie Palmer / Aureole / New York, NY)레스토랑에서는 당신의 기술을 발전시킬 수 있는 수많은 업무가 있다. 이 모든 것을 배우기 위해 분명한 목표를 세워라. 그렇지 않으면 주방 안에서 갇혀만 있으면서 더 넓은 세계로 나아갈 수 없을 것이다.미카엘 페라로 (Michael Ferraro / Delicatessen and Macbar / New York, NY)셰프가 되려면 의사가 되기 위해 들이는 시간과 비슷한 양의 시간을 투자해야 한다. 일반적으로승진을 하기 위해서는 8년에서 10년동안 훈련과 연구를 준비할 필요가 있다. 최근 요리학교를 졸업한 사람들이 너무 급하게 자기 레스토랑을 열고 있는 것 같다.제프 미쇼 (Jeff Michaud / Vetri Restaurants / Philadelphia, PA)주방에서는 보통 일주일에 75시간 넘게 아니면 더 오랜 시간을 일할 수 있다. 그렇게 되면 지인들이나 자신의 생일을 매년 놓칠 것이다. 또 매년 휴일과 주말에도 일을 해서 친구와 멀어질 것이다. 왜냐하면 친구들은 당신이 왜 더 일해야 하는지 이해하지 못한다. 하지만 대신 레스토랑에서 새로운 친구를 사귈 것이다. 이러한 레스토랑 사정을 이해하게 된다면 보람있는 일이 될 것이다. 스티븐 레지카우스키 (Steven Redzikowski / Acorn / Denver, CO)편안히 심호흡을 하면서 휴식을 취하라. 매사에 서두르지 않고 차근차근히 해라. 그리고 당신이 지금 어디에 서 있는지 확인하면서 그 순간을 즐겨라.· 셰프뉴스 웹사이트 가기 : http://chefnews.kr · 셰프뉴스에서 보기 : http://chefnews.kr/archives/7856">

<meta property="og:description" content="Editor’s Note : 본 기사는 지난해 10월12일 Thrillist에 소개된 영문 콘텐츠를 번역, 재구성한 기사입니다.원본 보기 : https://www.thrillist.com/eat/nation/chef-advice-lessons-for-young-cooks레스토랑 주방 안에서는 요리사들의 직급이 다양하다. 칼 갈기, 재료 기본 손질 등을 맡아 하는 주방보조commis, 일정 기간 수련 후 하나의 역할을 맡는 파트셰프line cook, 주방의 한 라인을 통솔하며 주방장을 보조하는 부주방장sous chef, 주방 전체를 통솔하고 지휘하는 주방장head chef까지의 단계가 있다. 셰프로 성장하기 위해선 보통 10년 이상의 시간이 걸린다.그 10년을 어떻게 버텨내야 하는지도 문제다. 버티느냐도 문제다. 주방이라는 제한된 공간에서 하루 평균 10~12시간 이상 일할 수 있는 체력이 필요하다. 더불어 트렌드를 먼저 읽어내고 독창적인 아이디어가 깃든 레시피를 꾸준히 개발해야 한다.셰프로 한 걸음 나아가기를 준비하는 이들에게 실제 미국에서 레스토랑을 운영하는 오너셰프들은 다음과 같은 조언을 전한다.토니 만투아노 (Tony Mantuano / Spiaggia / Chicago, IL )요리사는 영감을 받기 위해 각 나라의 전통과 문화를 이해해야 한다. 이것은 굉장히 중요하다. 관심이 있는 지역이 있다면 당장 떠나라. 그리고 그 지역의 역사를 살피고 퀴진을 공부해라. 이 지역 셰프들의 요리 기술과 각각의 특징을 배워라. 지역 주민들의 삶도 관찰해라. 이러한 경험은 당신의 음식과 특별한 관점을 형성하는 데 큰 도움을 줄 것이다.루도 르페뷔레 (Ludo Lefebvre / Petit Trois / Los Angeles, CA)먼저 칼의 사용법을 완벽하게 익혀라. 밀려드는 주문서에 파묻혀 눈코 뜰 새 없이 바쁘기 때문이다.민디 세갈 (Mindy Segal / Hot Chocolate / Chicago, IL)귀를 열고 겸손해져라. 완벽한 기술을 내기 위해 평생 배워야 한다는 사실을 잊지 마라.팀 러브 (Tim Love / Lonesome Dove / Dallas, TX)가능한 한 많은 사람에게서 배워라. 미래를 걱정하지 말고 지금 있는 자리에서 열심히 일하라.비니 도토로, 존 슉 (Jon Shook, Vinny Dotolo/ Animal / Los Angeles, CA) Vinny Dotolo(왼쪽) &amp; Jon Shook(오른쪽)매 순간을 즐겨라. 노트를 가지고 다니면서 많이 귀담아들어라. 그리고 여행에서 당신이 가진 돈을 먹고 즐기는 데 모두 써라. 100배 이상 값어치를 할 것이다.많은 것을 먹으러 돌아다녀라. 다른 사람들이 요리하는 모습을 보는 것도 아주 중요하다. 이건 당신의 미각을 훈련 시키고 발전시킬 것이고 영감도 줄 것이다. 이 조언은 나의 첫 번째 멘토인 미셸 번스타인Michelle Bernstein으로부터 받은 것이다. 타이슨 콜 (Tyson Cole / Uchi/Uchiko / Austin, TX)요리사가 하고 일은 특별하다. 그리고 우리는 충분히 이 일에 즐거움을 느끼고 있다. 하지만 가끔은 요리 자체에 파묻힌다. 우리는 돈을 내고 음식을 먹는 손님이나, 우리 음식을 먹을 사람들이 있다는 것을 잊어서는 안 된다.조쉬 케이폰 (Josh Capon / Lure Fishbar / New York, NY)좋아하는 것뿐만 아니라 어떤 것에서 겪었던 경험을 최대화시켜 하루빨리 당신의 경력으로 만들길 바란다. 흡수하고 기록해라, 계속 여행을 다니고 체험해라.사소한 경험도 경력으로 만들어라. 만약 당신이 미슐랭 3개의 별을 받은 주방에서 하루를 보냈다면, 당장 집에 가서 노트를 펴라. 당신이 눈으로 보고, 냄새를 맡고, 직접 만졌던 것들과 재료를 준비하고 요리하고 먹었던 것을 통틀어 기록해라. 나중에 기록한 노트를 다시 펴보았을 때, 모든 것을 맛이 날 수 있도록. 브라이언 말라키 (Brian Malarkey / Searsucker / San Diego, CA)현명한 셰프는 말한다. ‘당신이 저축한 돈을 세지 말고 지금 벌고 있는 돈을 세라’. 일관성이 최고다. 전력 질주 하지 말고 속도를 유지해라.다니앨 홀즈먼 (Daniel Holzman / The Meatball Shop / New York, NY)항상 스스로에게 물어라. 요리를 하는데 왜 특정한 방법으로 하는지. 주방에서는 모든 방법을 배울 수 있다. 그 방법에 대해 이유를 설명할 수 있다면 남들보다 앞서 나가고 있는 것이다.그리고 기억해라. 맛이 최우선이다. 맛이 없다면 다른 문제는 문제도 아니다.찰리 파머 (Charlie Palmer / Aureole / New York, NY)레스토랑에서는 당신의 기술을 발전시킬 수 있는 수많은 업무가 있다. 이 모든 것을 배우기 위해 분명한 목표를 세워라. 그렇지 않으면 주방 안에서 갇혀만 있으면서 더 넓은 세계로 나아갈 수 없을 것이다.미카엘 페라로 (Michael Ferraro / Delicatessen and Macbar / New York, NY)셰프가 되려면 의사가 되기 위해 들이는 시간과 비슷한 양의 시간을 투자해야 한다. 일반적으로승진을 하기 위해서는 8년에서 10년동안 훈련과 연구를 준비할 필요가 있다. 최근 요리학교를 졸업한 사람들이 너무 급하게 자기 레스토랑을 열고 있는 것 같다.제프 미쇼 (Jeff Michaud / Vetri Restaurants / Philadelphia, PA)주방에서는 보통 일주일에 75시간 넘게 아니면 더 오랜 시간을 일할 수 있다. 그렇게 되면 지인들이나 자신의 생일을 매년 놓칠 것이다. 또 매년 휴일과 주말에도 일을 해서 친구와 멀어질 것이다. 왜냐하면 친구들은 당신이 왜 더 일해야 하는지 이해하지 못한다. 하지만 대신 레스토랑에서 새로운 친구를 사귈 것이다. 이러한 레스토랑 사정을 이해하게 된다면 보람있는 일이 될 것이다. 스티븐 레지카우스키 (Steven Redzikowski / Acorn / Denver, CO)편안히 심호흡을 하면서 휴식을 취하라. 매사에 서두르지 않고 차근차근히 해라. 그리고 당신이 지금 어디에 서 있는지 확인하면서 그 순간을 즐겨라.· 셰프뉴스 웹사이트 가기 : http://chefnews.kr · 셰프뉴스에서 보기 : http://chefnews.kr/archives/7856" />

var _blogIsSubmit = false;
function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);
        }
     } else {
        getBlogContentsList();
     }
}
function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}
function setBlogContents(q_link) {
    if (typeof q_link == 'undefined' || q_link == '') {
        q_link = $("#listBlog [id^=q_link_]:checked").val();

        if (!q_link) {
            alert('블로그 컨텐츠를 선택하세요.');
            return;
        }
    }
    $("#btnSetBlogContents").html('<span class="fa fa-spinner fa-spin" style="width:' + $("#btnSetBlogContents").width() + 'px"></span>');
    if(fromOther == 'n') document.location.href = '/inbox/insert.html?ref_blog_url=' + encodeURIComponent(q_link);
    else if(fromOther == 'review') document.location.href = '/event/ins_review.html?seq=155&review_seq=&ref_blog_url=' + encodeURIComponent(q_link);
}

        $("#top_blog_select_id").show();
    }

}

function doCommentSubmit()
{
    var txtVal = $("#cmtTextarea").val();
    if(!txtVal)
    {
        alert('의견을 입력해주세요.');
        return ;
    }
    if(txtVal.length < 4)
    {
        alert('의견은 4자 이상으로 작성해 주세요.');
        return ;
    }
    $.ajax({
        type: "POST",
        url: "/bbs/ajax.html",
        data : $("#insCmtFrm").serialize(),
        success: function(result) {
            if(result == 'SUCCESS')
            {
                $("#cmtTextarea").val('');
                getCommentList();
                var ccnt = parseInt($("#recipeCommentListCount").html(),10);
                $("#recipeCommentListCount").html(ccnt+1);
            }
            else if(result == 'NO_PARAM')
            {
                alert('파라미터 부족. 다시 확인해 주세요.')
            }
            else
            {
                alert('등록 실패! 잠시후 다시 시도해 주세요.')
            }
        },
        error: function () {
            alert('의견 조회 실패! 잠시후 다시 시도해 주세요.')
        }
    });
}

}

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}

$(document).ready(function () {
    getViewRecipeList(1);
})
