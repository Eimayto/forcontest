function getBlogContentsList(page,added_params) {

        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');

        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');

    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",

 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>토마토 껍질을 벗겨줍니다. 토마토를 (+)십자로 칼집 내어 준 후 끓는 물에 살짝 데쳐 주면 손 쉽게 벗겨 집니다.</li>
 
 <li>찬물에 헹구고 껍질을 싹 다 벗겨주세요. 그냥 물에 헹구는 정도에도 벌써 껍질이 벗겨졌네요.</li>
 
 <li>토마토를 4등분해 꼭지부분을 제거하고 씨를 제거해준 후, 0.5cm크기로 잘라 줍니다.</li>
 
 <li>홍고추를 반으로 가르고 꼭지를 따낸 후 씨와 하얀 속 부분을 모두 숟가락으로 긁어서 제거해줍니다. 홍고추를 가늘게 채썬 후 사방 0.5cm로 잘게 잘라줍니다.</li>
 
 <li>청피망도 반을 갈라 꼭지를 제거해준후 씨와 흰부분을 제거해 줍니다. 그리고 홍고추와 같이 사방 0.5cm로 잘라줍니다.</li>
 
 <li>직접 닭 한마리를 뼈를 발라 잡으실 거라면 닭을 우선 소금으로 박박 문질러 이물질을 닦아내 주세요. 기름기를 적당히 제거하며 살을 발라내줍니다.</li>
 
 <li>먹기 좋은 크기 약 사방 3~4cm정도로 자르고 소금, 후추로 조금 간해줍니다.</li>
 
 <li>미리 풀어준 계란과 전분을 넣고 옷입혀주세요.</li>
 
 <li>180도 이상 열을 받은 튀김기름에 튀김 옷을 입은 닭을 조각 조각 떼어 넣고 바삭하게 한번 튀겨 냅니다.</li>
 
 <li>노릇해지면 체를 이용해 잠시 건져낸 후 기름이 다시 열받으면 다시 한번더 바삭하게 튀겨줍니다.</li>
 
 <li>팬에 기름을 살짝 두르고 홍고추, 토마토 순으로 넣고 볶아줍니다.</li>
 
 <li>물3T와 설탕 4~5T을 넣고 주걱으로 잘 섞어가며 끓여주세요. 케찹 2T, 고추장 1T, 간장 1T을 넣고 잘 풀어주며 섞어줍니다. 청피망을 넣고 빠르게 뒤적 뒤적해줍니다</li>
 
 <li>미리 튀겨둔 닭튀김(치킨)을 넣고 골고루 섞어주며 볶아주세요. 수분감이 (소스가 마른느낌) 없을 정도로 볶아주시고 불을 끕니다.</li>
 </ol>
