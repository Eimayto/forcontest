이렇게 많이 만든 멸치볶음. 아마 일주일이면 다 먹을것같아요. 밑반찬 걱정 끝~ 레시피 상세보기 3.고추잡채 만들기 by.쉐프cookie 분량 :3인분 조리시간 :60분 이내 난이도 :초급 [주재료] 돼지고기 300g, 새우(중하) 7마리, 대파 1뿌리, 양파 1개, 청,홍 피망 1/2 개씩, 청량고추 12개, 새송이 버섯, 당근, 쥬키니 호박 1개, 쪽파 1줌, 생강 1톨, 감자 전분 1/2컵, 계란 흰자 1개분량 [양념 및 소스재료] 고추기름 3스픈, 스시간장 1스픈(보통 간장으로 대체가능), 굴소스 1스픈, 정종 1스픈, 설탕 1스픈, 고운 고춧가루 1스픈, 참기름 1티스픈 돼지고기밑간 - 소금, 후추, 마늘, 다진생강 [요리순서] 먼저 야채를 위와 같이 손질 해 놓습니다. 새우는 길게 반으로 포를 떠서 소금 후추 간을 해 둡니다. 돼지고기는 길쭉길쭉하게 썰어서 소금, 후추 와 다진 마늘 약간, 다진 생강 약간으로 간을 해서 재어 둡니다. 20분 정도 재어 둔 돼지고기는 계란 흰자 1개 분량으로 버무린 뒤 마른 감자 전분을 입혀서 튀겨 냅니다. 튀겨진 고기는 기름을 빼고 대기시켜두세요. 돼지고기를 전분에 튀겨내는 이유는 식감을 좋게하기 위한 제 방식이랍니다. 먼저 팬에 고추기름을 두르고 다진 생강이랑 대파를 먼저 넣어 볶습니다. 기름에 향이 베면 모든 야채를 넣고 함께 고운 고춧가루도 같이 넣은 후 센불에서 볶아냅니다. 야채가 볶이면 튀겨 놓은 돼지 고기를 넣고 준비 해둔 소스를 넣어 간을 합니다. 중분히 간이 베이게 볶은 후 마지막으로 참기름을 두르고 접시에 담으면 끝. 그리고 고추 잡채에서 빠지면 안되는 것이 바로 꽃빵이죠? function doChangePortal(portal) {
    var header = '';

    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);

     } else {

function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}

}

 <h1><b>1.아삭이고추튀김</b>
 <span>by.아내의식탁</span>
 <div class="recipe_info">
 <b>분량 :</b>1인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>초급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/10/30/ae0cdb6fde738e7cbb2892e9dcc436821.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>아삭이고추(작은사이즈) 7~8개, 다진돼지고기 150g, 새우 6마리, 양파 1/4개, 튀김가루 약간</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>속양념재료 - 굴소스 1큰술, 청주 1/2큰술, 다진마늘 1/2큰술, 참기름 약간, 소금 약간, 후추 약간, 튀김반죽 - 튀김가루 1+1/4컵, 냉수 1컵, 달걀노른자 1개, 얼음 약간, 초간장 - 진간장 2큰술, 물 1큰술, 설탕 1큰술, 식초 1큰술, 다진양파 약간, 청고추 약간</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>재료를 준비해주세요.</li>
 
 <li>아삭이고추는 길게 반으로 갈라 속을 털어주세요. 양파는 곱게 다지고, 새우는 껍질과 내장을 분리한 후 굵게 다져주세요.</li>
 
 <li>볼에 다진 돼지고기와 새우, 양파를 넣고 속양념재료를 섞어주세요.</li>
 
 <li>아삭이고추의 겉과 속에 튀김가루를 묻힌 후 속양념을 넣어주세요.</li>
 
 <li>아삭이고추에 튀김가루-튀김반죽을 순서대로 묻힌 후</li>
 
 <li>160도의 기름에서 튀겨주세요.</li>
 
 <li>그릇에 튀김을 담고 초간장을 곁들여 주세요.(180도의 기름에서 아삭이고추를 한번 더 튀기면 좀 더 바삭하게 드실 수 있답니다.)</li>
 </ol>
 </dd>
 </dl>

 <h1><b>2.[견과류멸치볶음] 호두와 아몬드 청양고추 팍팍 멸치볶음 만들기</b>
 <span>by.양싸</span>
 <div class="recipe_info">
 <b>분량 :</b>6인분 이상 <b>조리시간 :</b>30분 이내 <b>난이도 :</b>아무나 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/12/08/52b48bec9fec8111ef11c8a5872fbdd51.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>잔멸치 200g, 청양고추 10개, 호두 150g, 아몬드 150g, 식용유 두번 둘둘 둘러주세요</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>간장 2T, 설탕 6T, 물엿 3T, 깨 1T, 참기름 1/2T</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>잔멸치 200g, 호두 150g, 아몬드 150g을 준비해주세요.
청양고추는10개 얇게 슬라이스해서 준비해주세요.

*호두랑 아몬드 베이킹용..
음식에 들어가면 다 똑같아요~~*</li>
 
 <li>양념은 간장 2T, 설탕 6T, 물엿 3T, 참기름 1/2T 를 넣고 섞어 주세요. 
멸치볶음은 조금 달달하게 볶아야 맛있을것같아서 조금 달게 했어요.
어차피 청양고추도 10개나 들어가니까 이정도의 달달함쯤이야..^^</li>
 
 <li>마른 팬에 멸치를 넣고 센불에서 볶아주세요.
타지않게 조심하세요.
멸치의 수분을 날려보내고
노릇 노릇 바삭해질때까지
볶아주세요.</li>
 
 <li>센불에서 멸치를 볶다가 호두와 아몬드도 넣고 달달 볶아줍니다.
호두와 아몬드가 살짝 노릇해질때까지 볶아주세요.</li>
 
 <li>식용유를 둘둘 두번 둘러 볶아주세요.
깨 1T 도 넣고 같이 볶아줍니다.
안넣으셔도 상관없어요.
청양고추도 넣고 바삭 바삭한
느낌이 들게 볶아줍니다.</li>
 
 <li>노릇 노릇 다 볶아지면
불을 끄세요.
그리고 아까의 양념을 휙 두르고
잘 섞어주세요.
잔열로 양념을 잘 버무리고
다시 불을 켜고 약불에서 살짝만 볶아줍니다.</li>
 
 <li>완성!!
매콤 달콤~ 사실 단맛이 더 강해요. 청양고추 10개는 어디로 간거니?
이렇게 많이 만든 멸치볶음.
아마 일주일이면 다 먹을것같아요. 밑반찬 걱정 끝~</li>
 </ol>
 </dd>

 <h1><b>3.고추잡채 만들기</b>
 <span>by.쉐프cookie</span>
 <div class="recipe_info">
 <b>분량 :</b>3인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>초급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/11/29/6c57e0e3fbeb51dedcfaeec0cdb700df1.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>돼지고기 300g, 새우(중하) 7마리, 대파 1뿌리, 양파 1개, 청,홍 피망 1/2 개씩, 청량고추 12개, 새송이 버섯, 당근, 쥬키니 호박 1개, 쪽파 1줌, 생강 1톨, 감자 전분 1/2컵, 계란 흰자 1개분량 </dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>고추기름 3스픈, 스시간장 1스픈(보통 간장으로 대체가능), 굴소스 1스픈, 정종 1스픈, 설탕 1스픈, 고운 고춧가루 1스픈, 참기름 1티스픈<br>
돼지고기밑간 - 소금, 후추, 마늘, 다진생강</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>먼저 야채를 위와 같이 손질 해 놓습니다. 새우는 길게 반으로 포를 떠서 소금 후추 간을 해 둡니다. 돼지고기는 길쭉길쭉하게 썰어서 소금, 후추 와 다진 마늘 약간, 다진 생강 약간으로 간을 해서 재어 둡니다.</li>
 
 <li>20분 정도 재어 둔 돼지고기는 계란 흰자 1개 분량으로 버무린 뒤 마른 감자 전분을 입혀서 튀겨 냅니다.</li>
 
 <li>튀겨진 고기는 기름을 빼고 대기시켜두세요. 돼지고기를 전분에 튀겨내는 이유는 식감을 좋게하기 위한 제 방식이랍니다.</li>
 
 <li>먼저 팬에 고추기름을 두르고 다진 생강이랑 대파를 먼저 넣어 볶습니다. 기름에 향이 베면 모든 야채를 넣고 함께 고운 고춧가루도 같이 넣은 후 센불에서 볶아냅니다.</li>
 
 <li>야채가 볶이면 튀겨 놓은 돼지 고기를 넣고 준비 해둔 소스를 넣어 간을 합니다. 중분히 간이 베이게 볶은 후 마지막으로 참기름을 두르고 접시에 담으면 끝. 그리고 고추 잡채에서 빠지면 안되는 것이 바로 꽃빵이죠? 솔직히 만들어 먹기는 일이많기도 하고..쉽지않으니 마트에서 사서 먹는 게 좋으실 듯해요.</li>
 </ol>
 </dd>
 </dl>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
