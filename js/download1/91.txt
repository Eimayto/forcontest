<meta property="og:description" content=" 1.아침식사 대용 포도 요거트 파르페 by.무지개 난이도 :아무나 [주재료] 포도 탐슨 시들리스 1/3컵, 떠먹는 요구르트 1+1/2개, 씨리얼, 견과류, 쵸콜릿, 쵸코과자, 스프링클 약간 [양념 및 소스재료] 쨈 [요리순서] 포도는 청포도 탐슨 시들리스를 깨끗이 씻어 준비하시고, 포도의 1/2는 적당한 크기로 썰어 줍니다. 떠먹는 요거트, 씨리얼, 견과류 등 토핑재료를 취향에 따라 준비하세요~ 투명한 유리컵에 씨리얼을 컵의 1/4정도 넣고, 그 위에 견과류를 적당량 넣어 준 다음 떠먹는 요구르트 1컵을 넣어 줍니다. Tip. 담백하게 요거트만 넣었는데 달콤한 맛을 더 강조하고 싶다면, 요거트를 반만 넣고, 딸기쨈을 넣은 뒤 다시 요거트로 살짝 덮어주면 되요~ 청포도 자른 것을 적당량 올리고 남은 요거트를 올려서 덮어 주면 화려하고 푸짐해 보이는 파르페 완성입니다. 레시피 상세보기 2.[수요미식회]최현석셰프의 포도쥬스 스테이크만들기 by.눈꽃식탁 분량 :4인분 조리시간 :60분 이내 난이도 :중급 [주재료] 소고기등심 800g, 파프리카색깔별로 1개씩, 새송이버섯 2개, 아스파라거스 4개, 양파 1개, 로즈마리 4줄기 [양념 및 소스재료] 소금,후추 약간씩, 올리브오일 4Ts, 포도쥬스 500ml [요리순서] 고기는 소금,후추로 밑간해주세요. ​ 하기전에 핏물은 키친타월로 제거해주세요. 가니쉬를 미리 만들건데요. 가니쉬란-접시에 올려서 먹는 익은야채를 말해요! 새송이는 얇게 썰어놓고 올리브오일과 소금으로 살짝 발라주세요. 파프리카는 통으로 구울거에요. 아스파라거스는 줄기부분은 필러로 까주세요. (겉면을 살짝 벗겨주셔야해요. 그래야 질기지 않아요.) 아스파라거스와 양파는 후라이팬에 굽고요.(이때 버터를 두르고 구우셔도 좋고 고기구울때 같이구우셔도 되요.) 파프리카는 직화로 석쇠위에서 돌려가면서 겉면을 태우세요. (껍질을 벗길과정인데 익혀서 껍질을 벗기고 드시면 더 달큰하고 영양가도 높아져요.) 포도쥬스는 냄비에서 보글보글 끓여주세요.1/3까지 졸여주세요.(포도쥬스는 될수있음 순수한걸로 준비) 차갑게 식히면 점성이 생기긴한데 마음에 안드시면 녹말물1Ts를 섞어주심 소스는 걸죽하게되요. 파프리카는 태웠으면 껍질을 벗겨주세요.(찬물에서 벗기면 쉽게 벗겨져요.) 버섯은 석쇠에서 살짝 구워주세요. 이제 고기를 구워줄건데요. 올리브오일을 두르고 로즈마리를 놓고 향이 나기시작하면 고기를 넣으세요. 고기는 도톰하니깐 첨에 센불에서 누룽지(시어링과정)을 거치시고 양쪽을 시어링해주세요. ​ 이때는 누르시면 안되요. 젓가락을 눌러봐서 턱밑에 젓가락을 대면 차가우면 레어 미지근하면 미디움 따뜻하면 웰던이에요. 전 미디움으로 구웠어요. 접시에 가니쉬를 먼저 담고 고기를 올리고 소스를 부어주면 완성입니다. 먹어보니 고기가 제대로 숙성해서 그런지 부드럽더라구요. 소스는 생각보다 아주 달지 않았구요. 고기도 적당히 익혀서 질기지도 않았답니다.(될수있음 미디움으로 구워드시면 부드럽게 드실수있어요.) 레시피 상세보기 3.새콤달콤~ 포도 크럼블 케이크 by.초록바람N 분량 :2인분 조리시간 :90분 이내 난이도 :중급 [주재료] ( 포도필링 ) 포도 250 g, 유기농설탕 4 큰술, 레몬즙 2 큰술, 옥수수전분 1/2 큰술, ( 크럼블 ) 실온버터 15 g, 유기농 설탕 15 g, 박력분 30 g, 아몬드가루 10 g, ( 케이크 ) 실온버터 40 g, 유기농설탕 40 g, 실온계란 30 g, 베이킹 파우더 1/2 작은술, 아몬드 ﻿가루 20 g, 박력분 60 g, 실온우유 30 g, 바닐라 익스트랙 2/3 작은술 [요리순서] 먼저 포도 필링을 만들어 식혀둡니다. 포도는 깨끗이 씻어 알알이 떼어낸 다음 반으로 갈라 씨를 제거하고 껍질과 과육을 분리해주세요. 씨를 발라낼때 즙이 나와요. 그 즙도 같이 사용합니다. 껍질만 모아서 꼬마 믹서기에 포도즙을 약간 넣고 갈아 준비해주세요. 소스팬에 포도즙과 포도 과육 그리고 설탕을 넣고 약불에서 저어가며 살짝 끓여주세요. 어느정도 끓었을때 껍질 간것과 레몬즙을 넣고 저어가며 끓여줍니다. 어느정도 졸아들었을때 체친 전분을 넣고 저어가며 한번 끓여주세요. 크럼블을 만들어요~ 볼에 버터를 부드럽게 풀고 설탕을 넣어 잘 섞어줍니다. 여기에 체친 밀가루와 아몬드가루를 넣고 손으로 부슬부슬 비벼가며 섞어주세요. 케이크를 만들어요~ 볼에 실온 버터를 부드럽게 풀고 설탕을 넣어 서걱거림이 없을 정도로 휘핑해주세요. 여기에 계란을 넣고 잘 섞어주세요. 바닐라 익스트랙도 넣고 잘 섞어주세요. 체친 아몬드가루와 박력분, 베이킹 파우더를 넣고 스패출러로 #자를 그리며 살살 섞어주세요. 어느 정도 섞이면 분량의 우유를 넣고 섞어 매끈한 반죽을 만들어주세요. 유산지를 깐 팬에 케이크 반죽을 팬닝하고 바닥에 두어번 떨어뜨려 반죽을 정리해주세요. 그 위에 포도필링을 취향껏 올리고 위에 크럼블을 올려줍니다. 180도로 예열된 오븐에서 30~40분 정도 구워주세요. 레시피 상세보기 " />

var _blogIsSubmit = false;
function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);
        }
     } else {
        getBlogContentsList();
     }
}
function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}
function setBlogContents(q_link) {
    if (typeof q_link == 'undefined' || q_link == '') {
        q_link = $("#listBlog [id^=q_link_]:checked").val();

        if (!q_link) {
            alert('블로그 컨텐츠를 선택하세요.');
            return;
        }
    }
    $("#btnSetBlogContents").html('<span class="fa fa-spinner fa-spin" style="width:' + $("#btnSetBlogContents").width() + 'px"></span>');
    if(fromOther == 'n') document.location.href = '/inbox/insert.html?ref_blog_url=' + encodeURIComponent(q_link);
    else if(fromOther == 'review') document.location.href = '/event/ins_review.html?seq=91&review_seq=&ref_blog_url=' + encodeURIComponent(q_link);
}

    }

}

function doCommentSubmit()
{
    var txtVal = $("#cmtTextarea").val();
    if(!txtVal)
    {
        alert('의견을 입력해주세요.');
        return ;
    }
    if(txtVal.length < 4)
    {
        alert('의견은 4자 이상으로 작성해 주세요.');
        return ;
    }
    $.ajax({
        type: "POST",
        url: "/bbs/ajax.html",
        data : $("#insCmtFrm").serialize(),
        success: function(result) {
            if(result == 'SUCCESS')
            {
                $("#cmtTextarea").val('');
                getCommentList();
                var ccnt = parseInt($("#recipeCommentListCount").html(),10);
                $("#recipeCommentListCount").html(ccnt+1);
            }
            else if(result == 'NO_PARAM')
            {
                alert('파라미터 부족. 다시 확인해 주세요.')
            }
            else
            {
                alert('등록 실패! 잠시후 다시 시도해 주세요.')
            }
        },
        error: function () {
            alert('의견 조회 실패! 잠시후 다시 시도해 주세요.')
        }
    });
}

        });

}

 <h1><b>1.아침식사 대용 포도 요거트 파르페 </b>
 <span>by.무지개</span>
 <div class="recipe_info">
 <b>난이도 :</b>아무나 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/04/23/a4a6ec791628559c831bcf7c0ca961a51.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>포도 탐슨 시들리스 1/3컵, 떠먹는 요구르트 1+1/2개, 씨리얼, 견과류, 쵸콜릿, 쵸코과자, 스프링클 약간</dd>

 <li>떠먹는 요거트, 씨리얼, 견과류 등 토핑재료를 취향에 따라 준비하세요~</li>

 Tip. 담백하게 요거트만 넣었는데 달콤한 맛을 더 강조하고 싶다면, 요거트를 반만 넣고, 딸기쨈을 넣은 뒤 다시 요거트로 살짝 덮어주면 되요~</li>
 
 <li>청포도 자른 것을 적당량 올리고</li>
 
 <li>남은 요거트를 올려서 덮어 주면</li>
 
 <li>화려하고 푸짐해 보이는 파르페 완성입니다.</li>

 <h1><b>2.[수요미식회]최현석셰프의 포도쥬스 스테이크만들기</b>
 <span>by.눈꽃식탁</span>
 <div class="recipe_info">
 <b>분량 :</b>4인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>중급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/08/21/da00c6e537d84c5b0debbc77800563a41.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>소고기등심 800g, 파프리카색깔별로 1개씩, 새송이버섯 2개, 아스파라거스 4개, 양파 1개, 로즈마리 4줄기</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>소금,후추 약간씩, 올리브오일 4Ts, 포도쥬스 500ml</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>고기는 소금,후추로 밑간해주세요.</li>
 
 <li>​

하기전에 핏물은 키친타월로 제거해주세요.

가니쉬를 미리 만들건데요. 가니쉬란-접시에 올려서 먹는 익은야채를 말해요!

새송이는 얇게 썰어놓고 올리브오일과 소금으로 살짝 발라주세요.

파프리카는 통으로 구울거에요.</li>
 
 <li>아스파라거스는 줄기부분은 필러로 까주세요.

(겉면을 살짝 벗겨주셔야해요. 그래야 질기지 않아요.)</li>
 
 <li>아스파라거스와 양파는 후라이팬에 굽고요.(이때 버터를 두르고 구우셔도 좋고 고기구울때 같이구우셔도 되요.)

파프리카는 직화로 석쇠위에서 돌려가면서 겉면을 태우세요.</li>
 
 <li>(껍질을 벗길과정인데 익혀서 껍질을 벗기고 드시면 더 달큰하고 영양가도 높아져요.) 

포도쥬스는 냄비에서 보글보글 끓여주세요.1/3까지 졸여주세요.(포도쥬스는 될수있음 순수한걸로 준비)</li>
 
 <li>차갑게 식히면 점성이 생기긴한데 마음에 안드시면 녹말물1Ts를 섞어주심 소스는 걸죽하게되요.

파프리카는 태웠으면 껍질을 벗겨주세요.(찬물에서 벗기면 쉽게 벗겨져요.)

버섯은 석쇠에서 살짝 구워주세요.</li>
 
 <li>이제 고기를 구워줄건데요.

올리브오일을 두르고 로즈마리를 놓고 향이 나기시작하면 고기를 넣으세요.</li>
 
 <li>고기는 도톰하니깐 첨에 센불에서 누룽지(시어링과정)을 거치시고 양쪽을 시어링해주세요.</li>
 
 <li>​

이때는 누르시면 안되요. 젓가락을 눌러봐서 턱밑에 젓가락을 대면 차가우면 레어 미지근하면 미디움 따뜻하면 웰던이에요.

전 미디움으로 구웠어요. 

접시에 가니쉬를 먼저 담고 고기를 올리고 소스를 부어주면 완성입니다.

먹어보니 고기가 제대로 숙성해서 그런지 부드럽더라구요.

소스는 생각보다 아주 달지 않았구요.

고기도 적당히 익혀서 질기지도 않았답니다.(될수있음 미디움으로 구워드시면 부드럽게 드실수있어요.)</li>
 </ol>
 </dd>

 <h1><b>3.새콤달콤~ 포도 크럼블 케이크</b>
 <span>by.초록바람N</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>90분 이내 <b>난이도 :</b>중급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/09/06/d1812eae9624aaa4c76c6c480eb8aea91.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>( 포도필링 ) 포도 250 g, 유기농설탕 4 큰술, 레몬즙 2 큰술, 옥수수전분 1/2 큰술, ( 크럼블 ) 실온버터 15 g, 유기농 설탕 15 g, 박력분 30 g, 아몬드가루 10 g, ( 케이크 ) 실온버터 40 g, 유기농설탕 40 g, 실온계란 30 g, 베이킹 파우더 1/2 작은술, 아몬드 ﻿가루 20 g, 박력분 60 g, 실온우유 30 g, 바닐라 익스트랙 2/3 작은술</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>먼저 포도 필링을 만들어 식혀둡니다. 포도는 깨끗이 씻어 알알이 떼어낸 다음 반으로 갈라 씨를 제거하고 껍질과 과육을 분리해주세요. 씨를 발라낼때 즙이 나와요. 그 즙도 같이 사용합니다. 껍질만 모아서 꼬마 믹서기에 포도즙을 약간 넣고 갈아 준비해주세요.</li>
 
 <li>소스팬에 포도즙과 포도 과육 그리고 설탕을 넣고 약불에서 저어가며 살짝 끓여주세요. 어느정도 끓었을때 껍질 간것과 레몬즙을 넣고 저어가며 끓여줍니다.</li>
 
 <li>어느정도 졸아들었을때 체친 전분을 넣고 저어가며 한번 끓여주세요.</li>
 
 <li>크럼블을 만들어요~ 볼에 버터를 부드럽게 풀고 설탕을 넣어 잘 섞어줍니다. 여기에 체친 밀가루와 아몬드가루를 넣고 손으로 부슬부슬 비벼가며 섞어주세요.</li>
 
 <li>케이크를 만들어요~ 볼에 실온 버터를 부드럽게 풀고 설탕을 넣어 서걱거림이 없을 정도로 휘핑해주세요. 여기에 계란을 넣고 잘 섞어주세요. 바닐라 익스트랙도 넣고 잘 섞어주세요.</li>
 
 <li>체친 아몬드가루와 박력분, 베이킹 파우더를 넣고 스패출러로 #자를 그리며 살살 섞어주세요. 어느 정도 섞이면 분량의 우유를 넣고 섞어 매끈한 반죽을 만들어주세요.</li>
 
 <li>유산지를 깐 팬에 케이크 반죽을 팬닝하고 바닥에 두어번 떨어뜨려 반죽을 정리해주세요. 그 위에 포도필링을 취향껏 올리고 위에 크럼블을 올려줍니다. 180도로 예열된 오븐에서 30~40분 정도 구워주세요.</li>
 </ol>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}
