<meta property="og:description" content="한입 먹으면 호랑이 기운이 솟아나는 갈비찜 소개합니다~ 1.쇠고기갈비찜 by.아내의식탁 분량 :2인분 조리시간 :120분 이내 난이도 :초급 [주재료] 쇠고기갈비 1kg, 무 1/4개, 표고버섯 6개, 당근 1개, 밤 10개, 홍고추 1개, 고기삶을때넣을재료 - 대파 1대, 양파 1개 [양념 및 소스재료] 진간장 5큰술, 배즙 1컵, 설탕 1큰술, 맛술 4큰술, 올리고당 3큰술, 다진파 2큰술, 다진마늘 1큰술, 생강즙 1큰술, 후추 약간, 참기름 약간 [요리순서] 재료를 준비한 다음, 갈비는 30분 정도 찬물에 담가 핏물을 빼고, 끓는 물에 살짝 데친 후 찬물에 헹궈 불순물을 제거해주세요. 냄비에 갈비와 고기 삶을 때 넣는 재료를 넣고 물 2리터를 넣어 1시간이상 삶아준 후 건져내어 고기결 반대방향으로 칼집을 넣어주세요.(갈비 삶은 국물은 체에 걸러 준비해주세요) 무와 당근은 한입 크기로 자른 후 모서리를 둥글게 다듬어 주시고, 표고버섯은 4등분으로 자르고 홍고추는 어슷하게 잘라주세요. 배는 강판에 갈고 양념장에 고루 섞어 갈비를 30분 이상 재워주세요. 냄비에 갈비와 갈비 삶은국물 3컵을 넣고 강불에서 20분 정도 끓입니다. 야채를 넣고 중간불에서 15~20분정도 끓인 후 참기름을 뿌려 마무리 해주세요. 레시피 상세보기 2.불금엔 화끈하게 매운갈비찜 by.메기맹이 분량 :3인분 조리시간 :120분 이내 난이도 :중급 [주재료] 소갈비 1kg, 감자 2개, 양파 1개, 대파 1대, [양념 및 소스재료] 향신채 : 대파 1/2대, 통후추 한줌, 통마늘 1줌, 청주 1컵 양념 : 소갈비끓인물 3컵, 다진양파 5큰술, 배주스 5큰술, 간장 10큰술, 국간장 1큰술, 고춧가루 6큰술, 청주 6큰술, 다진마늘 3큰술, 설탕 2큰술, 물엿 4큰술, 다진 청양고추 1개, 후춧가루 약간 [요리순서] 소갈비는 찬물에 담궈 1시간 동안 핏물을 빼주세요. 중간중간 물을 2~3번 갈아주세요. 끓는 물에 갈비를 넣고 파, 통후추, 마늘 등을 넣고 물이 다시 끓어오르면 청주를 1컵 넣어 1시간 동안 끓여주세요. 끓이는 동안 불순물과 기름을 숟가락으로 떠내주세요. 양념재료를 다 넣고 갈면서 섞어주세요. 배주스는 고기를 연하게 해주는 연육작용을 해주기 때문에 고기요리할 때 양념에 넣어주면 좋지요. 배가 있으면 배를 넣으면 더 좋습니다. 미리 만들어서 냉장숙성해두시면 더 맛있어요. 향신채를 걸러내고 육수를 따로 분리한다음 3컵만 고기와 감자와 넣고 끓여주세요. 짜잔! 그릇으로 담으면 완성!! 레시피 상세보기 3.[갈비찜]초보들도 쉽게 만들수 있는 품격 높은 단호박 소 갈비찜 by.논현쌤 분량 :3인분 조리시간 :60분 이내 난이도 :아무나 [주재료] 소갈비 700g, 단호박 1개, 당근 1/2개, 양파 1개, 감자 1개, 꽈리고추 8개, 통마늘 5개, 청주 또는 소주 2T, 물 1L, 후추 약간, 파인애플 100g [양념 및 소스재료] 간장 5T, 편생강 5개, 설탕 1T, 다진마늘 1T, 참기름 1t, 올리고당 1T, 청주 또는 소주 2T [요리순서] 갈비는 물에 담궈서 약 3시간 정도 핏물을 빼주세요. 냉장 갈비를 사면 핏물을 빼지 않아도 되요. 핏물을 빼는 동안 양념장을 만들고 냉장고에 숙성을 시켜주세요. 단호박은 깨끗이 씻은 후 뚜껑을 자르고 씨를 다 파주세요. 야채는 깨끗이 씻어서 준비해주세요. 감자와 당근은 껍질을 벗겨주세요. 이정도의 크기로 재료를 썰어주세요. 당근과 감자의 테두리를 돌려깍기 해주세요. 그래야 부서지지 않아요. 꽈리 고추는 포크로 구멍을 내주세요. 양념장이 들어가서 더 맛있습니다. 꽈리 고추가 없으면 풋고추를 사용해도 되요. 냄비에 물을 붓고 청주 또는 소주 2T를 넣고 갈비를 삶아주세요. 고기의 겉 표면만 익으면 됩니다. 불순물과 잡내를 잡아주는 과정입니다. 그리고 갈비를 헹궈주세요. 냉장 갈비를 사용할 경우 핏물을 빼지 않고 바로 이 과정을 진행하시면 됩니다. 파인애플은 으깨주세요. 파인애플은 으깨지 않고 손가락 한 마디 크기로 썰어주셔도 됩니다. 냄비에 양념장, 물1L, 파인애플, 당근, 감자를 넣고 졸여주세요. 중간불에서 20-30분정도 졸여주세요. 집집마다 가스불이 다르니 고기가 70%이상 익을때까지 졸여주세요. 그리고 양파, 고추, 통마늘, 후추 약간을 넣고 끓여주세요. 맛을 보시고 취향것 양념을 추가해주세요. 양파가 1/2정도 익으면 속을 판 단호박에 재료들을 넣고 뚜껑을 덮어주세요. 고기와 같이 졸였던 양념장도 같이 넣어주셔야 합니다. 양념장은 자박하게만 넣어주시면 됩니다. 찜기에 단호박을 쪄주세요. 약 20분 정도 쪄주시면 됩니다. 집집마다 가스불이 다르니 단호박이 익을때까지 쪄주시면 됩니다. 고기는 70%이상 익혔으니 단호박이 익으면 고기도 다 익었을겁니다. 짜짠~~~~ 완성된 단호박 갈비찜 입니다. 레시피 상세보기 " />

var _blogIsSubmit = false;
function doChangePortal(portal) {
    var header = '';
    var tail = '';
    if (portal == 'Naver') {
        header = 'http://blog.naver.com/';
        tail = '';
    } else if (portal == 'Daum') {
        header = 'http://blog.daum.net/';
        tail = '';
    } else if (portal == 'Tistory') {
        header = 'http://';
        tail = '.tistory.com';
    }
    $("#txtBlogHeader").text(header);
    $("#txtBlogTail").text(tail);
}

function doBlogSelect() {
     var select_type = $(':radio[name="q_blog_select_type"]:checked').val();
     if (select_type == 'url') {
        var q_link = $("#q_blogurl").val();
        if ($.trim(q_link) == '') {
            alert('URL을 입력해 주세요.');
            $("#q_blogurl").focus();
            return;
        } else {
            setBlogContents(q_link);
        }
     } else {
        getBlogContentsList();
     }
}
function getBlogContentsList(page,added_params) {
    if (page == 0) return;
    if (!page) {
        page = 1;
    }
    if ($("#q_blogid").val() == '') {
        alert("블로그 주소를 입력해 주세요.");
        return;
    }
    //var blog_url = $("#txtBlogHeader").text() + $("#q_blogid").val() + $("#txtBlogTail");

    if (_blogIsSubmit) {
        alert("처리중 입니다. 잠시만 기다리세요.");
        return;
    }

    if (page > parseInt($("#q_page").val(),10)) {
        $("#btnBlogNext").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogNext").width()+'px"></span>');
    } else if (page < parseInt($("#q_page").val(),10)) {
        $("#btnBlogPrev").html('<span class="fa fa-spinner fa-spin" style="width:'+$("#btnBlogPrev").width()+'px"></span>');
    }

    $("#q_page").val(page);
    _blogIsSubmit = true;
    var params = 'q_mode=get_blog_contents_list&q_portal='+$("[name=q_portal]:checked").val()+'&q_blogid='+$("#q_blogid").val()+'&q_scal='+$("#q_scal").val()+'&q_page='+page;
    if (added_params) {
        if (added_params.substring(0,1) != '&') params += '&';
        params += added_params;
    }

    $.ajax({
        type: "POST",
        cache: false,
        url: "/common/ajx_common.html",
        data: params,
        success: function(html) {
            $("#divBlogContentsList").html(html);
            $("[id^=spanBlogContentsTitle_]").ellipsis();
            $('#divModalBlogContentsList').modal('show');
            _blogIsSubmit = false;
            return html;
        },
        error: function () {
            alert('예기치 못한 오류로 인해 실패했습니다.');
            _blogIsSubmit = false;
        }
    });
}
function setBlogContents(q_link) {
    if (typeof q_link == 'undefined' || q_link == '') {
        q_link = $("#listBlog [id^=q_link_]:checked").val();

        if (!q_link) {
            alert('블로그 컨텐츠를 선택하세요.');
            return;
        }
    }
    $("#btnSetBlogContents").html('<span class="fa fa-spinner fa-spin" style="width:' + $("#btnSetBlogContents").width() + 'px"></span>');
    if(fromOther == 'n') document.location.href = '/inbox/insert.html?ref_blog_url=' + encodeURIComponent(q_link);
    else if(fromOther == 'review') document.location.href = '/event/ins_review.html?seq=342&review_seq=&ref_blog_url=' + encodeURIComponent(q_link);
}
function doChangeBlogSelectType(select_type) {
    if (select_type == 'url') {

        $("#top_blog_select_url").show();
    } else {
        $("#top_blog_select_url").hide();
        $("#top_blog_select_id").show();
    }

}

function doCommentSubmit()
{
    var txtVal = $("#cmtTextarea").val();
    if(!txtVal)
    {
        alert('의견을 입력해주세요.');
        return ;
    }
    if(txtVal.length < 4)
    {
        alert('의견은 4자 이상으로 작성해 주세요.');
        return ;
    }
    $.ajax({
        type: "POST",
        url: "/bbs/ajax.html",
        data : $("#insCmtFrm").serialize(),
        success: function(result) {
            if(result == 'SUCCESS')
            {
                $("#cmtTextarea").val('');
                getCommentList();
                var ccnt = parseInt($("#recipeCommentListCount").html(),10);
                $("#recipeCommentListCount").html(ccnt+1);
            }
            else if(result == 'NO_PARAM')
            {
                alert('파라미터 부족. 다시 확인해 주세요.')
            }
            else
            {
                alert('등록 실패! 잠시후 다시 시도해 주세요.')
            }
        },
        error: function () {
            alert('의견 조회 실패! 잠시후 다시 시도해 주세요.')
        }
    });
}

function doCommentDelete(commentSeq)
{

                alert('의견 조회 실패! 잠시후 다시 시도해 주세요.')
            }
        });
    }
}

 <h1><b>1.쇠고기갈비찜</b>
 <span>by.아내의식탁</span>
 <div class="recipe_info">
 <b>분량 :</b>2인분 <b>조리시간 :</b>120분 이내 <b>난이도 :</b>초급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/10/23/3301af1310828a2d35d797281db930751.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>쇠고기갈비 1kg, 무 1/4개, 표고버섯 6개, 당근 1개, 밤 10개, 홍고추 1개, 고기삶을때넣을재료 - 대파 1대, 양파 1개</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>진간장 5큰술, 배즙 1컵, 설탕 1큰술, 맛술 4큰술, 올리고당 3큰술, 다진파 2큰술, 다진마늘 1큰술, 생강즙 1큰술, 후추 약간, 참기름 약간</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>재료를 준비한 다음, 갈비는 30분 정도 찬물에 담가 핏물을 빼고, 끓는 물에 살짝 데친 후 찬물에 헹궈 불순물을 제거해주세요.</li>
 
 <li>냄비에 갈비와 고기 삶을 때 넣는 재료를 넣고 물 2리터를 넣어 1시간이상 삶아준 후 건져내어 고기결 반대방향으로 칼집을 넣어주세요.(갈비 삶은 국물은 체에 걸러 준비해주세요)</li>
 
 <li>무와 당근은 한입 크기로 자른 후 모서리를 둥글게 다듬어 주시고, 표고버섯은 4등분으로 자르고 홍고추는 어슷하게 잘라주세요.</li>
 
 <li>배는 강판에 갈고 양념장에 고루 섞어 갈비를 30분 이상 재워주세요.</li>
 
 <li>냄비에 갈비와 갈비 삶은국물 3컵을 넣고 강불에서 20분 정도 끓입니다.</li>
 
 <li>야채를 넣고 중간불에서 15~20분정도 끓인 후 참기름을 뿌려 마무리 해주세요.</li>
 </ol>
 </dd>

 <h1><b>2.불금엔 화끈하게 매운갈비찜</b>
 <span>by.메기맹이</span>
 <div class="recipe_info">
 <b>분량 :</b>3인분 <b>조리시간 :</b>120분 이내 <b>난이도 :</b>중급 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/05/15/6d93b7cba02a7dd200677764624e4bcb1.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>소갈비 1kg, 감자 2개, 양파 1개, 대파 1대,</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>향신채 : 대파 1/2대, 통후추 한줌, 통마늘 1줌, 청주 1컵 <br>
양념 : 소갈비끓인물 3컵, 다진양파 5큰술, 배주스 5큰술, 간장 10큰술, 국간장 1큰술, 고춧가루 6큰술, 청주 6큰술, 다진마늘 3큰술, 설탕 2큰술, 물엿 4큰술, 다진 청양고추 1개, 후춧가루 약간</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>소갈비는 찬물에 담궈 1시간 동안 핏물을 빼주세요. 중간중간 물을 2~3번 갈아주세요.</li>
 
 <li>끓는 물에 갈비를 넣고 파, 통후추, 마늘 등을 넣고 물이 다시 끓어오르면 청주를 1컵 넣어 1시간 동안 끓여주세요. 끓이는 동안 불순물과 기름을 숟가락으로 떠내주세요.</li>
 
 <li>양념재료를 다 넣고 갈면서 섞어주세요. 배주스는 고기를 연하게 해주는 연육작용을 해주기 때문에 고기요리할 때 양념에 넣어주면 좋지요. 배가 있으면 배를 넣으면 더 좋습니다. 미리 만들어서 냉장숙성해두시면 더 맛있어요.</li>
 
 <li>향신채를 걸러내고 육수를 따로 분리한다음 3컵만 고기와 감자와 넣고 끓여주세요.</li>
 
 <li>짜잔! 그릇으로 담으면 완성!!</li>
 </ol>
 </dd>
 </dl>

 <h1><b>3.[갈비찜]초보들도 쉽게 만들수 있는 품격 높은 단호박 소 갈비찜</b>
 <span>by.논현쌤</span>
 <div class="recipe_info">
 <b>분량 :</b>3인분 <b>조리시간 :</b>60분 이내 <b>난이도 :</b>아무나 </div>
 
 <img src="http://recipe.ezmember.co.kr/cache/recipe/2015/09/19/eb0ccf0ab0bb286eac431cbf7bdc3b001.jpg">
 </h1>

 
 <dl class="recipe_cont">
 <dt>[주재료]</dt>
 <dd>소갈비 700g, 단호박 1개, 당근 1/2개, 양파 1개, 감자 1개, 꽈리고추 8개, 통마늘 5개, 청주 또는 소주 2T, 물 1L, 후추 약간, 파인애플 100g</dd>
 </dl>
 <dl class="recipe_cont">
 <dt>[양념 및 소스재료]</dt>
 <dd>간장 5T, 편생강 5개, 설탕 1T, 다진마늘 1T, 참기름 1t, 올리고당 1T, 청주 또는 소주 2T</dd>
 </dl>

 
 <dl class="recipe_cont">
 <dt>[요리순서]</dt>
 <dd>
 <ol>
 
 <li>갈비는 물에 담궈서 약 3시간 정도 핏물을 빼주세요.
냉장 갈비를 사면 핏물을 빼지 않아도 되요.</li>
 
 <li>핏물을 빼는 동안 양념장을 만들고 냉장고에 숙성을 시켜주세요. 단호박은 깨끗이 씻은 후 뚜껑을 자르고 씨를 다 파주세요.</li>
 
 <li>야채는 깨끗이 씻어서 준비해주세요. 감자와 당근은 껍질을 벗겨주세요.</li>
 
 <li>이정도의 크기로 재료를 썰어주세요.</li>
 
 <li>당근과 감자의 테두리를 돌려깍기 해주세요. 그래야 부서지지 않아요.</li>
 
 <li>꽈리 고추는 포크로 구멍을 내주세요. 양념장이 들어가서 더 맛있습니다. 꽈리 고추가 없으면 풋고추를 사용해도 되요.</li>
 
 <li>냄비에 물을 붓고 청주 또는 소주 2T를 넣고 갈비를 삶아주세요. 고기의 겉 표면만 익으면 됩니다. 불순물과 잡내를 잡아주는 과정입니다. 그리고 갈비를 헹궈주세요.
냉장 갈비를 사용할 경우 핏물을 빼지 않고 바로 이 과정을 진행하시면 됩니다.</li>
 
 <li>파인애플은 으깨주세요. 파인애플은 으깨지 않고 손가락 한 마디 크기로 썰어주셔도 됩니다. 냄비에 양념장, 물1L, 파인애플, 당근, 감자를 넣고 졸여주세요. 중간불에서 20-30분정도 졸여주세요. 집집마다 가스불이 다르니 고기가 70%이상 익을때까지 졸여주세요.</li>
 
 <li>그리고 양파, 고추, 통마늘, 후추 약간을 넣고 끓여주세요. 맛을 보시고 취향것 양념을 추가해주세요.</li>
 
 <li>양파가 1/2정도 익으면 속을 판 단호박에 재료들을 넣고 뚜껑을 덮어주세요. 고기와 같이 졸였던 양념장도 같이 넣어주셔야 합니다. 양념장은 자박하게만 넣어주시면 됩니다. 찜기에 단호박을 쪄주세요. 약 20분 정도 쪄주시면 됩니다. 집집마다 가스불이 다르니 단호박이 익을때까지 쪄주시면 됩니다. 고기는 70%이상 익혔으니 단호박이 익으면 고기도 다 익었을겁니다.</li>
 
 <li>짜짠~~~~ 완성된 단호박 갈비찜 입니다.</li>
 </ol>
 </dd>
 </dl>

var curVRPage = 1;
var totalVRcnt = 0;
function getViewRecipeList(page)
{
    if($("#rRecipContDivPage_"+page).length > 0)
    {
        $("[id^='rRecipContDivPage_']").hide();
        $("#rRecipContDivPage_"+page).show();
        curVRPage = page;
        return ;
    }
    var rvArr = storageUtil.get('stRecentViewRecipe');
    if(!rvArr || rvArr == 'FAIL')
    {
        $("#recentRecipeDiv").hide();
        return ;
    }
    totalVRcnt = rvArr.length;
    var scale = 6;
    var skip = (page-1) * scale;
    var lastpage = Math.ceil(totalVRcnt/scale);
    var hstr = '';
    hstr += '<div id="rRecipContDivPage_'+page+'" class="row">';
    if(page > 1) hstr += '<a href="javascript:void(0);" class="list_btn_pre" onClick="getViewRecipeList('+(page-1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_l.png" alt="이전"></a>';
    if(page < lastpage) hstr += '<a href="javascript:void(0);" class="list_btn_next" onClick="getViewRecipeList('+(page+1)+')"><img src="http://recipe1.ezmember.co.kr/img/btn_arrow1_r.png" alt="다음"></a>';
    for(var i=skip; i<skip+scale; i++)
    {
        var rvs = rvArr[i];
        if(!rvs) continue;
        hstr += '<div class="col-xs-2">';
        hstr += '<a class="thumbnail" href="/recipe/'+rvs.seq+'"><img src="'+rvs.thumb+'" style="width:180px; height:180px;">';
        hstr += '<div class="caption elipsis_rrtitle"><b>'+rvs.title+'</b></div></a>';
        hstr += '</div>';
    }
    hstr += '</div>';
    if(page != 1 && curVRPage > page) $("#recentRecipeContDiv").prepend(hstr);
    else  $("#recentRecipeContDiv").append(hstr);
    getViewRecipeList(page);
    $(".elipsis_rrtitle").ellipsis({row: 2});
}

    getViewRecipeList(1);
})
